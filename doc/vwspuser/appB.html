WSP Environment Variables

The layout and layout.wspMgr environment variables associated with Width Spacing Patterns (WSPs) are listed below.

Only the environment variables documented in this chapter are supported for public use. All other WSP-related environment variables, regardless of their name or prefix, and undocumented aspects of the environment variables described below, are private and are subject to change at any time.

  • General WSP Environment Variables
    • figGroup
    • layer
    • regionMapFile
    • regionOnly
    • tpaShowWSPBtn
    • wspPattern
    • wspTemplateCell
    • wspTemplateFile
  • WSP Manager Environment Variables

      • autoComputePeriod
      • firstTrackOffset
      • patternRepeat
      • period
      • periodOffset
      • repeatOffset
      • trackColor
      • trackColumns
    • Import tab
      • importCellName
      • importLibName
      • importViewName
    • Derive tab

      • shiftColor
      • smartSnap
    • Options tab
      • autoClearOnLayerChange
      • ignoreBlockagesForInit
      • ignoreL0Pins
      • spacingMode
      • useAspectRatio
      • usePaletteVisibility
      • verbose
  • Track Pattern Assistant Environment Variables

    • enableAutoPushdown
    • pushdownLayers
    • removePriorPushdown
    • renameDefs
