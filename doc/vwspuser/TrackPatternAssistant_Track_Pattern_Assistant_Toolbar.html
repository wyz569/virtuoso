Track Pattern Assistant Toolbar

The following table describes the Track Pattern assistant toolbar buttons.

Icon Command Description

Display Mode

Controls the display of boundaries and track elements in the canvas. You can choose one of the following from the list: Full, Tracks, Boundary, Periods, and Period and Tracks.

Create Region

Creates width spacing pattern regions for areas where you want to use a grid that is different from the global grid.

Edit Region

Changes the attributes associated with WSP regions such as the region name, allowed patterns and pattern groups, the active pattern, wire types, and repeat mode.

Stretch Region

Stretches a width spacing pattern region based on the Number of X periods/Number of Y periods or a combination of X Step/Y Step and Multiplier.

Design Settings

Specifies the repeat mode, DEF offsets, and the offset reference for the WSSPDefs in the design:

Pull Up One

Pulls up the WSSPDefs for only this instance.

Pull Up All

Pulls up the WSSPDefs for all the instances of an instance master.

Pull Up WSPs For Layers

Pulls up the WSSPDefs for all instances of a layer.

EIP Auto Pushdown Options

Snaps the pins in a subcell to the top-level WSP tracks.

WSP Manager

Launches the WSP Manager from the Track Pattern assistant.

WSP Active Checker

Launches the WSP Active Checker from the Track Pattern assistant.
