tpaAutoRegionNameFromRSP

layout tpaAutoRegionNameFromRSP 'boolean { t | nil }

Description

Specifies when a related snap pattern is selected, the region name is automatically generated. The value specified by you for the region name is ignored. The default value is nil.

GUI Equivalent

None

Examples

envGetVal("layout" "tpaAutoRegionNameFromRSP")
envSetVal("layout" "tpaAutoRegionNameFromRSP" 'boolean t)
