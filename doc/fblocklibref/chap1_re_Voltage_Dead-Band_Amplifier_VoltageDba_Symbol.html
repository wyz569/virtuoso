Voltage Dead-Band Amplifier (VoltageDba) Symbol

The following table summarizes the details of this symbol.  

Description

Voltage dead-band amplifier, across a specified range, with separately controllable gains in the amplifying regions.
Library name

voltageDba
Transfer function

If Vin ≥ VinHigh

then Vout= gain2 × (Vin - VinHigh) + VOL

If Vin ≤ VinLow

then Vout
=
gain1 × (Vin - VinLow) + VOL

Vin = Vin+ - Vin-
Properties (Defaults)

VinLow (-10)

VinHigh(10)

VOL(0)

gain1(1)

gain2 (1)
Input impedance

Infinite
Output impedance

Zero
Frequency response

Flat
