Current Dead-Band Amplifier (currentDba) Symbol

The following table summarizes the details of this symbol.

Description

Current dead-band amplifier, across specified range, with separately controllable gains in the amplifying regions.
Library name

currentDba
Transfer function

  • If Iin ≥ IinHigh:
    Iout = gain2 × (Iin - IinHigh) + Iol
  • If Iin ≤ Iinlow:
    Iout = gain1 × (Iin - Iinlow) + Iol

Iin = Iin+ - Iin-
Properties (Defaults)

Iinlow (-1E-3)

IinHigh (1E-3)

Iol (0)

gain1 (1)

gain2 (1)
Input impedance

Infinite
Output impedance

Zero
Frequency response

Flat
