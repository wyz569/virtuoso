Divider Symbol

The following table summarizes the details of this symbol.

Description

Output is the quotient of the two input voltages. You need to consider the expected output when using this function; the output can become large even for simple inputs.
Library name

divider
Transfer function

If Vin2 is equal to zero, then Vin2 is set to 1E-12.
Properties (Defaults)

None
Input impedance

Infinite
Output impedance

Zero
Frequency response

Flat
