Supported Database Records

This appendix discusses Dracula Supported Database Records.

  • “Introduction”
  • “GDSII Records”

Introduction

Dracula supports the CAD system formats listed below. The SYSTEM or SYSIN command in the Description block specifies your database format.

  • GDSII
  • Design Framework II database (Cadence)
