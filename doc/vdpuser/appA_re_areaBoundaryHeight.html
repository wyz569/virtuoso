areaBoundaryHeight

layoutDP areaBoundaryHeight float floating_point_number

Description

Specifies the height of the virtual hierarchy area boundary.

The default is 5.0.

GUI Equivalent

Command:

Adjust Boundary

Field:

Height

Examples

envGetVal("layoutDP" "areaBoundaryHeight")
envSetVal("layoutDP" "areaBoundaryHeight" 'float 2.5)
