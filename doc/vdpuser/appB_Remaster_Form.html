Remaster Form

Use the Remaster form to replace the selected virtual hierarchy with the selected layout master that exists on disk.

Field Description

Library

Specifies the library of the layout variant to be used to replace the selected virtual hierarchy.

Cell

Specifies the cell name of the layout variant to be used to replace the selected virtual hierarchy.

View

Specifies the name of the master variant view to be used to replace the selected layout.

All Clones

Replaces all the clones of the selected virtual hierarchy with the selected layout variant.
