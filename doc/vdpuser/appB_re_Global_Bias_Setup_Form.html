Global Bias Setup Form

Use the Global Bias Setup form to create, modify, and delete global bias constraints that can be used to plan the routing of a net group.

Field Description

Global Bias

Lets you select an existing global bias constraint group.

New

Creates a new global bias constraint group by assigning a name to it. The default name for the global bias constraint group is considered as GB<number>.

Rename

Lets you rename the global bias constraint group.

Delete

Lets you delete the global bias constraint group.

Nets

Displays the name of the nets selected in the Navigator assistant and assigned to the selected global bias constraint group.

+ From Navigator

Adds pre-selected nets from the Navigator assistant to the selected global bias constraint group.

- Delete

Deletes the selected nets from the selected global bias constraint group.

Bias Areas

Lists the bias area or paths that have been created for the selected global bias constraint group.

Area

Displays the name of the bias area or path.

Bias Areas

Specifies whether the bias area is positive or negative. If the check box is selected, it indicates a positive bias. However, if the check box is deselected, it indicates a negative bias.

Layers

Displays the list of layers that have been defined for the bias area or path.

+ Rectangles

Lets you define the coordinates of the region for the bias area.

+ Paths

Lets you define the bias path that you want the nets in the bias constraint group to follow.

- Selected Area

Removes the bias area or path.

Define Layers

Displays the Bias Layers form
