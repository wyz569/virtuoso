Congestion Histogram Customize Form

Use the Congestion Histogram Customize form to filter the histogram to display specific buckets of congestion.

Field Description

Customize As

Selects one of the following methods to customize the histogram.

  • Default: Customizes the histogram and displays the congestion data in the original format.
  • Interval: Specifies the congestion percentage with which to start and the interval between congestion buckets.
  • Start and End: Specifies the congestion percentage with which to start and end on. The intervals between congestion buckets are derived automatically.
  • Specified: Specifies the exact congestion percentage buckets. The start percentage for each congestion bucket is manually entered.

Start At

Specifies the value from which you want to start congestion filtering. This field is enabled when the method to customize the histogram is selected as Interval or Start and End.

End At

Specifies the value on which you want to end congestion filtering. This field is enabled when the method to customize the histogram is selected as Start and End.

With Interval

Specifies the interval between the congestion buckets. This field is enabled when the method to customize the histogram is selected as Interval.

Refresh

Refreshes the values in the congestion buckets displayed in the form.
