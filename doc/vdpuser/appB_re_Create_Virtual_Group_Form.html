Create Virtual Group Form

Use the Create Virtual Group form to manually create a virtual group for the selected top-level instances or instances inside a virtual hierarchy.

Field Description

Name

Specifies a name for the virtual group to be created.

Orientation

Specifies an orientation for the virtual group.

Choose orientation from:

  • R0: No rotation
  • R90: Rotate 90 degrees clockwise
  • R180: Rotate 180 degrees clockwise
  • R270: Rotate 270 degrees clockwise
  • MY: Mirror over the Y axis
  • MYR90: Mirror over the Y axis and rotate 90 degrees clockwise
  • MX: Mirror over the X axis
  • MXR90: Mirror over the X axis and rotate 90 degrees clockwise
