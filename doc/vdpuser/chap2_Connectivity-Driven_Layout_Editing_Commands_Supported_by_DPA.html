Connectivity-Driven Layout Editing Commands Supported by DPA

For optimal planning of virtual hierarchies in a layout design, Design Planning and Analysis (DPA) supports several Virtuoso Layout Suite commands.

Command Use to... Related

Generate All From Source    

Generate layout counterparts for soft and hard blocks.

For schematic instances that have no layout representations available, generate a virtual hierarchy with a selectable area boundary that is snapped to the width spacing patterns (WSP) grid.

Generate soft blocks for schematic components that are missing. Use the Snap Pattern Snapping option on the Layout Editor Options form to control snapping of the generated objects.

Generate Layout Form

Virtual Hierarchy Generation

Generating a Soft Block for a Virtual Hierarchy

Generate Selected From Source    

Generate a virtual hierarchy for the selected schematic instances that have no layout representations.

The generated virtual hierarchy has a selectable area boundary that is snapped to the width spacing patterns (WSP) grid. Use the Snap Pattern Snapping option on the Layout Editor Options form to control the snapping.

Selected Component Generation in DPA

Generating Components As In Schematic

Edit In Place    

Edit in place the generated virtual hierarchy clones.

To edit the virtual hierarchies that are not clone instances, adjust the display depth to make them transparent.

Editing In Place

Generating Transparent Layout Hierarchy

Check Against Source    

Check and report mismatches in connectivity, instance terminal mismatches, and so on, between the schematic and layout views for the virtual hierarchies selected at the top level or for the ones available on Edit In Place.

Check Against Source Form

Update Components And Nets    

Update the components from source and if the selected set that has missing instances contains a virtual hierarchy group, generate the missing components inside the virtual hierarchy group. In addition, preserve the area boundary of the instances inside the virtual hierarchy group and generate soft blocks for schematic components that are missing.

Update Components And Nets Form

Generate Chained Devices    

Generate device chains for instances inside a virtual hierarchy when the selection is made at the top level.

Generate Chained Devices Form

Interactive Chaining    

Chain devices inside a virtual hierarchy when the device selection is made from the top level.

Chaining Devices Interactively

Interactive Abutment    

Abut devices inside a virtual hierarchy when the devices are selected from the top level.

Device Abutment

Interactive Folding    

Fold devices inside a virtual hierarchy when the device selection is made from the top level.

Transistor Folding
