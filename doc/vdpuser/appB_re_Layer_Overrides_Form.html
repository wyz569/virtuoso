Layer Overrides Form

The following table describes the fields available on the Layer Overrides form.

Field Description

Direction

Displays the information of valid routing layers with the routing direction.

Layers

Lists the valid routing layers.

Direction

Displays the preferred routing direction for the routing layer. The supported routing directions are: Horizontal , Vertical , and None .

Range

Lets you specify the valid routing layer range.

Top

Specifies the top valid routing layer limits and overrides the validLayers setting in the selected constraint group.

Bottom

Specifies the bottom valid routing layer limits and overrides the validLayers setting in the selected constraint group.
