Bias Layers Form

The following table describes the fields available on the Bias Layers form.

Field Description

Specify each Bias Area

Lists the name of the global bias area or path.

Layers

Lets you select routing layers for the selected global bias are or path.

All Layers

Specifies that the selected global bias area or path must be completed on all routing layers.

Specify

Lets you specify the routing layers to which the selected global bias area or path should be constrained.
