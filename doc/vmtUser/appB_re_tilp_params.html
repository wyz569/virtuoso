Edit Instance Properties Form (Die/Package TILP Parameters)

Use the Parameter tab of the Edit Instance Properties form to specify the die or package TILP parameters.

Field Description

Parameter

This section provides options to customize die or package TILP parameters.

Angle

Specifies the placement angle of a die. The following translation table defines the angle to be set based on instance orientation:

The following list defines the TILP angle to be set based on instance orientation:

  • oacR0 - 0
  • oacR90 - 90
  • oacR180 - 180
  • oacR270 - 270
  • oacMY - 0
  • oacMYR90 - 0
  • oacMX - 0
  • oacMXR90 - 0

The orientation of the layout instance during GFS is set based on the lxGenerationOrientation environment variable, which is, by default, set to R0, so the TILP angle is always set to 90.

When the environment variable is set to preserve, the layout instance orientation takes the value from schematic instance and sets accordingly as mentioned in the table.

Mirrored

Specifies if the die is mirrored to its original position with respect to the package. A mirrored die is placed under the package.

Stackup Offset/Package Stackup Offset

Specifies the distance from the die/package TILP stack.

Flipped/Geometric Mirror

Specifies whether the die is flipped before placing.

Die Stack Order/Package Stack Order

Specifies the die/package TILP stack order. The position of die or package from the substrate.

Shrink Factor

Specifies the ratio by which the die is shrunk optically while placing on the package. It is indicated as 1, which is equivalent to 100%. Therefore, if a die is expanded, the value could be greater than 1 (100%).

X Thermal Shrink Factor (%)

Specifies the percentage by which a die is shrunk thermally on the x-axis.

Y Thermal Shrink Factor (%)

Specifies the percentage by which a die is shrunk thermally on the y-axis.

Tilp Version

Specifies the version of a TILP indicating the parameters supported.

Target Library Name

Specifies the name of the library where a TILP has been created. This field is auto-populated but can be modified.

Target Cell Name

Specifies the name of the cell where a TILP has been created. This field is auto-populated but can be modified.

footprint Library Name

Specifies the name of the library where a footprint has been created. This field is auto-populated but can be modified.

footprint Cell Name

Specifies the name of the cell where a footprint has been created. This field is auto-populated but can be modified.

footprint View Name/Base Cell View Name

Specifies the name of the abstract view.

footprint ALT Cell Name

Specifies the alternate name of a footprint cell. It is specified if several footprints are available for the same cell.

Scribe North

Specifies an area around the PR boundary by using the scribe parameters. If a die is shrunk, the scribe area is also shrunk, but the spaces around the PR boundary always honor the scribe parameter values. An area boundary is created in the die instance sub-master that identifies the scribe area. If the scribe parameters are equal to 0, the area boundary has the same dimensions as the PR boundary. To enforce the scribing effect, Scribe North is used to define values in the north direction.

Scribe South

Specifies the scribe values in the south direction.

Scribe East

Specifies the scribe values in the east direction.

Scribe West

Specifies the scribe values in the west direction.

Thickness

Indicates the thickness value of a scribe. It is a CDP parameter but not a TILP parameter. Therefore, it has no effect on the TILP evaluation results.

Model Name

Refers to the S-parameter model name.

Parameters to Display

Filters the list of parameters displayed in the Edit Instance Properties form.

Select Parts

Specifies the part of a specific configuration that you want to use in the layout. For instance, if your cell is a CAPACITOR, you can choose a specific value capacitor and subsequently, the layout is updated after GFS.
