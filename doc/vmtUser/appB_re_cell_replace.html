Cell Replacement Form

Lets you choose the source cell and define the mapping of the pins between the current and the replacement cell.

Field Description

Source Cell

The name of the cell to be replaced.

Replacement Library

The library name of the target symbol view.

Replacement Cell

The cell name of the target symbol view. When specified, the Unmapped Pins section is automatically populated.

Replacement View

The name of the target symbol view.

Mapping Mode

Specifies the method of mappings pins. The default value is Auto.

Auto: Runs all mapping modes and selects the mode in which highest number of pins match.

Symbol Pin Name: Maps the symbol pin names of the source with the symbol pin name of the replacement cell.​​

Footprint Pin Name: Maps the footprint pin name of the source with the footprint pin name of the replacement cell. In this mode, the layout information is used to drive the connectivity at schematic level.​

Footprint Pin Position: Maps the footprint pin position of the source with the footprint pin position of the replacement cell. An additional option, Use flipped footprint, is available in this mode. When selected, one of the abstracts or footprints is flipped to do the matching. This flipping is only for computation and not an actual footprint change.

Best Pin Name Match: Maps the symbol pin names using the prefix or suffix of the source with the symbol pin names of the replacement cell.

Unmapped Pins

This section provides options to specify the new schematic to be created by the enablement flow.

Source Pins

Lists the source pins that cannot be mapped. Clicking Set as Ignored indicates that it has been concluded after a review that the pin cannot be mapped and it will be ignored.

Target Pins

Lists the replacement cell pins that cannot be mapped. Clicking Set as Unconnected indicates that it has been concluded after a review that the pin cannot be mapped and it will be left unconnected.

Mapped Pins

This section shows the mapped pins of source and replacement cells. It is populated automatically when the Mapping Mode is selected. You can select a row to remove the mapping.
