Bump Connectivity Assignment Form

The following table describes the fields available in the Bump Connectivity Assignment form.

Field Description

Auto Assignment

Enables automatic bump connectivity assignments. This option is selected by default and cannot be edited.

Bumps

Specifies the bumps for which connectivity is to be assigned. The available options are All and Selected. In the Selected mode, select the required bumps in the layout canvas.

Nets

Specifies the nets to which bumps can be assigned. The available options are All and Selected. In the Selected mode, select the required nets either in the layout canvas or in the Navigator assistant.
