Import Die Text File Form

Use the Import Die Text File form to specify the parameters for importing the die text file.

Field Description

Die Text File

Specifies the name of the die text file to be imported in Layout MXL.

Delimiter

Specifies the delimiter options, which function as separators, from the drop-down list box.

Pad Width

Specifies the width of the pad in the unit of measurement that you set for the padstack.
