Cross Fabric Check Violation Summary and Navigation Form

Use the Cross Fabric Check Violation Summary and Navigation form to compare connectivity across fabrics and display results in a system check report.

Field Description

CellView

Lists the cellviews in which violations are reported.

Fabric Type

Specifies the fabric type of the cellviews.

Violation For Connectivity Extraction

Lists the number of violations reported during connectivity extraction.

Violation For Check Against Source

Lists the number of violations reported by the Check Against Source (CAS) checker.

Violation For Layout Vs Abstract

Lists the number of violations reported by the Layout vs Abstract (LVA) checker.
