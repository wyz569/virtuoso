Create Guides Form

Use the Create Guides form to create line-shaped or arc-shaped guides around dies.

Field Description

Die Side

Specifies the slide along which guides must be created. Options are: All, North, East, West, and South.

Distance

Specifies the distance from the reference at which the guides must be created.

From edge to the die boundary

Specifies that the distance is to be calculated from the PR boundary of the die boundary.

From edge to the die outline

Specifies that the distance is to be calculated from the die outline.

Radius

Lets you create arc-shaped guides.

Guide Vertices offset (+/-)

Specifies the distance of the diagonal line drawn from the two vertices of the guide to the center of the die.
