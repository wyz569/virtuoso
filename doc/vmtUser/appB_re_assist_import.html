Allegro Design Layout Importer Form

Updates the package layout from the specified SiP file through an imported library.

Field Description

File Name

The name of a SiP file that is used to update the package layout.

Layout Reference

This section provides options to specify package layout that will be updated.

Library Name

The library name of a Virtuoso layout to be compared with a SiP file.

Cell Name

The cell name of a Virtuoso layout to be compared with a SiP file.

View Name

The view name of a Virtuoso layout to be compared with a SiP file.

Imported Library

This section provides an option to specify a staging library to be used to perform various checks.

Library Name

The name of the library created for importing the updates from a SiP file to a package layout.
