Product Version IC23.1, November 2023

Create MultiTech Schematic Form

Creates a schematic view based on the connectivity information defined in a SiP file.

Field Description

Allegro Layout

The SiP file to be used for creating a schematic.

Setup Map Table

The action to fill the Library Map Table by using the information from the SiP file.

Map File

The name of the file that was created in a previous mapping run in the current or another cellview. It contains the mapping information, which can be reused while creating a schematic.

Available Libraries

The list of available unified libraries from which you can select the libraries to be used for mapping.

Mapping Libraries

The list of selected unified libraries to map the instances in the SiP file with the cells and views in the libraries.

Library Map Table

The table that gets populated with the mapping information from the mapped libraries, map file, or can be edited manually to add information.

Add and Remove buttons

The buttons to add or remove libraries from the Mapping Libraries list.

Up and Down arrow buttons

The buttons to order libraries in the Mapping Libraries list.
