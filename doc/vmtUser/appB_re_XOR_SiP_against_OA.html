XOR SiP against OA Form

Compares a SiP file against an OpenAccess layout.

Field Description

Allegro Input File

The name of a SiP file for the XOR operation.

Layout Reference

This section provides options to specify the layout for the XOR operation.

Library Name

The library name of a Virtuoso layout to be compared with a SiP file.

Cell Name

The cell name of a Virtuoso layout to be compared with a SiP file.

View Name

The view name of a Virtuoso layout to be compared with a SiP file.

Output XOR location

This section provides options to specify the new layout created after the XOR operation.

XOR Library Name

The library name of the new layout to be generated.

XOR Cell Name

The cell name of the new layout that will be generated.

XOR View Name

The view name of the new layout that will be generated.

Options

This section provides options to customize the XOR operation.

Filter XOR shapes smaller than:

The size of a window used to remove small shapes from the generated layout. Shapes smaller than the specified aperture are not visible in the generated layout.

Compare Static Shapes

Creates static shapes from the Virtuoso layout at runtime and compares them with the shapes specified in the SiP file.

Compare Conducting Layers Only

Limits the comparison to only the shapes on conducting layers, that is, layers defined in a cross-section.

Generate True Marker Shapes

Uses the XOR-operation-generated shapes in a Virtuoso layout as markers.

Open Destination Design When Done

Opens the newly generated layout when the XOR operation is completed.
