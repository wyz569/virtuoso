Convert Allegro Libraries to Unified Libraries Form

Converts Allegro libraries into multi-platform unified library components that can be used across the platforms.

Field Description

Library

The name of the unified library.

Part File

The name of the part file that contains information of the symbols in the Allegro platform.

Run directory

The name of the directory where conversion happens.
