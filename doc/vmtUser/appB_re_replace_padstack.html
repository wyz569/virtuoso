Pad Stack Replacement Form

Replaces a padstack cell of a footprint by another padstack cell, which is from the current footprint library or another library.

Field Description

Footprint

Specifies the footprint cellview of the selected TILP instance.

Source PadStack

Specifies the padstack cell to be replaced in the current footprint.

Target PadStack

Specifies the target padstack cell based on the following options:

  • From Current Footprint Library: (Default) Uses a padstack cell from the current library.
  • Copy From Another Library: Copies the padstack cell from another library.

Drop-down List enabled when the Copy From Another Library option is selected

Lists the libraries available to choose from when the Copy From Another Library option is selected.

Second drop-down list

Lists the padstack cells available in the current footprint library or in the selected target library when the Copy From Another Library option is selected.
