Layer Stack Editor Form

Use the Layer Stack Editor form to specify the options for layers. Whenever you add or delete any layer, a confirmation dialog box is displayed.

Field Description

Thickness

Specifies the default material thickness.

Material

Specifies the name of the material used for the layer.

Conductivity[S/m]

Specifies the electrical conductivity of the material. When the value is 1000 mho/cm or less, the material is considered a dielectric, otherwise it is a conductor.

Permittivity

Specifies the permittivity of the material. It is a measure of capacitance that is encountered when forming an electric field in a particular material.

Loss Tangent

Specifies the loss tangent of the material. Loss tangent refers to the dielectric material's inherent dissipation of electromagnetic energy.

Lock

This section provides options to prevent adding or removing layers or attributes in the form.

Add Layers

When selected, prevents adding or removing layers but modifying layer attributes can be done. The Add Layer, Delete Top, and Delete Bottom buttons are disabled in the form.

Values Change

When selected, prevents modifying layer attributes but adding or removing layers can be done. The text fields to modify the attributes in the form are disabled.
