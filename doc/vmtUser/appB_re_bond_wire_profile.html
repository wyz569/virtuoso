Set Bond Wire Profile Form

Use the Set Bond Wire Profile form to apply bond wire profiles to existing bond wires to update their structures and properties.

Field Description

Bondwire Profile

Lists all bond wire profiles that are defined in the technology file.

Honor Constraints

Honors the constraints that are listed below.

Direction

Direction of the bond wire. The valid values are:

  • forward: The bond wire connection begins at the IO pad and ends at the bond finger.
  • reverse: The bond wire connection begins at the bond finger and ends at the IO pad.

Material

Material of the bond wire. The valid values are GOLD and COPPER.

Diameter

Diameter of the bond wire.

Point #

Point of curvature of the bond wire. A bond wire has one or more points that divide it into steps. Each point marks the start of a step. Each step has a horizontal section and a vertical section.

Horizontal

Attribute that defines the horizontal segment. The valid values are: Length, Percentage, and Angle.

Value

Value of the length, percentage, or angle of the horizontal segment.

Vertical

Attribute that defines the vertical segment. The valid values are: Length, Percentage, and Angle.

Value

Value of the length, percentage, or angle of the vertical segment.
