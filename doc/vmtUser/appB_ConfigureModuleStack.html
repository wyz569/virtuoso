Configure Module Stack Form

The following table describes the fields available in the Configure Module Stack form.

Field Description

Module

Lists lists all the modules or dies that are part of the current design.

Flipped

Specifies whether each die is to be flipped.

Order

Specifies the order in which the die are to be placed in the stack.

Mirrored

Specifies the orientation of each die.
