Select Row Template Form

The following table describes the fields available in the Select Row Template form.

Field Description

Library

Specifies the library name for the row template.

Cell

Specifies the cell name for the row template.

View

Specifies the view name for the row template.

Select Row Template

Lists the existing row templates.

Details

Provides the property details of the selected row template.
