Add Supply Nets Form

Lets you specify the supply nets for the Power Manager setup.

Field Name Description

Filter

Filters the list of supply nets to limit the search scope.

Add Nets

Adds the user-specified supply nets.

Net Type

Specifies the type of the supply or ground net.

Voltage

Specifies the voltage value of the supply nets.

External Switch Type

Specifies whether the supply net is externally switchable.
