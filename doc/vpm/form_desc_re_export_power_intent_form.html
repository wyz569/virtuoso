Export Power Intent Form

Imports the power intent to the currently open cellview from the specified 1801 file.

Field Name Description

Library

Specifies the library of the cellview from which the power intent is exported.

Cell

Specifies the cell of the cellview from which the power intent is exported.

View

Specifies the view of the cellview from which the power intent is exported.

1801 File

Specifies the name of the file for exporting the 1801 power model.

The file path from the last session is retained.

Overwrite Existing File

Select the check box if you want the latest changes to be saved in the same 1801 file.
