wireOffset

layout wireOffset float wireOffsetValue

Description

Allows you to create a wire a specified distance away from the cursor. The default is 0.0 user units. Value should be a positive float number.

GUI Equivalent

Command

Create – Wiring – Wire

Field

Offset

Examples

envGetVal("layout" "wireOffset")
envSetVal("layout" "wireOffset" 'float 1.0)
