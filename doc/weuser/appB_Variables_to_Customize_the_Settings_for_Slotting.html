Variables to Customize the Settings for Slotting

The environment variables listed below are slt environment variables that provide a way to predefine some of the parameters prior to running the slotting SKILL command.

  • If an environment variable has an invalid value (negative or out of range), the default behavior applies.
  • The value specified in the environment variable can be overridden if the associated SKILL argument is specified in the command line.
