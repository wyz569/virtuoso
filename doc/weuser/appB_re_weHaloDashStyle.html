weHaloDashStyle 

we weHaloDashStyle boolean { t | nil }

Description

Creates a dashed halo around a wire, bus, or stranded wire while creating or editing an object that violates a spacing constraint. The default is nil.

GUI Equivalent

Command

Options – Editor

Form Field

Dashed

Examples

envGetVal("we" "weHaloDashStyle")
envSetVal("we" "weHaloDashStyle" 'boolean t)
