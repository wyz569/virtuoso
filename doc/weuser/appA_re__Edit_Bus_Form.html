Edit Bus Form

Use the Edit Bus command to interactively change the width and spacing gap for the selected bus bits.

Field Description

Select

Net Name

Displays the net names of the selected bus bits. The net name corresponding to each selected bus bit appears in the bus bit table.

Bus Bit Table

This table is automatically populated with the selected bus bits. It lets you update the width and spacing for the bus bits.

Net

Name of the net corresponding to the bus bit. The name of bus bit can be a single net name or the bit-range names.

Width

Minimum width of the bus bit. The default value of the bus bit width is the constraint look-up minimum width. You can change the width of a bus bit.

Gap

Minimum spacing between two adjacent bus bits. The default value  is the constraint look-up minimum spacing. You can also modify the spacing between two bus bits.

All Nets

Lets you specify a value to override the width and spacing values for all the bus bits in the form. A warning message is displayed in the CIW if the value specified is less than the default look-up value.

Via

This section lets you specify the settings for all vias in a layout design.

Adjust Edited Vias Parameters

Calculates via parameters (such as alignment or enclosures) if you stretch the wire connected to the via or while editing width and layer properties of a pathSeg. By default the option is deselected.

minNumCuts

Specifies the via cuts as minimum number of cuts for creating and editing wires. If the override value is less than either the technology rule or the current constraint group on the net or on the design, then the value specified in the Edit Bus form is reset to the minimum value.

Row/Col

Specifies a via cuts as the number of rows or number of columns. The increase and decrease in the number of via rows and columns is dependent on the minimum number of via cuts. 

number of via rows * number of via columns >= minimum number of via cuts.
