enableSlotSpecCell

slt enableSlotSpecCell boolean { t | nil }

Description

Uses the slotSpecCell environment variable to override the default slotting parameters. The default is t. When set to t, it enables the Specification Cell field. Otherwise, the Specification Cell field is disabled.

GUI Equivalent

Command

Window – Toolbar – Metal Density

Options – Slot Geometry

Field

Check box next to the Specification Cell field

Examples

envGetVal("slt" "enableSlotSpecCell")
envSetVal("slt" "enableSlotSpecCell" 'boolean nil)
