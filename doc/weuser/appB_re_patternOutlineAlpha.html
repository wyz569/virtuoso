patternOutlineAlpha

we patternOutlineAlpha int patternOutlineValue

Description

Determines the outlineAlpha parameter given to the hiliteSet used for WSP dynamic display (see geSetHilightSetAlphaBlendingParameters). The default is 600.

GUI Equivalent

None

Examples

envGetVal("we" "patternOutlineAlpha")
envSetVal("we" "patternOutlineAlpha" 'int 50)
