slotToEdge

slt slotToEdge float slotToEdgeValue

Description

Defines the minimum distance from the slot to the edge. The default is -0.1µm.

GUI Equivalent

Command

Window – Toolbar – Metal Density

Options – Slot Geometry

Field

Distance to Edge

Examples

envGetVal("slt" "slotToEdge")
envSetVal("slt" "slotToEdge" 'float 1.0)
