addFilletOnPadWithoutDrills 

slt addFilletOnPadWithoutDrills boolean { t | nil }

Description

Creates fillets on pads without drills. The default is nil.

GUI Equivalent

Command

Window – Toolbar – Metal Density

Options – Fillet

Field

Objects – Pad without drills

Examples

envGetVal("slt" "addFilletOnPadWithoutDrills")
envSetVal("slt" "addFilletOnPadWithoutDrills" 'boolean t)
