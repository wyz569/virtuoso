useOverlapRegionForVia

layout useOverlapRegionForVia boolean { t | nil }

Description

Creates square vias to connect collinear wires. The default is t.

GUI Equivalent

Command

Route – Design Setup – Routability Check

Window – Assistant – Wire Assistant – Automatic – Optimization

Window – Assistant – Wire Assistant – Net – Via Configuration

Field

Maximize Cuts

Examples

envGetVal("layout" "useOverlapRegionForVia")
envSetVal("layout" "useOverlapRegionForVia" 'boolean nil)
