wSpacing

slt wSpacing float wSpacingValue

Description

Defines the slot spacing along the width. The default is -0.1µm.

GUI Equivalent

Command

Window – Toolbar – Metal Density

Options – Slot Geometry

Field

Width Spacing

Examples

envGetVal("slt" "wSpacing")
envSetVal("slt" "wSpacing" 'float 1.0)
