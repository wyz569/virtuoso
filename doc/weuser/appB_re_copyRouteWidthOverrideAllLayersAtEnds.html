copyRouteWidthOverrideAllLayersAtEnds

we copyRouteWidthOverrideAllLayersAtEnds float overrideValue

Description

Defines the width of copy routes for connected pin sections.

The default is 0.0.

GUI Equivalent

Command

Window – Assistant – Routing – CopyRoute

Edit – Wiring – Copy Route

Field

Width – Value (First/last sections)

General – Width At Ends

Examples

envGetVal("we" "copyRouteWidthOverrideAllLayersAtEnds")
envSetVal("we" "copyRouteWidthOverrideAllLayersAtEnds" 'float 0.5)
