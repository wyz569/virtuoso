copyRouteClearPreviewWhenEsc

we copyRouteClearPreviewWhenEsc boolean { t | nil }

Description

Clears the route preview when the copy route command is canceled.

The default is t.

GUI Equivalent

Command

Window – Assistant – Routing – CopyRoute

Field

Preview – Clear preview when cmd cancelled

Example

envGetVal("we" "copyRouteClearPreviewWhenEsc")
envSetVal("we" "copyRouteClearPreviewWhenEsc" 'boolean nil)
