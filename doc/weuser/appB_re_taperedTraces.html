taperedTraces

slt taperedTraces boolean { t | nil }

Description

Creates fillets on tapered traces. The default is t.

GUI Equivalent

Command

Window – Toolbar – Metal Density

Options – Fillet

Field

Tapered Traces – Tapered traces

Examples

envGetVal("slt" "taperedTraces")
envSetVal("slt" "taperedTraces" 'boolean nil)
