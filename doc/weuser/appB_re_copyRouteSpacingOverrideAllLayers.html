copyRouteSpacingOverrideAllLayers

we copyRouteSpacingOverrideAllLayers float overrideValue

Description

Defines the spacing value to be used for copy route.

The default is 0.0.

GUI Equivalent

Command

Window – Assistant – Routing – CopyRoute

Field

Spacing – Value (intermediate sections)

Examples

envGetVal("we" "copyRouteSpacingOverrideAllLayers")
envSetVal("we" "copyRouteSpacingOverrideAllLayers" 'float 0.05)
