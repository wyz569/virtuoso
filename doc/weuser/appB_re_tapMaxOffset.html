tapMaxOffset 

slt tapMaxOffset float tapMaxOffsetValue

Description

Specifies the maximum distance the fillet may extend from the vertex to the point of cline width transition. The default is 635 µm.

GUI Equivalent

Command

Window – Toolbar – Metal Density

Options – Fillet

Field

Tapered Traces – Max offset

Examples

envGetVal("slt" "tapMaxOffset")
envSetVal("slt" "tapMaxOffset" 'float 500)
