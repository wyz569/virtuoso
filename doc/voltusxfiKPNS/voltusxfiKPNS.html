Voltus-XFi Custom Power Integrity Solution Known Problems and Solutions
Product Version IC23.1, November 2023

Known Problems and Solutions

This document describes the known issues with Voltus-XFi Custom Power Integrity Solution and suggests the workarounds for these issues. Each issue is identified by a Cadence Change Request (CCR) number.

CCR 2818835: Result Browser issues with Voltus-XFi

Description: The Voltus-XFI Result Browser is showing incorrect or invalid values in the following columns for metal resistors:

  • No. of Cuts
  • Cut Area
  • Required Number of Cut Values

Solution: Currently, a workaround is not available. This will be fixed in a later release.
