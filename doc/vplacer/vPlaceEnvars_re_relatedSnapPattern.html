relatedSnapPattern 

layoutXL.APAssist relatedSnapPattern string "pattern_name"

Description

Specifies the snap pattern of the WSP region created in addition to the row region.

The default is "", in which case no WSP region is created.

GUI Equivalent

Command

Place – Row Template Manager – Edit

Field

Related Snap Pattern

Examples

envGetVal("layoutXL.APAssist" "relatedSnapPattern")
envSetVal("layoutXL.APAssist" "relatedSnapPattern" 'string "pattern_1")
