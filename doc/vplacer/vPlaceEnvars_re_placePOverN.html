placePOverN

layoutXL.AP placePOverN boolean { t | nil }

Description

Places all P-devices to the top and N-devices at the bottom of the selected region.

The default is nil.

GUI Equivalent

Command

Place – Placement Options

Field

Automatic – Devices – Group P over N

Examples

envGetVal("layoutXL.AP" "placePOverN")
envSetVal("layoutXL.AP" "placePOverN" 'boolean t)
