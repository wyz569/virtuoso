horizontalOffset 

layoutXL.APAssist horizontalOffset float horizontal_Offset

Description

Specifies the reference grids along the x axis for snapping the devices during placement.

The default is 0.

GUI Equivalent

Command

Place – Row Template Manager – Edit

Field

Placement Grid – Horizontal Offset

Examples

envGetVal("layoutXL.APAssist" "horizontalOffset")
envSetVal("layoutXL.APAssist" "horizontalOffset" float 3)
