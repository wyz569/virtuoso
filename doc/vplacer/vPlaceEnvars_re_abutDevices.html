abutDevices

layoutXL.AP abutDevices boolean { t | nil }

Description

Automatically abuts devices during placement.

abutDevices cannot be used when Preserve Abutment on the Automatic tab of the Placement Options form is selected.

The default is nil.

GUI Equivalent

Command

Place – Placement Options

Field

Automatic – Devices – Abut Devices

Examples

envGetVal("layoutXL.AP" "abutDevices")
envSetVal("layoutXL.AP" "abutDevices" 'boolean t)
