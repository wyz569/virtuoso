Save a Copy Form

Use the Save a Copy form to save a copy of the currently open design.

Field Description

Name

Shows the name of the cell. The other columns in the list box show the current library, cell, and view in which the cell exists.

The following three fields let you specify a different name and location to save the cellview:

  • The first cyclic field lists the libraries and you can choose the one in which you want to save the design. By default, the library of the current design is selected.
    Environment variable: saveACopyLibName
  • The second text field lets you specify a different name for the cell. If you retain the name and also do not change the library and view locations, the existing cellview will be overridden.
  • The third and the last cyclic field lets you select the view type for the cell you are saving.

Open Saved Copy

Opens the saved copy of the design after you create it. The Open for and Open in fields are enabled when this option is selected.

Environment variable: openSavedCellViewInWindow

Open for

Specifies whether to open the saved copy of the design for reading or editing.

Open in

Specifies whether the saved copy of the design is opened in a new tab, the current tab, or a new window.

Open Connectivity Reference

Specifies whether the schematic view for connectivity reference is opened when you open the saved copy of the design. This setting overrides the lxSchematicDefaultApp environment variable and the Connectivity form.
