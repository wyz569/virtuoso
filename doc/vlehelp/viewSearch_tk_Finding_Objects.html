Finding Objects

To find objects:

  1. In the layout window, choose Tools – Find/Replace. Alternatively, press Shift+S.
    The Find/Replace form appears.
  2. Select the type of object you want to find from the Search for list.
  3. Select the location where you want to perform the search by choosing from the Search Hierarchy Range list. By default, the search is performed in the current edit cellview in the layout window.
  4. Click Add Criteria to add boolean expressions, which control the search criteria.
  5. Set the search criteria:
    1. From the first list, choose the object property that you want to search.
    2. From the boolean operator list, choose the operator you want to use, such as equal (==) or not equal (!=).
    3. Type or choose the value that you want to search.
      • Click Delete at the end of the row if you want to delete a row of search criteria.
      • Click Add Criteria again if you want to add more search criteria.
  6. Click Find.
    All the objects matching the specified search criteria are highlighted on the canvas and are put in a search group. The first or current object in the group is highlighted in a different color.
  7. Optionally, click Previous or Next to search through the group.
  8. Optionally, click Add Select to select the current object in the search group.
  9. Optionally, click Select All to select all the highlighted objects.
  10. Click Close when you are done selecting objects.
