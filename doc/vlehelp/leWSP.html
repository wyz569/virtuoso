Width Spacing Patterns in Layout Editor

A width spacing pattern (WSP) definition is a sequence of width and spacing values that define the track patterns to which the shapes in a design are aligned. WSPs are an advanced form of snap pattern definitions and can have multiple width and space combinations. The width spacing pattern definitions used with widthSpacingSnapPatternDefs are stored either in the technology database or the design database.

The figure below illustrates that each width spacing pattern is a sequence of width-space pairs. Patterns are specified as width and space pairs. Groups of width and space pairs can be repeated. The first track of the pattern is anchored to the period grid using an offset.
