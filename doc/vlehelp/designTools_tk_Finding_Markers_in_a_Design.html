Finding Markers in a Design

Use the Verify – Markers – Find command to search for markers in a design. You can search for markers anywhere in a design hierarchy.

To find markers in a design:

  1. In the layout window, choose Verify – Markers – Find.
    The Find Marker form appears.
  2. Change the options on the form as required. You can narrow down your search or expand the scope.
  3. Click Apply.
    The first error or warning marker is highlighted and the explanation for the marker is displayed in the marker text window.
  4. To find the next marker, click Next.
    The marker text window displays the reason for the marker.
    Use the Next and Previous buttons to navigate through the markers.

Schematic editor provides a similar Find Marker command. The command options are a little different than the layout editor version and provide additional functionality. You can run the schematic editor Find Marker command in a layout window by using the geHiCommonFindMarker SKILL function.
