Setting Filters for a Customized Report

You use the Pad Opening Info: Filter form to filter the data shown in a customized report.

To filter the data for a customized report:

  1. Open the Create Encore Report form from the Pad Opening Info form.
  2. Click Set Filter.
    The Pad Opening Info: Filter form is displayed.
  3. To enable a filtering criterion, select a check box, and from the corresponding list, select the mode as Pad Opening Name, X, Y, or Area.
  4. Choose the appropriate operator from the second list.
    For example, to specify the criterion as equal to the content in the left field, select the == operator. The other valid operators in Pad Opening Name are !=, contains, and doesn’t contain. For X, Y, and Area, the valid operators are ==, !=, <, <=, >, and >=.
  5. Specify the required values in the input field.
  6. From the last list for a criterion, select the operator for logical operation with the subsequent filters defined. The valid operators for logical operation are and and or.
  7. Click OK.
    The filters are set for the customized report and displayed in the Filter field in the Create Encore Report form.
