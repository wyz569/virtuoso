viaCheckMinNumCut

layout viaCheckMinNumCut boolean { t | nil }

Description

Determines whether the minNumCut constraint for the cut layers specified in the technology file is ignored while creating a via.

The default is t.

GUI Equivalent

None

Examples

envGetVal("layout" "viaCheckMinNumCut")
envSetVal("layout" "viaCheckMinNumCut" 'boolean nil)
