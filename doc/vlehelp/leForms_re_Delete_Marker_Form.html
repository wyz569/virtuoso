Delete Marker Form

Use the Delete Marker form to delete individual markers in the design window.

Field Description

Options

This section enables you to specify whether the cellview is to be opened in edit or read mode.

Cellview Access Mode

Specifies the mode for opening a cellview.

  • read: Deletes the selected markers and re-opens the cellviews in edit mode with no dialog box.
  • write: Deletes all the markers after opening the cellview in edit mode. In this mode, you can choose to select the Confirm Access Mode Change check box and specify the cellviews to be opened in read mode.

Environment variable: markerDeleteAccessModeIsEdit

Confirm Access Mode Change

Opens the Delete Marker dialog box for you to confirm whether you want to open the selected cellview in edit mode. Markers are not deleted in cellviews that stay in read mode. This check box is enabled only if Cellview Access Mode to set to edit.

Environment variable: markerDeleteConfirmModeChange
