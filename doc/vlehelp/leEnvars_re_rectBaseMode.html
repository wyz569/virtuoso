rectBaseMode

layout rectBaseMode cyclic { "Bottom left" | "Bottom right" | "Top left" | "Top right" | "Center" }

Description

Specifies the base creation point of rectangle that you create: Bottom left, Bottom right, Top left, Top right, Center.

The default is Bottom left.

GUI Equivalent

Command:

Create – Shape – Rectangle

Field:

Justification

Examples

envGetVal("layout" "rectBaseMode")
envSetVal("layout" "rectBaseMode" 'cyclic "Top right")
envSetVal("layout" "rectBaseMode" 'cyclic "Center")
