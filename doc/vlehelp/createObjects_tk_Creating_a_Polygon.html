Creating a Polygon

To create a polygon:

  1. In the layout window, from the Layers panel, select the layer on which you want to draw a polygon.
  2. Choose Create – Shape – Polygon.
  3. Press F3 to open the Create Polygon form.
  4. Select a snap mode from the Snap Mode list to specify how the pointer snaps when you create a polygon.
    The following figure illustrates snap mode settings. If you chose an L90 mode, you can switch to the other L90 mode by clicking the middle mouse button. Keep clicking the middle mouse button to toggle between L90XFirst and L90YFirst.
  5. In Net Name, type the name of the net with which you want to associate the polygon.
  6. Select Create as ROD object if you want to create a ROD object polygon.
    The Name field becomes editable, displaying the default name assigned to the polygon. Change the default name if required.
  7. Click on the canvas to enter the points. As you enter each point, a dotted line shows how the layout editor will close the polygon if you stop at that point.
    Alternatively, you can specify the points of the polygon using the Enter Points form.
  8. Double-click on the canvas to finish drawing the polygon.
