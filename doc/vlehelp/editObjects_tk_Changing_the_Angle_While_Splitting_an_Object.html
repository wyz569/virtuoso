Changing the Angle While Splitting an Object

As you stretch objects, you can change the angle of the segments.

To stretch path segments to any angle:

  1. In the layout window, choose Edit – Advanced – Split.
  2. Press F3 to open the Split form.
  3. Deselect the Lock angles check box.
  4. Ensure the Snap Mode is set to anyAngle or diagonal to change the angle of the segment.
  5. Create a cut line through the segment.
  6. Stretch the highlighted path segment to the angle you want.

You can add a jog to several paths (a bus) so that it wraps around an object by splitting a section of the paths and stretching the split section.
