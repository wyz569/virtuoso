Stretching an Object by Direct Manipulation

You can stretch objects by selecting and dragging their edges or corners.

To stretch an object:

  1. Choose Edit – Stretch or click the Stretch icon on the Edit toolbar.
    Alternatively, right-click any free space on the design canvas to display the Layout shortcut menu and select Stretch.
  2. Select one or more edges or corners in partial selection mode as follows:
    In partial selection mode, if you press Shift and click an edge or corner, that edge or corner is the reference point for the stretch.
    If you create a selection box around the edges, Stretch prompts you for a reference point. The reference point does not have to be on the selected shapes.
  3. Move the mouse pointer over the edge until you see the stretch pointer:
  4. Click and drag the mouse pointer to stretch the object and then release the mouse button.

You cannot use Stretch to delete objects. If you try to stretch an object to a zero area, a warning appears in the CIW and a dialog box opens stating that the shape could not be modified because the operation is invalid.
