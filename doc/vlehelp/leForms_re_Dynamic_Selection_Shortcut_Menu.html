Dynamic Selection Shortcut Menu

The shortcut menu of the Dynamic Selection assistant contains the following items.

Command

Description

Open Instance (<mode>/<tab>)

Opens the master cellview of the instance currently under the mouse pointer with the mode specified in the command name.

For example, Open (edit/new tab) opens the cellview in edit mode in a new tab; Open (read/current tab) opens the cellview in read mode in the current tab.

You specify the default behavior for this command in the

Dynamic Selection Assistant Options form.

This command is available only if the mouse pointer is over an instance.

To open a different view of the current cell, use the Open Cellview command instead.

Open Cellview

Displays the Open form where you can specify:

  • Which view of the current cell you want to open
  • Whether that view is opened in edit or read mode
  • Whether it is opened in a new tab, the current tab, or in a new window.

This command is available only if the mouse pointer is over an instance.

Edit In Place

Enables you to edit in place the currently selected cellview in the layout window.

This command is available only if the mouse pointer is over an instance, a group, or an object contained in a hierarchy.

Expand All Below

Expands the hierarchy to display the hierarchical objects under the current node.

This command is available only for hierarchical objects. The hierarchy is expanded down to the display level set in the Display Options form. So if, for example, the display level is set to 0, the hierarchy cannot be expanded. The assistant does not necessarily list all the objects contained in a particular hierarchical object, only those that were under the pointer when the display was frozen.

Collapse All Below

Collapses the hierarchy to hide the hierarchical objects under the currently selected node.

This command is available only for hierarchical objects.

Options

Opens the Dynamic Selection Assistant Options form where you can specify:

  • How selected objects are displayed in the assistant
  • Default open mode

Freeze DSA/UnFreeze DSA

Freezes or unfreezes the display in the assistant.

Settings

Enables you to save and load your display preferences for the Dynamic Selection assistant. This ensures that the order of fields that you choose for the assistant and the show or hide states are retained even after you close the current Virtuoso session,
