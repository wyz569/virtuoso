Copying an Object to Another Cellview

You can copy objects from one cellview to another interactively or by specifying spacing mode and values in post- or pre-selection modes.

  • Copying an Object Interactively
  • Copying an Object Using Spacing Values in Post-Selection Mode
  • Copying an Object Using Spacing Values in Pre-Selection Mode

Copying an Object Interactively

To copy objects interactively to another open cellview:

  1. In the source cellview, select the objects you want to copy.
  2. Choose Edit – Copy.
  3. Move the pointer into the other open target cellview.
  4. Click on the canvas to place the copies.

You cannot use the Array fields in the Copy form to create and copy an array of copied objects from one cellview to another. Only the original objects you selected are copied to the next cellview even if you have specified values in the Rows and Columns fields.

Copying an Object Using Spacing Values in Post-Selection Mode

To copy objects to another open cellview in post-selection mode by specifying spacing values:

  1. In the source cellview, choose Edit – Copy to invoke the Copy command.
  2. Select the object on the canvas that you want to copy.
  3. Move the pointer into the target cellview.
  4. Press F3 to open the Copy form.
  5. Select one of the following modes in the Spacing section and specify the spacing values:
    • Pitch: Select the Pitch radio button and specify the spacing in the Delta X,Y fields.
    • Spacing: Select the Spacing radio button and specify the spacing in the Spacing X,Y fields.
    • Absolute: Select the Absolute radio button and specify the spacing in the X,Y fields.
  6. Click Apply in the Spacing section.
    The object is placed at the specified location in the target cellview.

Copying an Object Using Spacing Values in Pre-Selection Mode

To copy objects to another open cellview in pre-selection mode by specifying spacing values:

  1. In the source cellview, select the object on the canvas that you want to copy.
  2. Choose Edit – Copy to invoke the Copy command.
  3. Click on the canvas to specify the reference point for the copy operation.
  4. Move the pointer into the target cellview.
  5. Press F3 to open the Copy form.
  6. Specify the spacing mode and values in the Spacing section.
  7. Click Apply in the Spacing section.

The object is placed at the specified location in the target cellview.
