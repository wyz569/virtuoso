useRotateReference

layout useRotateReference boolean { t | nil }

Description

Activates the options in the Rotate form for specifying the reference settings used for rotating an object.

The default is nil.

GUI Equivalent

Command:

Edit – Rotate – Rotate

Form Field:

Specify reference and angle

Examples

envGetVal("layout" "useRotateReference")
envSetVal("layout" "useRotateReference" 'boolean t)
