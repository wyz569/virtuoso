Export Image Form

Use the Export Image form to take a screen capture of the design contained in the specified window and save it to a standard image format.You can access the form by selecting File – Export Image.

Field Description

Export Region

Specifies the area of the design to capture:

  • Visible area: Select this option if you want to capture only the part of the design that is currently visible in the window.
  • Entire design: Select this option if you want to capture the entire design, not just the part that is currently visible in the window. In other words, this option will disregard the current zoom level and capture the entire design.
  • Selected area: Select this option if you want to capture a region. Click Select and then click-and-drag the mouse in the graphics window to create a bounding box around the area you want to capture. The screen coordinates of the selected area are then displayed in this field.

Size

Specifies the desired size of the resulting image. You can either provide the width and height for the image or specify a scale factor.

  • Scale exported region by: Specify the factor by which to scale the image. A factor greater than 1.0 scales the image up (for example, a factor of 2.0 doubles the size of the image) and a factor less than 1.0 scales the image down (for example, a factor of 0.5 reduces the size by half). The default factor is 1.0.
An image requires <width>*<height>*<scaleFactor>*32 bits of memory, therefore scaling an image to a high factor can cause an out of memory condition. If insufficient memory to save an image is an issue, a warning message will be displayed advising that you scale down the image size before attempting to save it again.
  • Custom: Specify a custom size for the image. You can specify either the width or the height, or both. If you specify only the width or only the height, then the other value is calculated automatically according to the aspect ratio of the region being captured.
  • Width and Height: If you specify both the width and height, then they are taken to be the maximum values and, in order to maintain the aspect ratio of the original region, either one might be reduced. The resulting image is never larger than the width and height specified. For example, if you specify a width of 400 and a height of 400, and the area being captured is not as high as it is wide, then the resulting image will be 400 pixels wide but less than 400 pixels high to maintain the aspect ratio.
  • Maintain aspect ratio: If Maintain aspect ratio is unchecked, then an image of the specified size is created. The aspect ratio of the original region is still maintained and the image is padded with blank space as required.

Appearance
  • Full color/Monochrome/Bi-color: Specify the color format to be used.
  • Transparent Background: Select for the image to have a transparent background.
  • Background/Foreground: Specify the color to be used for the image background or foreground using the Select Color form.
  • Swap: Switches the Background and Foreground settings.

Halos
  • Style: Select the halo style to be export, Gradient (default) or Flat.
  • Scale with image size: Select to re-size halos in accordance with image export size.

Output
  • Name: Specify the name of the file for the image to be saved as.
  • Type: Specify the format to save the image in.
  • Directory Icon: Specify the location to save the image to.
  • Save to File: Saves the image to the specified location.
  • Copy to Clipboard: Copies the image to the clipboard for re-use.
  • Close dialog after saving image: By default, the Export Image dialog closes after you capture an image. If you want the dialog box to remain open in order to capture more images, uncheck this option.
  • Reset to Defaults: Restores field settings to original values.
