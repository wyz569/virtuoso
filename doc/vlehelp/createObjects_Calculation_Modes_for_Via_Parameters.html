Calculation Modes for Via Parameters

You can set the calculation mode for via parameters by using the Via Parameters Calculation Mode option in the Layout Editor Options form. You can set the calculation mode to any of the following:

  • Min Rules & ViaDef: For all the via parameter fields, such as the number of cut rows and columns, and cut sizes, the minimum rule or the viaDef default values, whichever is greater, are used.
  • Minimum Rules: For all the via parameter fields, only the minimum rules, such as minWidth and minSpacing, are used.
  • ViaDef Defaults: For all the via parameter fields, viaDef defaults are used.
