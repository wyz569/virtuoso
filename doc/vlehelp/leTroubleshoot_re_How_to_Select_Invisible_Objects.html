How to Select Invisible Objects

If None Visible (NV) outlines an area of your cellview that appears blank, you probably turned off the visibility of all layers that appear in an instance and selected that invisible instance.

For example, if you clicked NV in the Layers panel to set all layers invisible, but Instances in the Objects panel is still on, you can select invisible instances.

To select invisible objects, do one of the following:

    • Set Instances off in the Objects panel so that instances are not selectable.
    • Click AV in the Layers panel to make all layers visible.
