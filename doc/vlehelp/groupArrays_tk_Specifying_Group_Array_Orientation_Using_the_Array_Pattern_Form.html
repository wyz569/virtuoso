Specifying Group Array Orientation Using the Array Pattern Form

You can define the orientation of individual cells of a group array. While creating or editing a group array, you can specify the orientation by entering values directly in the Array Pattern field, by selecting an existing pattern from the drop-down list, or by using the Array Pattern form.

To specify the orientation pattern to be applied to a group array by using the Array Pattern form:

  1. In the layout design window, open the required form.
    • If you are creating a group array, ensure the required objects are selected on the canvas and the relevant command form is open; for example, the Copy form. Also, ensure you have specified the required settings in the form.
      A group array can be created by using the Copy, Generate Selected Components, and Generate Clones forms.
    • If you are editing a group array, ensure the Edit Group Array Properties form is open for the group array.
  2. Click the Open Array Pattern editor button , which is next to the Array Pattern field.
    The Array Pattern form opens.
  3. Select a pattern from the Pattern list.
    The selected pattern is populated in the Pattern Editor section and a preview is displayed in the Preview section.
    Note that the table in the Pattern Editor section displays the orientation values and the number of rows and columns available in the selected pattern. Whereas, the Preview section displays how the selected pattern is applied to the actual number of cells in your group array. The actual number of cells is derived from the number of rows and columns you specify in the Copy form or the Edit Group Array Properties form.
  4. Optionally, use the spin boxes in the Pattern Editor section to modify the row and column count in the pattern.
    When you change the row and column count, the updated pattern gets populated in the Pattern Editor section. Also, the updated orientation values are listed in the Pattern field of the form and the Preview section displays the updated pattern applied to the group array, as shown in the following image.
  5. Click OK.
  6. Click OK. If you are using the Copy form, click Apply.
    The specified orientation pattern is applied to the group array.
