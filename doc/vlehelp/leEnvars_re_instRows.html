instRows

layout instRows int numberOfRows

Description

Specifies the number of rows for an arrayed instance.

The default is 1.

GUI Equivalent

Command:

Create – Instance

Field:

Mosaic: Rows

Examples

envGetVal("layout" "instRows")
envSetVal("layout" "instRows" 'int 7)
