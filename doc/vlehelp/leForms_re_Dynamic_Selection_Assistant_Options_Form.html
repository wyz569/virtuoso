Dynamic Selection Assistant Options Form

Use the Dynamic Selection Assistant Options form to specify how selected objects are displayed in the assistant and the mode used when opening a cellview from the shortcut menu of the assistant.

Field

Description

Display Controls

Enables you to specify how the selected objects are displayed in the Dynamic Selection assistant.

  • Automatically zoom to selected : Zooms to the selected objects in the design display area automatically. If this option is selected, you can zoom shapes in the hierarchy by selecting the shapes in the assistant. You can also zoom to the corresponding shape on the canvas if you double-click the shape in the assistant.
  • Automatically expand hierarchy : Expands the hierarchical instances, groups, and ROD objects shown in the assistant, automatically. The hierarchy is expanded down to the display level set in the Display Options form. If the display level is set to 0, the hierarchy cannot be expanded. The assistant does not necessarily list all the objects contained in a particular hierarchical object, only those that were under the pointer when the display was frozen.
  • Hide non-selectable objects: Removes from the assistant any objects that cannot be selected.
  • Reverse LPP order: Displays the layer-purpose pairs (LPPs) for selected objects from bottom to top. By default, LPPs of selected objects are displayed from top to bottom.
  • Flat view: Displays the flattened version of the hierarchy. If this option is selected, LPP, Data, and Hierarchy columns are displayed in the assistant.

Open mode

Specifies whether the cellview is opened for in the edit or read mode.

Open in

Specifies whether the cellview is opened in a new tab, the current tab, or a new window.
