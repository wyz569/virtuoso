pathEndExt

layout pathEndExt float pathExtension

Description

Specifies the end extension of paths with variable end type.

The default is 0.0 user units.

GUI Equivalent

Command:

Create – Shape – Path and Create – Wiring –Wire

Field:

End Extension and End

Examples

envGetVal("layout" "pathEndExt")
envSetVal("layout" "pathEndExt" 'float 1.0)
