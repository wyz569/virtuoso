Technology File

Each design library must be associated with a technology library that is compiled from a technology file. The layout editor requires the technology file to contain complete definitions of the layers used in your design.

A technology file defines the following:

  • Design layers and layer properties
  • Constraint and constraint groups
  • Via definitions
  • Site definitions
  • Technology devices
