Custom Show Layer/Custom Show Purpose Form

Use the Custom Show Layer or Custom Show Purpose form to define the criteria for filtering layers in the Layers panel. In the Custom Show Layer form, you can filter on the basis of layers, and in the Custom Show Purpose form, you can filter on the basis of purposes.

Field

Description

Match any of the following

Displays in the Layers panel only those layers for which at least one of the specified criteria match.

Match all of the following

Displays in the Layers panel only those layers for which all the specified criteria match.

Add criteria

Adds a row where you can define a new criterion for displaying layers in the Layers panel.

To define a criterion, you need to select an operator and a value for the column. In the Custom Show Layer form, you can select a layer name, and in the Custom Show Purpose form, you can select a purpose name.
