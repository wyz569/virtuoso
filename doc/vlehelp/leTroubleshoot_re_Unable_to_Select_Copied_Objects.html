Unable to Select Copied Objects

If you move or copy an object into a cellview of a different library and then are not able to select the object:

  1. Click in the new cellview.
  2. Look for the name of the layer for the object in the Layers panel of the Palette assistant.
    • If the layer name is unselectable (shaded), right-click the layer name to make it selectable.
    • If the layer name does not appear in the Layers panel, the layer either is not defined for this library or has been removed from the panel.

If the layer is not defined for the library, cancel the copy by doing either of the following:

  • Choose Edit – Undo in the original cellview.
  • Choose File – Discard Edits in the new cellview to delete all edits you made since you last saved the cellview.

If the layer has been removed from the Layers panel, replace it by choosing Edit – Set Valid Layers.
