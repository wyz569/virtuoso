groupArrayXSpacing

layout groupArrayXSpacing boolean float spacingValue

Description

Sets the spacing value used as the distance between cells on the X-axis when creating a locked group array by using the Copy form. This value is read when the copyGroupArrayLockXY environment variable is t.

The default value is 1.

GUI Equivalent

None

Examples

envGetVal("layout" "groupArrayXSpacing")
envSetVal("layout" "groupArrayXSpacing" 'float 5.0)
