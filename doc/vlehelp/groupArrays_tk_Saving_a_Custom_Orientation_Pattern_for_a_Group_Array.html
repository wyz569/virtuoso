Saving a Custom Orientation Pattern for a Group Array

While creating or editing a group array, you can define and save your own orientation patterns. You can reuse these patterns for group arrays across cellviews and sessions. If required, you can delete the custom orientation patterns.

To define and save a custom orientation pattern for a group array:

  1. In the layout design window, open the required form.
    • If you are creating a group array, ensure the required objects are selected on the canvas and the relevant command form is open; for example, the Copy form. Also, ensure you have specified the required settings in the form.
      A group array can be created by using the Copy, Generate Selected Components, and Generate Clones forms.
    • If you are editing a group array, ensure the Edit Group Array Properties form is open for the group array.
  2. Click the Open Array Pattern editor button , which is next to the Array Pattern field.
    The Array Pattern form appears.
  3. In the Pattern Editor section, use the spin boxes to modify the row and column count for the pattern.
  4. In the orientation pattern table in the Pattern Editor section, double-click any cell and select the desired orientation from the drop-down list.
  5. Similarly, select the orientation for other cells of the group array as per your requirement.
  6. In the Pattern field, delete the orientation values and enter a name for the custom pattern.
    As you specify a name, the Save orient pattern button is enabled.
  7. Click to save the custom orientation pattern.
    The custom orientation pattern is listed in the Pattern drop-down list.
    The Delete orient pattern button is now enabled, which means that you can delete this pattern if required. Note that you can save and delete only the custom orientation patterns and not the predefined orientation patterns.
  8. Click OK.
    The specified orientation pattern is created.
