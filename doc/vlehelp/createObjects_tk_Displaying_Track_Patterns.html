Displaying Track Patterns

To display all track patterns:

  1. Ensure track layers, which are layers with the purpose track, are valid and visible.
    If necessary, update the technology file.
  2. In the Grids panel in the Palette assistant, enable the Visibility check box for Tracks.
  3. To display individual track patterns, turn the visibility on or off, as required for specific track layers in the Layers panel.
  4. Use the View – Redraw command to update the view each time display changes are made to track patterns.
    The track pattern spacing and how closely the design is viewed using Zoom determines if track patterns can be seen.
