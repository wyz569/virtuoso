alphaBlendingFinOutLineValue

graphic alphaBlendingFinOutLineValue int outlineValue

Description

Sets the outline value for the alpha blending fin.

The default is 1000.

GUI Equivalent

None

Examples

envGetVal("graphic" "alphaBlendingFinOutLineValue")
envSetVal("graphic" "alphaBlendingFinOutLineValue" 'int 100)
