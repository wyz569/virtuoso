Creating a Track Pattern

Create tracks in both preferred and non-preferred layer directions.

To create a track pattern:

  1. In the layout window, choose Create – P&R Objects – Track Patterns.
    The Track Pattern Editor form opens.
  2. Filter the routing layers of the track patterns that you want to display in the form by selecting from the Filter by Routing Layer list.
    This allows you to limit the form to only show the track patterns on the routing layer you want to edit. This does not affect the routing layers displayed in the design window.
  3. Select the routing layer for the track pattern from the Layer field.
  4. Select track pattern Direction as Horizontal or Vertical.
  5. Specify the track spacing routing pitch in the Spacing field.
  6. Optionally, type a value to offset the wire from the track, in the Origin Offset field.
  7. Type the number of required tracks in the Number of Tracks.
  8. Set other options such as the color pattern for track patterns, width associated with the track, and the constraint group, as required.
  9. Click Update.
  10. Redraw the design window by using View – Redraw to view the track pattern you have created.
