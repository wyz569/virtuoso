Opening an Existing Cellview

To open an existing cellview in the layout editor:

  1. Choose File – Open from an editor window or the CIW.
    The Open File form appears.
  2. From the Library list, choose the library that contains the cellview you want to open.
  3. In the Cell field, specify a cell name using one of the following methods:
    • Type the cell name.
    • Click a cell name in the Cells list box on the right.
    • Click Browse to select the cell in Library Browser.
  4. Choose the View to be opened.
    The Type field automatically updates to display the type of view of the selected cell view.
    If a thumbnail of the selected cellview is available, it is displayed in the form as an aid to selection. If no thumbnail is available, the space appears blank. For more information about the thumbnail of a cellview, see
  5. From the Open with list, choose the application that the cellview should open with.
  6. Optionally, select the Always use this application for this type of file check box to associate the selection made in the Open with list with the current View type selected. This will save you in future from having to select what application should be opened with a selected cellview Type.
  7. Choose whether to open the cellview in edit or read mode.
    The Library path file field displays the location where the cellview is stored.
  8. If the Open in option is displayed, choose to open the cellview in a new tab in the current session window, the current tab in the current session window, or a new window in a new session.
  9. Click OK.
