autoExtractSaveEnabled

layout autoExtractSaveEnabled boolean { t | nil }

Description

Saves the hierarchical cellviews extracted by the Extract Layout command automatically when the extraction scope is set to Current Cellview and Cellviews in Hierarchy.

The default is t.

GUI Equivalent

Command:

Connectivity – Update – Extract Layout

Field:

Save Extracted Cellviews
