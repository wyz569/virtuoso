filterSizeForResolution

graphic filterSizeForResolution string {"(High highVal)" | "(Medium medVal)"| "(Low lowVal)" }

Description

Enables you to customize the level of details, filter size applied to resolutions High, and/or Medium, and/or Low.

By default, this environment variable is defined as below:

graphic filterSizeForResolution ‘string "(High 2.0) (Medium 6.0) (Low 15.0)”

The table below lists the different filter sizes used and the lowVal, MedVal, and HighVal values defined in the filterSizeForResolution environment variable.
