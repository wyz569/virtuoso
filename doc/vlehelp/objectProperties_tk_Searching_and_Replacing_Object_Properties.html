Searching and Replacing Object Properties

You can search for one or more objects that have identical properties and replace those properties with a new value. This enables you to quickly update a large number of objects at the same time.

To search for objects and replace their properties:

  1. In the layout window, choose Tools – Find/Replace to display the Find/Replace form.
  2. Choose the object that you want to search for and set the required search criteria.
  3. In the Replace section, from the drop-down list, choose the type of property you want to replace.
  4. Type or select the new value.
  5. Click Select All.
    The layout editor highlights on the canvas all the objects that it finds. All the objects are now in a search group. The first object in the group is highlighted in a different color.
  6. Click Previous or Next to select different objects in the search group.
  7. To change the properties, do one of the following:
    • Click Replace to change the property for the current object in the search stack.
    • Click Replace All to change properties for all of the highlighted objects.
  8. Click Cancel when you are done updating the objects.
