Displaying Options Forms Automatically

There are two ways to display forms in Layout XL, depending on whether a command has a standard form or an options form.

  • Standard forms: These forms have three dots (...) following the command name on the menu. These forms appear automatically.
  • Option forms: These forms let you change command settings while you are running commands. These forms do not appear automatically. You can display an options form by double-clicking the right mouse button or pressing F3 when a command is active.

To display options forms automatically when you choose a command:

  1. In the CIW, choose Options – User Preferences.
    The User Preferences form opens.
  2. Select the Options Displayed When Commands Start check box and click OK or Apply.

Many forms do not have a Defaults button because there are no appropriate default settings for that command.
