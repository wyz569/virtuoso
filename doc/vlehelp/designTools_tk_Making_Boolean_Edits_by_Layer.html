Making Boolean Edits by Layer

You can perform Boolean logical operations on objects on a single layer or two layers. You edit copies of the objects, so the originals are unchanged.

  1. On the layout canvas, select the objects you want to edit.
  2. Choose Tools – Layer Generation.
    The Layer Generation form opens.
  3. Select the layers that you want to edit, the Boolean operator you want to use, and the output layer.
  4. Click OK.
