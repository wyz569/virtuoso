Points Formats Supported in the Enter Points Forms

You can use the Enter Points form to specify the points to create an object. This is especially useful when you want to create an object or a list of objects at specific coordinates in the design window.

In the Enter Points form, you specify the points for an object in the XY field. The Use Relative XY field indicates that the points specified in the XY field are the points relative to the last entered points of the command (delta X and Y).

You can enter points in the XY field in the following formats. You can prefix or suffix @ to a point to make it a relative point. The formats @(x y) or @x y or x@ y or x @y or x y@ formats imply that x y are relative points.

Format Use Relative XY Field Deselected Use Relative XY Field Selected

x y

The points are added as absolute points.

The points are added as relative points.
  • x1 y1 x2 y2 x3 y3
  • x1 y1, x2 y2, x3 y3
  • x1 y1; x2 y2; x3 y3
  • (x1 y1)(x2 y2)(x3 y3)
  • x1:y1 x2:y2 x3:y3
  • {x1 y1}{x2 y2}{x3 y3}
  • [x1 y1][x2 y2][x3 y3]
  • <x1 y1><x2 y2><x3 y3>

The points are added as absolute points.

Points are added as per the following:

  • If a point has already been added to the command, the points in this list are added as relative points.
  • If a point has not been added to the command, a warning message is displayed in the CIW. No points are added to the canvas.
  • (x1 y1) @(x2 y2) (x3 y3)
  • x1 y1 @x2 y2 x3 y3
  • x1 y1 x2@ y2 x3 y3
  • {x1 y1} {x2 @y2} {x3 y3}
  • x1 y1 x2 y2@ x3 y3

(x1 y1) and (x3 y3) are added as absolute points and (x2 y2) are added as relative points.

All the points are added as relative points.
Combination of the formats listed in the table is not supported in Virtuoso Layout Suite. For example, {x1 y1}[x2 y2] is not supported as it uses a combination of curly and square brackets.

You can also enter points in the XY field for nested commands, for example, you run the Rectangle command and then run the Ruler command. Use the following format: (x1 y1)(x2 y2) (x3 y3) (x4 y4). The points (x1 y1) and (x2 y2) are used to create a ruler. The points (x3 y3) and (x4 y4) are used to create a rectangle.
