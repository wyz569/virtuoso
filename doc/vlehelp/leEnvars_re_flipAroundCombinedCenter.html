flipAroundCombinedCenter

layout flipAroundCombinedCenter boolean { t | nil }

Description

Lets you flip the multiple selected objects around the specified reference point of the imaginary bounding box around the objects.

The default is t.

GUI Equivalent

Command:

Options – Editor

Edit – Flip – Flip

Edit – Flip – Flip Vertical

Edit – Flip – Flip Horizontal

Field:

Flip Around Combined BBox

Examples

envGetVal("layout" "flipAroundCombinedCenter")
envSetVal("layout" "flipAroundCombinedCenter" 'boolean nil)
