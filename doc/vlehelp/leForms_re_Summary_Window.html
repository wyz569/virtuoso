Virtuoso Layout Suite XL: Basic Editing User Guide
Product Version IC23.1, November 2023

Summary Window

Use the Summary window to view statistics related to the current cellview.

Field Description

Show Environment

Displays information about the cellview and its edit mode, display levels, and current entry layer.

Layer Object Statistics

Lists the number of shapes created in the cellview, the layer on which shapes are present, and the color and lock state of the shapes. The shape types listed are Arc, Bend, Donut, Dot, Ellipse, Label, Line, Path, PathSeg, Polygon, Rect, Trl, Taper, and TextDisplay.

Routing Grid Statistics

Displays the x and y steps, offsets and the manufacturing grid information of the valid routing layers.

Valid routing layers belong to a valid constraint group. The default constraint group for the design is considered to be the valid constraint group. The following constraint groups are considered in the given order: default constraint group, active constraint group, or foundry constraint group. If one does not exist, the next one in the order is considered. If the foundry constraint group also does not exist, then Metal and Poly are considered as valid routing layers.

Instance Statistics

Displays names of the cell instances and arrays of instances (mosaics) that have been placed in the current cellview, as well as the total number of unplaced instances.

Design Statistics

Displays the layer summary that lists the total number of metal and routing layers. The Component Summary portion of this section reports the placement status of components.

Wire Statistics

Displays the following that have been placed in the current cellview:

  • Total number of standard and custom vias
  • Total number and length of paths, path segments, 45-degree paths, and 45-degree path segments

Marker Object Statistics

Displays the number of error markers that have flagged design violations in the current cellview.

Row Object Statistics

Displays the number of horizontal and vertical rows that have been placed in the current cellview.

Track Pattern Statistics

Displays the number of horizontal and vertical tracks that have been placed in the current cellview.

Blockage Object Statistics

Displays the number of blockage objects, placement area blockage objects, placement area halo objects and layer blockage objects that have been placed in the cellview. It displays the total number of blockage objects, the blockage count of each type of blockage in the cellview, and is sorted by layers.

PR Boundary Object Statistics

Displays whether a PR boundary object has been placed in the cellview, shows the origin point of the PR boundary and its area measurement in user units, and the shape of the PR boundary.

Net Statistics

Lists the number of terminals, instance terminals, guides, and steiners for each net in the cellview. For each constituent figure of the net, the report lists the layers, wire length, horizontal and vertical wire length, 45-degree wire length, vias, routes, and wire segments.

Connectivity Summary

Lists connectivity statistics for a cellview based on signal types, such as signal, power, ground, and clock.

Routing Statistics

Displays routing statistics for the selected set based on layers. For each layer, the report lists total wire length, horizontal and vertical wire length, 45-degree wire length, guide length, and the number of down vias and guides.
