Auto Via Assistant

Use the Auto Via Assistant to analyze and create vias. This assistant is available in Virtuoso Layout Suite EXL and higher tiers only.

The Auto Via Assistant user interface has the following components.

Analyze Tab

Lets you analyze vias in your design to check for locations where you can create vias.

Create Tab

Lets you create vias at the suggested locations.

The following table describes the options available on the Constraint Group drop-down.

Field Description

Constraint Group

Specifies the constraint group that lists the valid vias to be used when creating vias. The available options are: Same as Wire, LEFDefaultRouteSpec, foundry, skip_a_track, virtuosoDefaultExtractorSetup, virtuosoDefaultSetup, and wide.

The default is Same as Wire.
