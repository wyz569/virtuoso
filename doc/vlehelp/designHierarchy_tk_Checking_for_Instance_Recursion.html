Checking for Instance Recursion

You can prevent commands such as Create Instance, Make Cell, and Search/Replace from creating an instance of itself in a cellview. You can enable recursion checking by using the Layout Editor Options form.

To automatically check for recursion when placing instances:

  1. In the layout window, choose Options – Editor.
    The Layout Editor Options form opens.
  2. Select Recursion Check and click OK.
