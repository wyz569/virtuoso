Generating Labels for Bond Pads

Use Generate Labels mode in the Pad Opening Info form to specify the label names that are generated and how they are drawn on the canvas.

To generate labels for bond pads:

  1. Open a hierarchical cellview that contains bond pads.
  2. Choose Tools – Pad Opening Info.
    The Pad Opening Info form is displayed.
  3. Select the Generate Labels radio button at the top of the form.
  4. Set the Search and Sort criteria as required.
  5. In the Label Control section, set the following:
    1. Set the pattern for pad naming in the Label (Pattern) field. The label names include the label prefix and the pad numbers, for example, PAD1, PAD2, and so on.
      Alternatively, choose From File to specify a file from which the names are to be read.
    2. Select the appropriate Layer/Purpose. The labels are printed on the specified layer.
    3. Set the Font, Height, and Justification for the labels as required.
  6. If required, click Delete Bond Pad Labels to remove all the labels from the top level.
  7. Click Apply.
    You can now zoom in to any bond pad and see the label information placed on the pad opening.
