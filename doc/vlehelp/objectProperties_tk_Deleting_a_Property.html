Deleting a Property

To delete a property:

  1. In the layout window, do one of the following:
    • If you want to delete a cellview property, choose File – Properties.
    • If you want to delete an object property, select the object and then choose Edit – Basic – Properties.

    The Edit Cellview Properties or the Edit Properties form of the object appears.
  2. Click the Property tab.
    The properties of the object or cellview appear.
  3. Click the property label you want to delete.
  4. Click Delete.
    The property is deleted from the form.
