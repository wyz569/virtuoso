Aligning a Layer with Another Layer

To align a layer with another layer:

  1. In the layout window, choose Edit – Advanced – Align.
    The Align form opens.
  2. In the Reference section:
    1. Choose Align Direction as horizontal or vertical.
    2. Choose Layer as the reference to align items to a layer.
    3. Select the layer you want to use to align the layer to.
    4. Choose the layer edge you want the objects to align with, from the Use list.

    You are prompted to select a reference object for the alignment.
  3. On the canvas, click the object you want to align other objects or layers to.
    The alignment pointer appears.

  4. In the Align Using section:
    1. Select the Layer radio button.
    2. Select the layer you want to align with the reference layer.
    3. Select the edge of the object or layer you want to align, from the Use list. For example, Left Edge.
  5. To evenly space the objects in the Align Direction, select the Apply Spacing check box and enter the value in user units by which you want to separate the objects. If you do not use the Apply Spacing, the objects retain their original spacing. They are aligned, but not evenly spaced.
  6. Select the Orthogonal Spacing check box to space the objects perpendicular to the Align Direction as well. This option is available only if Apply Spacing is selected.
  7. Select how you want the objects sorted, from the Sort Objects By list.
    You are prompted to select the objects to be aligned. You can select the objects by clicking on each object one at a time while holding down the Shift key, or you can select the objects by drawing a selection box around them.
  8. Click Apply.
    The following figure shows an example where the reference is Layer of material poly1 dg, the Align Direction is Vertical, and the Use field is set to Left Edge. The Align Using is Layer, the Use cyclic is set to Left Edge. The Apply Spacing is selected, with 5.0 user units Between Objects. Objects are sorted by Instance Name.
