Moving an Object by Direct Manipulation

You can move objects by selecting and dragging them on the canvas.

To move an object:

  1. On the layout canvas, select an object.
    Alternatively, you can select an object by specifying the points at which the object is placed in the design area by using the Enter Points form.
  2. Move the pointer over the object to see the move pointer.
  3. Press the left mouse button at the reference point, which is the point from which the move operation starts, and drag the pointer to the destination point. Alternatively, you can specify the reference and destination points using the Enter Points form.
  4. Release the left mouse button.
