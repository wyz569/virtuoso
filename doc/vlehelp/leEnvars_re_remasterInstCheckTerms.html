remasterInstCheckTerms

layout remasterInstCheckTerms boolean { t | nil }

Description

Checks the terminals of cellviews in the specified libraries for matching or a subset of matching terminals.

The default is nil.

GUI Equivalent

Command:

Tools – Remaster Instances

Field:

Check Terminals

Examples

envGetVal("layout" "remasterInstCheckTerms")
envSetVal("layout" "remasterInstCheckTerms" 'boolean t)
