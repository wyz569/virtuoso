Flip Vertical Form

Use the Flip Vertical form to define how objects are flipped vertically.

Field Description

Reference    

Specifies the reference point around which an object is mirrored along the Y axis. Valid values are LowerLeft, CenterLeft, UpperLeft, LowerCenter, CenterCenter, UpperCenter, LowerRight, CenterRight, and UpperRight.

Environment variable: flipReference

Flip around combined bbox

Flips multiple selected objects around the specified reference point of the combined bounding box of the objects.

Environment variable: flipAroundCombinedCenter
