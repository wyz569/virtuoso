Manual Mode Placement Override

The manual mode placement override is used in cases where a third-party IP block is used in a design. This mode enables you to manually specify the placement mode or offset on a per-instance or master basis. This is done by setting the leSnapPatternSnapping property on either the instance or the master.

If the leSnapPatternSnapping property is set on both the instance and its master, the instance definition overrides the master definition.
