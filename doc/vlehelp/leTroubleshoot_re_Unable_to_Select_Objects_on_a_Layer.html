Unable to Select Objects on a Layer

The only objects that you can select in a cellview are on selectable layers in the Layers panel of the Palette assistant.

All layers are selectable if they are included and set to selectable and valid in the techDisplays subsection in the technology file, whether or not they appear in the Layers panel in the Palette Assistant. If they do appear in the Layers panel, you can turn on or off their selectability or validity.

In the Palette assistant, you can turn off selection for one or more layers. When you do, a layer that is not selectable appears with its name shaded in the Layers panel.

To make a layer selectable:

  • Right-click the layer name in the Layers panel.

To display a layer on the Layers panel, choose Edit – Set Valid Layers from the shortcut menu in the panel. If you are using the leLswLayers section of the technology file, only the layers listed in that file are displayed.
