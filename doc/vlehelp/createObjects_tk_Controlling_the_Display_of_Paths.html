Controlling the Display of Paths

You can display path borders and centerlines, borders only, or centerlines only.

  1. In the layout window, choose Options – Display.
    The Display Options form appears.
  2. In the Display Control section, select one of the following from the Path Display list:
    • Borders and Centerlines: Displays both the path centerline and its outer edges.
    • Borders only: Displays only the path borders and eliminates the centerline display.
    • Centerlines only: Displays only the centerline.
  3. Click OK.
