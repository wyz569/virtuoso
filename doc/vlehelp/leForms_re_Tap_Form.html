Tap Form

Use the Tap form to access information from the following objects: paths, wires, rectangles, polygons, circles, ellipses, donuts, microwave objects, blockages, instances, vias, rows, labels and text displays.

Field Description

Layer

Sets the tapped layer as the entry layer, and updates the Layer assistant. This selection includes paths, path segments, wires, rectangles, polygons, circles, ellipses, donuts, microwave objects, labels, and textDisplays.

Net

Captures net names from rectangles, polygons, and paths, wires, wire segments, labels, or text displays. You can capture the net names to automatically insert the net names when applying connectivity to a via.

Attributes

Captures attributes from paths, instances, vias, rows and microwave objects.

Pop up selection window

Displays the Choose Object to Tap form.

When you click an object with multiple layers, the Choose Object to Tap form appears with a list of layers on which you can create the object.

Command (shapes, wires)

Starts the create command for the tapped shape or wire.

The Command (shapes/wires) option works only for shapes and wires. It does not work for other object types such as instances, vias, and blockages. When you select this option, the following commands are started for each object type:

  • Rectangle: leHiCreateRect
  • Polygon: leHiCreatePolygon
  • Path: leHiCreatePath
  • Ellipse: leHiCreateEllipse
  • Donut: leHiCreateDonut
  • PathSeg: leHiCreateGeometricWire
