Product Version IC23.1, November 2023

Yank Form

Use the Yank form to copy objects, and parts of objects, into a temporary buffer.

Field Description

Yank Shape

Sets the type of shape you draw around, or through, the objects you want to copy.

Yank Levels

Sets the number of levels of design hierarchy through which Yank can copy shapes. The current cellview is level 0, instances inside it are level 1, and so forth.

Snap Mode

Controls the shape of line segments when Yank Shape is set to polygon.
