alignMatchRefSize

layout alignMatchRefSize boolean { t | nil }

Description

Lets you change the size of the target pin to that of the reference pin.

The default is nil.

GUI Equivalent

Command:

Edit – Advanced – Align

Field:

Match size with the reference pin

Examples

envGetVal("layout" "alignMatchRefSize")
envSetVal("layout" "alignMatchRefSize" 'boolean t)
