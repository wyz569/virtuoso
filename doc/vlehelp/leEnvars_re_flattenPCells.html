flattenPCells

layout flattenPCells boolean { t | nil }

Description

Specifies whether Pcells are flattened by default or by leHiFlatten() or by leFlattenInst().

The default is nil.

GUI Equivalent

Command:

Edit – Hierarchy – Flatten

Field:

Pcells

Examples

envGetVal("layout" "flattenPCells")
envSetVal("layout" "flattenPCells" 'boolean t)
