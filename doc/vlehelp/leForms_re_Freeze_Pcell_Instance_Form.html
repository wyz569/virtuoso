Freeze Pcell Instance Form

Use the Freeze Pcell Instance form to specify the library, cell, and view of the non-Pcell instance of the new master created on the disk to freeze a Pcell instance.

Field Description

Library

Lets you select the name of the library in which you want to create the non-Pcell instance.

If one Pcell instance is selected for freeze, the library name in which the Pcell instance is saved is automatically populated in the Library field. If multiple Pcell instances are selected, the value Default is populated in the Library field. The Default value implies that each non-Pcell instance will be created in its respective library.

Cell

Lets you type the name of the non-Pcell instance.

A unique name is automatically populated in the Cells field. However, you can update the name of the cell. Also, when multiple Pcell instances are selected, the value Default (non-editable) is automatically populated in the Cells field.

View

Displays the view of the non-Pcell instance.

Use Existing Master

Lets you use the existing frozen Pcell master, if available, when converting a Pcell instance to a non-Pcell instance.

The Use Existing Master and Overwrite Existing Master options are mutually exclusive; only one of them can be enabled at a time.

Environment variable: reuseExistingFrozenMaster

Overwrite Existing Master

Enables you to overwrite the existing master.
