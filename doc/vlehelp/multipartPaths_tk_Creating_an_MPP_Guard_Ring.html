Creating an MPP Guard Ring

The following tasks are involved in creating a multipart path (MPP) guard ring:

  • Loading a technology file into your local directory.
  • Creating an MPP template.
    An MPP guard ring template must contain at least one subrectangle, and it should not have any connectivity in it.
  • Using the MPP template in the Guard Ring form and fine-tuning other settings.

To create an MPP guard ring when you have an MPP template available:

  1. In the layout window, choose Create – MPP Guard Ring.
    The Create Guard Ring form appears.
  2. From Guard Ring Template, select the MPP template you want to use to create the guard ring.
  3. Select the Technology Library in which the selected template is defined.
  4. Specify the connectivity information for the MPP guard ring in the Net Name field.
  5. Adjust the value in the Enclose by field to set the distance between the MPP guard ring and the selected objects.
  6. Select Rectangular or Rectilinear shape for the MPP guard ring.
  7. On the canvas, select the objects around which the MPP guard ring will be created. A preview of the guard ring appears around the selected objects.
  8. Click Apply to place the MPP guard ring around the selected objects.
