Object Yanking and Pasting

The Yank command performs a cookie-cutter copy. Yank copies objects and parts of objects into a temporary buffer. The Paste command places copies of objects from the temporary buffer.

Consider the following while using the Yank and Paste commands:

  • ROD objects are not supported. The Yank command copies only the shapes, but not the names of the shapes or the alignments. When you paste, the result is unnamed, unaligned, non-ROD shapes.
  • Blockage objects are not supported. The Yank command copies only the shapes and ignores the blockage objects.
  • Boundaries objects are not supported. The Yank command copies only the shapes and ignores the boundary objects.
  • Mosaics are not supported.

Yank and Paste commands are similar to Copy except that objects are chopped by the Yank shape. The Yank command is available both in read-only and edit modes.
