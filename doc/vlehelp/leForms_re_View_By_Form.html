View By Form

Use the View By form to specify multiple parameters for displaying layer-purpose pairs in a tree structure in the Layers panel.

Field Description

View items by

Enables you to select the top-level column based on which layer-purpose pairs are grouped.

You can specify the display order of layer-purpose pairs by choosing Ascending or Descending in the View items by section. By default, layer-purpose pairs are displayed in ascending order.

Then by

Specifies the next level grouping column based on which layer-purpose pairs are displayed. There are three Then by sections in the View By form using which you can define the second, third, and fourth level of grouping columns. For example, you can choose to view layer-purpose pairs based on four columns in the following order: Layer, Purpose, Priority, and Color.

You can specify the display order of layer-purpose pairs by choosing Ascending or Descending in each Then by section. By default, layer-purpose pairs are displayed in ascending order.
