Region Group Snapping

A region group is an oaFigGroup that contains only snap pattern shapes or width spacing pattern (WSP) region shapes. The layer-purpose pair of each shape in the oaFigGroup has a snapPatternDef or widthSpacingSnapPatternDef.

The reference region shapes along each direction are determined based on the region with the largest period of the snapping widthSpacingSnapPatternDef and are snapped similar to region shape snapping. If there is no region along a direction in the group, snapping does not occur along that direction.

If a region group contains both WSPs and snap patterns, WSP snapping takes precedence over snap patterns.
