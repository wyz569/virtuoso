Subparts of a Multipart Path

You can add any number of subparts to the current master path of a multipart path (MPP) in the layout window.

To create a subpart, you use the ROD Subpart form, which can be accessed by clicking the Subpart button in the Create Multipart Path form. You can choose to create an offset subpath, an enclosure subpath, or a set of subrectangles.

  • Adding an Offset Subpath
  • Adding an Enclosure Subpath
  • Adding a Set of Subrectangles Subpart

If any subparts of the type you selected already exist, the ROD Subpart form displays one line of information for each subpart in the box at the top of the form.

While creating an MPP, you can choose to edit or delete a subpart as required.

  • Editing a Subpart of an MPP
  • Deleting a Subpart of an MPP

If you want to use the subpart values again, you need to save them as a template along with the master path.
