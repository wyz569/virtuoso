Net Setting on Instance Terminals

You can use the Property Editor assistant to set up the nets on the terminals of a selected instance. Select Connectivity from the property category selector list and specify the nets in the property value column for each terminal on the instance.

By using the Property Editor assistant, you can set up multiple terminals of different instances at the same time.
