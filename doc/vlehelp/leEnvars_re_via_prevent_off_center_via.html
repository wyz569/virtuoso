viaPreventOffCenterVia

layout.ava viaPreventOffCenterVia boolean { t | nil }

Description

Places the via on the centerline when analyzing or creating vias using Auto Via Assistant.

The default is nil.

GUI Equivalent

Command:

Assistants – Auto Via Assistant – Constraint Overrides – Show All

Field:

Place via on centerline

Examples

envGetVal("layout.ava" "viaPreventOffCenterVia")
envSetVal("layout.ava" "viaPreventOffCenterVia" 'boolean t)
