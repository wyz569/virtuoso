remasterInstUpdateCellText

layout remasterInstUpdateCellText string { "any_cell_name" }

Description

Specifies the name of the cellview that you want updated with instances specified by remasterInstSearchLibText.

GUI Equivalent

Command:

Tools – Remaster Instances

Field:

Update to: CellName

Examples

envGetVal("layout" "remasterInstUpdateCellText")
envSetVal("layout" "remasterInstUpdateCellText" 'string "MyCell")
