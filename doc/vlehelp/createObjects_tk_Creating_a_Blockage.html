Creating a Blockage

To create a blockage:

  1. In the layout window, choose Create – P&R Objects – Blockage.
    The Create Blockage form opens.
  2. Select the type from Blockage Type.
  3. If the blockage is a layer blockage:
    1. Select the appropriate material type from Material.
    2. Select the appropriate layer from the Layer list.
  4. Specify how you want to compute blockage spacing. You can choose from Blockage geometry or Effective width, or use Specify to enter a spacing value.
  5. Optionally, specify blockage settings such as the color, owner, maximum density, and ROD object name.
  6. If you want to use Manual as the Create Method method:
    1. Select the appropriate snap mode in the Create Method section.
    2. On the canvas, click the first point of the blockage. Then, click the next point of the blockage. Just as you would for a polygon, continue to enter points until the shape is the final shape you want for the blockage.
    3. Press Enter or double-click on the canvas to complete the blockage object.
  7. If you want to use Automatic as the Create Method method:
    1. Select the appropriate method from Derive From.
      Depending on the method you select, you are prompted to select objects or instances to derive the blockage.
    2. Click the instances or objects on the canvas that you want to include in the blockage area.
    3. Click the Apply button in the form or press Enter to complete the blockage object.
