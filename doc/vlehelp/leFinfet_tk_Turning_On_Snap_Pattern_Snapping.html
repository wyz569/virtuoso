Turning On Snap Pattern Snapping

To turn on the snap pattern snapping in layout editor:

  1. In the layout window, choose Options – Editor.
    The Layout Editor Options form opens.
  2. Select the Snap Pattern Snapping check box and click OK.

To turn on the Snap Pattern Snapping option at startup, you can set the snapPatternSnapping environment variable in your .cdsenv file.

For more information on snap pattern snapping, see the video Snap Pattern Enabling.
