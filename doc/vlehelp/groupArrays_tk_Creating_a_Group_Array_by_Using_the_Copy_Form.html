Creating a Group Array by Using the Copy Form

To create a group array by using the Copy form:

  1. In the layout design window, select the objects that you want to repeat as a pattern in your design.
  2. Choose Edit – Copy and then press F3 to open the Copy form.
  3. In the Copies section, select the type of copy you want to create. Depending on the option you select, specify the number of copies or rows and columns in the required fields.
  4. Select the Create as Group Array check box.
  5. In Array Pattern, specify the orientation value of individual cells of the group array by using any of the following methods:
    • Enter the orientation values directly in the text field by enclosing them in quotation marks.
    • Select a pattern from the drop-down list, which shows predefined and custom patterns.
    • Use the Array Pattern form, which you can access by clicking the Open Array Pattern editor button .
      For information, see Specifying Group Array Orientation Using the Array Pattern Form.
  6. In the Spacing section, select a mode and specify the spacing values:
    • Pitch: Specify the spacing in the Delta X,Y fields.
    • Spacing: Specify the spacing in the Spacing X,Y fields.

    If copyGroupArrayLockXY is set to t, you cannot specify spacing mode and spacing values for a group array.
  7. Optionally, in Reference LPP, select the reference layer-purpose pair that needs to be used for computing the bounding box of the cells of the group array.
  8. Specify other settings in the Copy form as required.
  9. Click Apply in the Spacing section of the form.
    The group array with the specified parameters is created.
