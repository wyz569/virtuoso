Flattening Instances

To display the detail in all the instances that you want to flatten:

  1. In the layout window, choose Options – Display.
    The Display Options form opens.
  2. Set Display Controls to display the detail in all instances you want to flatten.
  3. Click OK.
  4. Choose Edit – Hierarchy – Flatten.
    The Flatten form opens.
  5. Set the Hierarchy Level to one level, displayed level, or user level.
  6. Use the Preserve options to preserve the required information and attributes.
  7. Click OK.
    The instance is flattened as per the specified settings.

When flattening Pcell instances, you can delete fluid shapes by enabling the deleteFluidShapes environment variable.
