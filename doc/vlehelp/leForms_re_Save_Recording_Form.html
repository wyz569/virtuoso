Save Recording Form

Use the Save Recording form to save the recording of your actions, which you can replay later to perform those actions quickly. You can record your actions by using the Capture and Relay assistant in Layout EXL or a higher tier.

Field Description

Command Name

Specifies the name of the recording.

Description

Specifies information about the recording.

The tooltip for a saved recording shows a brief description in the Capture and Relay assistant.

Group

Specifies the group under which you want to categorize your recording for easy access.

Save To

Specifies the location where the recording is saved.

The list of locations is derived from setup.loc. Your recording is saved as a .car file under the dfII/captureReplay directory at the selected location.

Save

Saves the recording at the specified location.
