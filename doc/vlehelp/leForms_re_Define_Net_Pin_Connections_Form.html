Define Net/Pin Connections Form

Use the Define Net/Pin Connections form to define a set of pins on a net to be connected externally. Depending on the pin connectivity command selected, the name of the form might be Define Must Connected Pins, Define Strongly Connected Pins, Define Weakly Connected Pins, or Define Pseudo Parallel Connected Net.

Field Description

Net

Lets you enter the name of a net you want to select. You can type the following in the Net field.

  • A net name. The list box in the form highlights the net and scrolls the list to make the net name visible.
  • A part of a net name (net, for example). The list box highlights all the nets with net as an element in the name, for example, net10, net12, net23, net30, net31, and scrolls the list to make the first example visible.
  • A regular SKILL expression using part of a net name, for example *bias. The list box highlights all the nets with the string bias, for example Nbias, NMbias, Pbias, and scrolls the list to make the first example visible.

Search

Finds and highlights in the list box the names of nets you type in the Net field.

The list box displays the names of all the nets in the design. To select a net from which to define externally connected pins, you can click the net you want to select or click and hold down the Ctrl key to select more than one net. When you select a net in the list box, the software selects the net on the canvas and displays the flight lines for the net.
