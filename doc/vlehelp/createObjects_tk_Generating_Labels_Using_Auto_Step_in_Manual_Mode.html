Generating Labels Using Auto Step in Manual Mode

You can use the auto step method to create labels. This method is available only in Manual mode.

To create labels by using the auto step method:

  1. In the layout window, choose Create – Label.
    The Create Label form opens.
  2. Set the Mode as Manual.
  3. Type the label pattern in the Label (Pattern) field.
  4. Select the Step check box in the Auto Step section.
  5. From the Step list, if you select Input X&Y:
    1. Specify values in the X and Y fields.
      All the generated labels attach to the pointer and are visible as per the specified spacing.
    2. Click on the canvas to place the labels.
      If the Keep Label check box in the Label Options section is selected, the label sequence remains attached to the pointer after you click to place the label.
  6. From the Step list, if you select Custom:
    The first label attaches to the pointer.
    1. Set up the Snap Mode setting.
    2. Click to place the first label in the design display area.
      The remaining generated labels attach to the pointer, as shown in the figure.
      You can move the pointer around to determine the position of the second label.
    3. Click on the canvas to place the second label.
      The remaining labels are placed by using the same spacing as between the first and second placed labels and along the same direction as the first two labels.
  7. Press Esc or click Cancel in the form to finish creating labels.
