Show Selected Set Window

The Show Selected Set section shows information about the each type of object that is selected in the current cellview.

Field Description

Net Statistics

Lists the number of terminals, instance terminals, guides, and steiners for each net in the selected set. For each constituent figure of the net, the report lists the layers, wire length, horizontal and vertical wire length, 45-degree wire length, vias, routes, and wire segments.

Connectivity Summary

Lists connectivity statistics for the selected set based on signal types, such as signal, power, ground, and clock.

Routing Statistics

Displays routing statistics for the selected set based on layers. For each layer, the report lists total wire length, horizontal and vertical wire length, 45-degree wire length, guide length, and the number of down vias and guides.

Total Shapes Selected

Lists the selected objects, their layers, the number of points used to create them, and the coordinates of the bounding box that surrounds each object.

Total Instances Selected

Lists all selected cell instances, including each instance name, placement orientation, and the coordinates of its origin.

Total Mosaics Selected

Displays information about arrays of cell instances are selected.

Total Boundaries Selected

Lists the selected boundaries, their types, the number of edges used to create them, the area, and the boundary names.

Total Blockages Selected

Lists the selected blockages, their types, their layers, their owners, the number of points used to create them, and their areas.

Total Vias Selected

Lists the selected vias, their types (standard or custom), their names, their nets, their directions, and in the case of custom vias their library, cell and view names.
