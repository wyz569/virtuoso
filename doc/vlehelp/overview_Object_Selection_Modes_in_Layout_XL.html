Object Selection Modes in Layout XL

You can select objects either before or after you start a command.

Preselect Mode

If you select objects before starting a command, the command is said to be operating in preselect mode. In this mode, the command operates only on the objects that were selected when the command was started. When the command is completed, the selected objects remain in the selected set.

In preselect mode:

  • Editing commands do not automatically repeat, even if the Repeat Commands option is switched on in the Layout Editor Options form.
  • Some editing commands prompt you for a starting point, called the reference point, for the edit operation.

Postselect Mode

If you select the command and then the objects, the command is said to be operating in postselect mode. In this mode, you make your selection within the command and when the command is completed, nothing remains selected.

In postselect mode:

  • Editing commands automatically repeat if Repeat Commands is on (the default).
  • Editing commands do not necessarily prompt you for a reference point. If you click to select the first object, the editor uses that point as the reference point.

Object Deselection

To deselect an object, right-click the object and select (De)Select Under Cursor from the shortcut menu. On the De(select) submenu, hover over the name of the object you want to deselect. The object gets highlighted on the canvas. Click the object name to deselect the object on the canvas.
