viaConstraintGroup

layout viaConstraintGroup string "name_of_constraint_group"

Description

Specifies the constraint group from which to use valid vias for creating and editing vias.

The default is Same as Wire.

GUI Equivalent

Command:

Options – Editor

Field:

Wire Editing: Create Via: Via

Examples

envGetVal("layout" "viaConstraintGroup")
envSetVal("layout" "viaConstraintGroup" 'string "foundry")
