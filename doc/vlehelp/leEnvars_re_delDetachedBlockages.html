delDetachedBlockages

layout delDetachedBlockages boolean { t | nil }

Description

Deletes blockages which would otherwise remain after an instance containing blockages is flattened.

The default is nil.

GUI Equivalent

Command:

Edit – Hierarchy – Flatten

Field:

Detached blockages

Examples

envGetVal("layout" "delDetachedBlockages")
envSetVal("layout" "delDetachedBlockages" 'boolean t)
