pteLSManagerAsAssistant

layout pteLSManagerAsAssistant boolean { t | nil }

Description

Allows you to enable the layer set manager as an assistant. The default is nil.

The shell environment variable CDS_PALETTE_TYPE should be set to MultiAssistant for this environment variable to work, as shown below:

setenv CDS_PALETTE_TYPE MultiAssistant

The environment variable should be added to .cdsinit or .cdsenv files.

GUI Equivalent

None

Examples

envGetVal("layout" "pteLSManagerAsAssistant")
envSetVal("layout" "pteLSManagerAsAssistant" 'boolean t)
