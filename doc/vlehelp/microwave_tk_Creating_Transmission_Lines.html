Creating Transmission Lines

You can use the Trl command to create a transmission line (trl). A transmission line shows connections in microwave designs. You create a transmission line the same way you create paths.

To create a transmission line:

  1. In the layout window, choose Create – Microwave – Trl.
  2. Press F3 to open the Transmission Line form.
  3. Set Bend Style to bend.
  4. Set Bend Factor to 1.
  5. Set the Width to 2.
  6. Click on the canvas to create each point of the transmission line.
  7. To complete the transmission line, double-click the last point.
