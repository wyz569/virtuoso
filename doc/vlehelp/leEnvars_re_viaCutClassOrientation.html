viaCutClassOrientation

layout viaCutClassOrientation cyclic { "SameOrientation" | "Vertical" | "Horizontal" | "Auto" }

Description

Lets you specify the preferred orientation for the cut class as SameOrientation, Vertical, Horizontal, or Auto.

The default is Auto.

GUI Equivalent

Command

Create – Via; Mode: Auto

Field

Cut Class Orientation

Examples

envGetVal("layout" "viaCutClassOrientation")
envSetVal("layout" "viaCutClassOrientation" 'cyclic "SameOrientation")
envSetVal("layout" "viaCutClassOrientation" 'cyclic "Horizontal")
