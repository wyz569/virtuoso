viaAnnotateViasNotRetained

layout.ava viaAnnotateViasNotRetained boolean { t | nil }

Description

Creates annotations on overlaps on which vias have not been created because they add a metal or create DRC violations.

The default is t.

GUI Equivalent

Command:

Assistants – Auto Via Assistant – Create – Annotations

Field:

Vias not retained

Examples

envGetVal("layout.ava" "viaAnnotateViasNotRetained")
envSetVal("layout.ava" "viaAnnotateViasNotRetained" 'boolean nil)
