Tapping Objects Using the Tap Command

The Tap command accesses information from the following objects: paths, wires, rectangles, polygons, circles, ellipses, donuts, microwave objects, blockages, instances, vias, rows, labels and text displays. In the Tap form, you can specify how the Tap command affects the current layer, net, attributes, and create commands when each type of object is tapped.

To tap objects by using the Tap command:

  1. In the layout window, choose Tools – Tap.
  2. Press F3 to open the Tap form.
  3. Select the required tap types. You can select Layer, Net, Attributes, and Command. You can also choose to display a selection window if there are overlapping objects in the clicked area. The selection window allows you to choose the object to be tapped.
  4. Click an object on the canvas that you want to tap.
    The specified attributes are picked from the selected object.
  5. Choose a create command. For example, you can choose to create a rectangle. If you selected Command in the Tap form, the command to create a rectangle is automatically started if you tapped a rectangle.
    The create form is populated with the information obtained from the original object that you tapped and seeds the form with the information you selected in the Tap form.
