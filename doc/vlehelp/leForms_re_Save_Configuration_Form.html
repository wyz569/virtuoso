Save Configuration Form

Use the Save Configuration form to save the configuration for displaying information about shapes and objects.

Field Description

Select configuration name

Specifies the name for the object display configuration you are saving.

Select path

Specifies the path for the configuration display file.

The configuration is saved to odc.il at the location selectedPath/dfII/ODC/odc.il. The path that you choose to save the configuration determines the sequence in which a configuration is loaded and is derived from the setup.loc file.
