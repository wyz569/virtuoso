Instance Snapping for Track Groups in Detail Mode

When snapping instances containing snap patterns with track groups, non-uniform snap patterns have a lower precedence than the uniform snap patterns. The following snapping behavior is used:

  • If an instance has both uniform and non-uniform snap patterns, only the uniform snap pattern is used for snapping.
  • If an instance contains only non-uniform snap patterns, snapping is done to period lines.
  • If there are no non-uniform snap patterns, there is no change in the snapping behavior.

Instance snapping based on snap patterns is usually more accurate than shape-based snapping. As a result, if a snap pattern shape is found, shape snapping is disabled for that direction. In this case, shape snapping is not performed even if snapping is not possible for the snap pattern shape.
