Yanking and Pasting Objects in Post-Selection Mode

To copy part of an object, group of objects, or an instance:

  1. In the layout window, choose Edit – Basic – Yank.
  2. Press F3 to open the Yank form.
    In the Yank form, by default, Yank Shape is set to create a rectangle. You can change it to a polygon, as required.
  3. If you are in full selection mode, press F4 to switch to partial selection mode.
  4. Create the yank shape around the objects you want to copy.
    All objects and parts of objects inside the box are copied into a special yank buffer. If an instance is inserted by the Yank shape, the objects in it are brought to the top level (flattened).
  5. Choose Edit – Basic – Paste.
    The outline of the objects you yanked appears and follow the pointer.
  6. Click on the canvas at the location where you want to place the copied objects.

You can paste an object whose layer is not visible in the Layers panel of the Palette assistant. However, you cannot yank an object whose layer is not visible in the Layers panel and whose object type is not visible in the Objects panel of the Palette assistant.
