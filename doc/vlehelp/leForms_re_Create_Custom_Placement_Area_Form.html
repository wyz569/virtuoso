Create Custom Placement Area Form

Use the Create Custom Placement Area form to create a custom placement area. This form contains the Dressing Template Editor button that brings up the Dressing Template Editor sub form.

Field Description

Create Method

Lets you create an area in the following modes:

  • Manual: Lets you create a custom placement area by creating a rectangle using two points with an area size greater than zero. In this mode, the coordinates are not necessarily on placement grid.
  • Assisted: Lets you use all the options except the Number of Placement Areas field. If the values are not compatible with the placement grid coordinates the values are adjusted to multiples of the placement grid.
  • Auto: Lets you use all the options. If the values are not compatible with the placement grid coordinates the values are adjusted to multiples of the placement grid.

Environment variable: customPlacementAreaCreateMethod

Definition Section

This section lets you define the attributes of the custom placement area.

Base Name

Assigns by default a unique name of CPA_integer, where the integer is incremented each time a custom placement area is created within the cellview. You can overwrite this name with a user assigned name, as long as the name is unique to the cellview.

Height

Sets the height of the placement area.

Environment variable: customPlacementAreaHeight

Width

Sets the width of the placement area.

Environment variable: customPlacementAreaWidth

Number of Placement Areas

Sets the number of custom placement areas to create.

Environment variable: customPlacementAreaNumberOfAreas

Channel Width

Sets the width between the placement areas you create.

Environment variable: customPlacementAreaChannel

Orientation

Sets the orientation of the placement areas.

Environment variable: customPlacementFirstUnitOrient

Flip Every Other Area

Lets you flip every other placement area. You can put grounds and power together when you flip the rows

Environment variable: customPlacementAreaFlipUnit

Flip First Area

Lets you flip the first custom placement area.

Environment variable: customPlacementAreaFlipFirstUnit

Abut boolean button

Lets you abut custom placement areas.

Environment variable: customPlacementAreaAbutUnit

Areas

Lets you enter a positive integer that sets the number of custom placement areas that will abut together.

Environment variable: customPlacementAreaAbutNumberOfUnits

Abut First Area

Lets you abut the first placed custom placement area.

Environment variable: customPlacementFirstUnitOrient

Templates Section

This section lets you specify the rails and allowed component types automatically placed in the custom placement area.

Type

Lets you select Rail, Allowed Component, or Both.

If pre-defined rails or components exist, they are listed in the Pre-Defined box. Select the ones you want to use for the custom placement area and move them to the Selected box by using the arrow icons. Once they are in the Selected box, they are available to be placed in the cellview. You can then add the components individually selected in the Selected box.
