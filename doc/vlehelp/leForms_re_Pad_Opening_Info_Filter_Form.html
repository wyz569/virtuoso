Pad Opening Info: Filter Form

Use the Pad Opening Info: Filter form to set the filter for the customized report that you want to generate for pad shapes in a hierarchical cellview. You can access this form by using the Set Filter button in the Create Encore Report form.

Field Description

Filter Options
  • Mode: Select a mode from the first list as Pad Opening Name, X, Y, or Area. You can set filter for bond pads based on their name, width, height, and area.
  • Operator: Select an operator from the second list. The valid operators for Pad Opening Name are ==, !=, contains, and doesn’t contain. For X, Y, and Area, the valid operators are: ==, !=, <, <=, >, and >=.
  • Input: Enter the value in the input field based on which you want to filter the data.
  • Logical operator: Select an operator from the last list for logical operation with any subsequent filters defined. The valid operators for logical operation are and and or.
