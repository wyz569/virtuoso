viaValidPurposes

layout viaValidPurposes string "list_of_valid_purposes"

Description

Lets you specify a list of valid purposes for via creation in automated modes: auto, fast edit, compute from shape: single, compute from shape: stack. The via is not be created even if one purpose of the overlap shapes does not belong to the list of valid purposes specified in this environment variable.

The default is drawing pin.

GUI Equivalent

Command

Create – Via; Mode: Auto, Fast Edit; Options

Field

Valid Purposes

Examples

envGetVal("layout" "viaValidPurposes")
envSetVal("layout" "viaValidPurposes" 'string "drawing")
