dynamicMeasurementFromCursor

layout dynamicMeasurementFromCursor boolean { t | nil }

Description

Displays measurement between the pointer and closest edge. It also displays the distance between the closest edges.

The default is nil. The dynamic measurement displayed is the distance between closest edges.

GUI Equivalent

None

Examples

envGetVal("layout" "dynamicMeasurementFromCursor")
envSetVal("layout" "dynamicMeasurementFromCursor" 'boolean t)
