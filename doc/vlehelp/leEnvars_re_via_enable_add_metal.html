viaEnableAddMetal

layout.ava viaEnableAddMetal boolean { t | nil }

Description

Adds a metal layer to the via when analyzing or creating vias using Auto Via Assistant.

The default is t.

GUI Equivalent

Command:

Assistants – Auto Via Assistant – Constraint Overrides – Show All – Add Metal

Field:

Enable

Examples

envGetVal("layout.ava" "viaEnableAddMetal")
envSetVal("layout.ava" "viaEnableAddMetal" 'boolean nil)
