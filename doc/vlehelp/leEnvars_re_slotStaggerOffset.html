slotStaggerOffset

layout slotStaggerOffset float rowspacing

Description

Specifies the spacing between the edges of slots in alternate rows.

The default is 0.0.

GUI Equivalent

Command:

Create – Slot, Edit – Advanced – Slot, Create – Shape – Path – Slot Options

Field:

Stagger

Slot Offset: Stagger

Examples

envGetVal("layout" "slotStaggerOffset")
envSetVal("layout" "slotStaggerOffset" 'float 1.0)
