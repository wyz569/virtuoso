Find Functionality

Use the find functionality to search for objects that have an attribute or property that matches your search criteria. You can choose from a list of conditions and logic to form the search criteria. For example, you can search for ROD objects, blockages, and boundaries based on the following criteria:

  • ROD objects: You can search for ROD names on labels, paths, polygons, rectangles, any shape, and text display objects.
  • Blockages: You can search for blockages by blockage type, layer name, and owner name. You can use the Zoom to Figure and Previous and Next buttons to navigate through all blockages. However, you cannot replace any of the values based on search results.
  • Boundaries: You can search for different types of boundaries in the following manner:
    • PR and snap boundary by object type
    • Area boundary by boundary name and property
    • Cluster boundaries by boundary name, cluster name, and property

    You cannot replace any of the values based on the search results.

For information about the complete list of conditions and logic to use when searching for objects, see Object Search Criteria.
