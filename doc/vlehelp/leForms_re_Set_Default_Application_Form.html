Set Default Application Form

Use the Set Default Application form to set the default application for opening the current cellview and all cellviews with the same view type as the current cellview.

Field Description

Current application

Displays the current application in which the cellview is open.

Current cellview default

Displays the default application set for the current cellview.

Current default for all maskLayout cellviews

Specifies the default application set for all cellviews with the view type of the current cellview.

Set default application for current cellview to

Sets the default application for the current cellview.

  • As Is: Retain the current application setting.
  • None: Removes any default application setting.
  • <anyRegisteredApplication> Sets the selected registered application, such as Layout XL, as the default application for the current cellview.

To apply this setting, the saveViewSubType environment variable must be set to nil, otherwise the Layout cellview automatically opens in the same application in which it was last saved.

Set default application for all layout cellviews to

Sets the default application for all cellviews with the view type of the current cellview.

  • As Is: Retain the current application setting.
  • None: Removes any default application setting.
  • <anyRegisteredApplication> Sets the selected registered application, such as Layout XL, as the default application.

Environment variable: maskLayoutDefaultApp

To apply this setting, the saveViewSubType environment variable must be set to nil, otherwise the Layout cellview automatically opens in the same application in which it was last saved.
