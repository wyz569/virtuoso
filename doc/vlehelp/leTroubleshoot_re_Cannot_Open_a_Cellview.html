Cannot Open a Cellview

If you cannot open a cellview in a library, you might not have the read access to the cellview files or the library might be empty. In these cases, the Open File form displays <None> in the Cell Name and View Name fields.

To gain read access to the cellviews in a library, do one of the following:

    • Change the access permissions using the Access Permissions form. You can access the form by choosing Tools – Library Manager – Edit – Access Permissions in the CIW.
    • Use the UNIX command chmod to change the access permissions in the UNIX directory containing the library.
