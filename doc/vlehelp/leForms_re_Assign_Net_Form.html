Assign Net Form

Use the Assign Net form to assign nets to shapes, instance pins, and routed objects like vias and pathsegs. You can access the form by pressing F3 when the leHiAssignNet function is active.

Field Description

Net Name

Specifies the net name to be assigned. You can select a net name from the list or type a name. If the specified net does not already exist, a new net is created by that name. You can also use the icon displayed next to this field to pick an existing net name by clicking a shape in the canvas.

Pop up selection window

Displays a form in case of overlapping shapes. The form lists all the shapes on various layers, with details of the assigned net name. You can use this form for both picking and assigning a net.

Propagate connectivity

Propagates connectivity to physically connected shapes, vias, instance pins, and pathSegs during net assignment. This option follows Extractor settings specified for the constraint group and the Extract connectivity to level option in the Connectivity form.

Environment variable: propagateAssignNet
