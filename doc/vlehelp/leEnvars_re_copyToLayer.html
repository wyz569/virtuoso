copyToLayer

layout copyToLayer boolean { t | nil }

Description

Specifies whether shapes are copied to a different layer with leHiCopy. This does not change the form field in the form.

The default is nil.

GUI Equivalent

Command:

Edit – Copy

Field:

Change To Layer

Examples

envGetVal("layout" "copyToLayer")
envSetVal("layout" "copyToLayer" 'boolean t)
