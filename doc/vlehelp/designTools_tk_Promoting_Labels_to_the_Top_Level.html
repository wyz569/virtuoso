Promoting Labels to the Top Level

Use Promote Label to Top mode in the Pad Opening Info form to promote generated labels to the top level of a cellview. You can also control how the labels are drawn on the canvas and whether the corresponding lower-level labels should be removed.

  1. Open a hierarchical cellview that contains bond pads.
  2. Choose Tools – Pad Opening Info.
    The Pad Opening Info form is displayed.
  3. Select the Promote Label to Top radio button at the top of the form.
  4. Set the Search and Sort criteria as required.
  5. In the Promote Label section:
    1. Choose the object type as Label, Text Display, or Both.
    2. Select the appropriate Layer/Purpose where the labels need to be printed.
    3. Set the Font, Height, and Justification for the labels as required.
    4. Choose whether to Remove Original Labels.
  6. Click Apply.
    The required labels are promoted to the top level of the cellview.
