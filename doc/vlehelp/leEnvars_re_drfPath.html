drfPath

graphic drfPath string path

Description

Specifies the path to the Display Resource File.

The default is none.

GUI Equivalent

None

Examples

envGetVal("graphic" "drfPath")
envSetVal("graphic" "drfPath" 'string "~/.display.drf")
