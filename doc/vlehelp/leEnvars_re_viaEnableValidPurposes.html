viaEnableValidPurposes

layout viaEnableValidPurposes boolean { t | nil }

Description

Lest you specify a list of valid purposes for via creation in automated modes: auto, fast edit, compute from shape: single, compute from shape: stack.

The default is nil.

GUI Equivalent

Command:

Create – Via; Mode: Auto, Fast Edit; Options

Field:

Valid Purposes

Examples

envGetVal("layout" "viaEnableValidPurposes")
envSetVal("layout" "viaEnableValidPurposes" 'boolean t)
