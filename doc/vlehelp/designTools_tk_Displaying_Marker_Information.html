Displaying Marker Information

Use the Verify – Markers – Explain command to view the information about a marker. You can select a marker either before or after starting the command.

To view the reason or explanation associated with markers:

  1. On the layout canvas, select a marker. To select multiple markers, draw a selection box around them.
  2. Choose Verify – Markers – Explain.
    The marker text window appears. It displays the information about the error.
  3. Continue clicking the markers for which you want information.
    The text scrolls in the marker text window so you can see it.
  4. Press Esc to end the command when you are finished viewing the errors and warnings.
    You can correct the errors and then run the command again for verification.
