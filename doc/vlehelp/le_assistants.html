Layout XL Assistants

Virtuoso® Layout Suite XL provides various assistants to enable you to work on your designs effectively and efficiently. The following table provides an overview of the assistants.

Assistant Description

Auto Via Assistant

Lets you analyze vias in your design to check for missing vias. It also lets you create local overlaps with no vias.

This assistant is available in Virtuoso Layout Suite EXL and higher tiers only.
