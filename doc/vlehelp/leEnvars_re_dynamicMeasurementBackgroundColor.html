dynamicMeasurementBackgroundColor

layout dynamicMeasurementBackgroundColor string color

Description

Lets you select the background color for the dynamic measurement labels.

The default is black.

GUI Equivalent

None

Examples

envGetVal("layout" "dynamicMeasurementBackgroundColor")
envSetVal("layout" "dynamicMeasurementBackgroundColor" 'string "white")
