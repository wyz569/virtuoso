Creating an Array by Specifying Spacing Mode and Values

To create an array of copied objects by specifying spacing mode and spacing values in the Copy form:

  1. On the layout canvas, select the object to copy.
  2. Choose Edit – Copy and press F3 to open the Copy form.
  3. With the Array check box selected as the copy type, in the Rows and Columns fields, type the number of rows and columns you want in the array.
  4. Select one of the following modes in the Spacing section and specify the spacing value:
    1. Pitch: Select the Pitch radio button and specify the spacing in the Delta X,Y fields.
    2. Spacing: Select the Spacing radio button and specify the spacing in the Spacing X,Y fields.
    3. Absolute: Select the Absolute radio button and specify the spacing in the X,Y fields.
  5. Click Apply in the Spacing section.
    An array with the specified settings is created. The object you copied is part of the array.
