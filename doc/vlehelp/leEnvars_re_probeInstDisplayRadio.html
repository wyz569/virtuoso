probeInstDisplayRadio

graphic probeInstDisplayRadio string { "instancesOnly" | "instancesAndNet" }
layout probeInstDisplayRadio string { "instancesOnly" | "instancesAndNet" }

Description

Specifies whether the nets associated with probed or selected instances are also highlighted.

The default is instancesOnly.

GUI Equivalent

Command:

Options – Highlight

Field:

From Selected Objects (Pins, Instances, Shapes)

Examples

envGetVal("graphic" "probeInstDisplayRadio")
envSetVal("graphic" "probeInstDisplayRadio" 'string "instancesAndNet")
envGetVal("layout" "probeInstDisplayRadio")
envSetVal("layout" "probeInstDisplayRadio" 'string "instancesAndNet")
