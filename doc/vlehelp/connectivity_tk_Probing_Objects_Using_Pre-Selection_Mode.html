Probing Objects Using Pre-Selection Mode

To probe objects using pre-selection mode:

  1. In the layout window, choose Options – Highlight.
    This displays the Highlight Options form.
  2. Select the Flightlines check box.
  3. Select the object that you want to probe. For example, select Pins to highlight pins on the probed/selected nets.
  4. In the cellview, select one, a few, or all objects irrespective of their connectivity.
  5. Enter the following SKILL function in the CIW:
    geEnterAddNetProbe()
    Objects with the same connectivity are probed with the same color.
  6. Repeat steps 3 and 4 to probe more objects, if required.
    Use the command geDeleteAllProbe in the CIW to remove the probes.
