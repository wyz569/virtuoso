Setting an Area Display

To set the display detail for an area:

  1. In the layout window, choose View – Area Display – Set.
    The Set Area View Level form opens.
  2. Set the From and To display levels for the area.
  3. On the canvas, create a box around the area in which you want to view the detail of the lower levels of the design.

To remove one detail area at a time:

  1. Choose View – Area Display – Delete.
  2. Click the detail area you want to remove.

To remove all detail areas:

  • Choose View – Area Display – Delete All.
