Layout Editor Basics

The Virtuoso® Layout Suite XL layout editor (Layout XL) is an editing tool that provides the basic creation and editing capabilities to work with layout designs. The layout editor lets you create custom layouts or edit any existing layouts. It lets you work with objects, such as rectangles, pathSegs, vias, and multipart paths, and hierarchical designs. You can also use the layout editor to view the connected shapes in a design.

For information about connectivity driven editing, see Virtuoso Layout Suite XL: Connectivity Driven Editing User Guide.

Before you start using the layout editor, it is good to become familiar with the basics of the Virtuoso Studio design environment, such as the Command Interpreter Window (CIW), cellviews, and workspaces. For information, see Virtuoso Studio Design Environment. For information on licensing in the Virtuoso Studio design environment, see Virtuoso Software Licensing and Configuration User Guide.
