Rename Command Name Form

In Layout EXL or a higher tier, use the Rename Command Name form to change the name, description, or group category of any recording.

Field Description

Command Name

Specifies the name of the recording.

Description

Specifies information about the recording.

The tooltip for a saved recording displays a brief description in the Capture and Relay assistant.

Group

Specifies the group under which you want to save your recording for easy access.

When you rename a recording, you can specify a new group category or you can choose a group from the existing group categories for the recording. You can also change the group name or description of any recording without changing the command name.
