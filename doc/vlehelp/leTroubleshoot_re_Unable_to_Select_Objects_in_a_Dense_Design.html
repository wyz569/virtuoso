Unable to Select Objects in a Dense Design

If you click objects in a dense design and the layout editor does not select the object you want, try any of the following:

  • If objects share the edge you chose, click again in the same place.
    The layout editor selects another of the overlapping objects.
  • If possible, move the cursor closer to another edge of the object and click it.
  • If possible, zoom in on the edge you want to select.
    Use the View – Save/Restore – Previous View command to zoom out again after selecting the object.
  • Use the Palette assistant to turn off selectability for:
    • Layers that overlap the object
    • Pins or instances, if a pin or instance overlaps the object
