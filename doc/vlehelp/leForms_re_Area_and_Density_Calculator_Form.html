Area and Density Calculator Form

Use the Area and Density Calculator form to compute the total area and the density of shapes with specified layers within a region.

Field Description

Object

Lets you specify the objects for which you want to calculate the area and density. You can select All (all objects) or Selected (only selected objects will be considered for area calculation). If an instance, group, via, or guard ring is selected, the area of the shapes inside the selected instance is calculated.

Layer

Lets you select the layer for which you want to calculate the area and density. You can select the layer pursose pair from the drop-down list or you can specify a layer purpose pair. You can select a single layer or multiple layers for area calculation. The syntax that you should use to specify the layer is:

  • Single Layer: Met1/drawing
  • Multiple Layers: Met1/drawing1 Met1/drawing2 Met2/drawing1
  • All Purpose: Met1
  • Use () to merge different layers: (Met1 M1) (Met1/drawing1 Met1/drawing2)

Hierarchy Range

Sets the levels of hierarchy to be considered for area and density calculation.

  • current cellView: Performs the calculation in the current edit cellview in the layout window.
  • current to bottom: Performs the calculation from the current edit cellview level to the bottom of the hierarchy.
  • current to stop level: Performs the calculation from the current edit cellview level to the level specified in the Display Levels Stop field in the Display Options Form.
  • current to user level: Performs the calculation from the current edit cellview level to the level that you specify.

Region
  • Entire CellView: Considers the entire cellview for area and density calculation. This option is selected by default.
  • Viewing Area: Considers the portion of the cellview that is displayed in the canvas at that instant for area and density calculation.
  • A Selected Boundary: Considers a selected boundary for area and density calculation.
  • Draw a Rectangle Region on the Canvas or Draw a Polygon Region on the Canvas: Enables you to specify a rectangular or polygonal area for area and density calculation. Select the required option, and then click in the canvas to specify the vertices of the rectangle or polygon.
