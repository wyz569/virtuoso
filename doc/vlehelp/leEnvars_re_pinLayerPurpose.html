pinLayerPurpose

layout pinLayerPurpose string { "any_valid_layer_purpose" | "" }

Description

Specifies the layer purpose on which a shape pin is placed. You can specify any valid layer purpose, such as "pin" or "drawing".

The default is " ".

GUI Equivalent

None

Examples

envGetVal("layout" "pinLayerPurpose")
envSetVal("layout" "pinLayerPurpose" 'string "pin")
