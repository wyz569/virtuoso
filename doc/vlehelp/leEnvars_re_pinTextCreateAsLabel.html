pinTextCreateAsLabel

layout pinTextCreateAsLabel boolean { t | nil }

Description

Specifies that a pin is created as a label or a text display.

The default is nil.

GUI Equivalent

Command:

Create – Pin – Create Label – Options – Set Pin Label Text Style

Field:

Text Options: Create As Label

Examples

envGetVal("layout" "pinTextCreateAsLabel")
envSetVal("layout" "pinTextCreateAsLabel" 'boolean t)
