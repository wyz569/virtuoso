Editing Cut Patterns

To edit a cut pattern:

  1. In the layout window, select a via array. For example, a via array of five rows and eight columns.
  2. Choose Edit – Basic – Properties.
    The Edit Via Properties form opens.
  3. Click the Edit Cut Pattern button in the Cut section.
    The Edit Cut Pattern form appears.
  4. Select Wall Paper as the editing method.
  5. Use the Paper Size left and down arrow buttons to reduce the rows and columns, for example, to three and three.
  6. Edit cut-by-cut by clicking on the squares to turn off the top row and the far-right column, as shown below:
  7. Click OK.
  8. Click Apply on the Edit Via Properties form.
    The pattern then repeats across the entire cut array, starting from the lower left corner of the array and repeating upwards and to the right.
