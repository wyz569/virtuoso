Deleting Objects

To delete objects:

  1. On the layout window canvas, choose the objects, edges, or corners you want to delete.
  2. Choose Edit – Delete or press Delete or click the Delete icon on the Edit toolbar. Alternatively, right-click any free space on the design canvas to display the Layout shortcut menu and select Delete.
    All selected objects are deleted. If you selected an edge or corner, the edge or corner is deleted and the object is redrawn.
    You cannot delete a point if it changes the shape in a way that the shape becomes invalid. For example, a path must have at least two points; if you try to delete one point of a two-point path, a warning dialog box is displayed. You must close the dialog box and choose a different point to delete.
  3. Press F3 to open the Delete form if you want to specify the type of objects to be deleted.
  4. Select the type of objects that you want to delete. You can choose to delete net interconnect and path segments and specify the device chains.
    If you select the Net Interconnect option, all ROD objects and shapes on the selected net, except pins, are deleted.
  5. Select the objects on the design canvas that need to be deleted.
  6. Click Cancel in the Delete form.

Canceling Delete

If you discover that you deleted the wrong object, choose Edit – Undo to restore the object.

Delete automatically repeats if you select the Delete command first and then choose objects. If delete automatically repeats, press Esc to cancel the command.

You can cancel an ongoing delete operation by pressing Ctrl + C.
