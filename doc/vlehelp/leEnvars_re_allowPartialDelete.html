allowPartialDelete

layout allowPartialDelete boolean { t | nil }

Description

Specifies if deletion of partially selected objects is allowed.

The default is t.

GUI Equivalent

None

Examples

envGetVal("layout" "allowPartialDelete")
envSetVal("layout" "allowPartialDelete" 'boolean nil)
