mppTemplate

layout mppTemplate string { "New" | "the_name_of_any_MPP_template_defined_in_your_technology_file" }

Description

Specifies the default for the MPP Template field in the Create Multipart Path form. You can choose New or the name of a specific MPP template you created.

The default is New.

GUI Equivalent

Command:

Create – Multipart Path

Field:

MPP Template

Examples

envGetVal("layout" "mppTemplate")
envSetVal("layout" "mppTemplate" 'string "cellTechLibName.mpp.templates.il")
