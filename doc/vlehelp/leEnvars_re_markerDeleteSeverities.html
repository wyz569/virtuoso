markerDeleteSeverities

graphic markerDeleteSeverities string { "info" | "acknowledgedWarning" | "warning" | "signedOffError" | "error" | "signedOffCriticalError" | "criticalError" | "fatalError" }

Description

Defines the default severities. By default, the environment variable is set to the following marker severities: info acknowledgedWarning warning signedOffError error signedOffCriticalError criticalError fatalError.

GUI Equivalent

Command:

Verify – Markers – Delete All

Field:

Severity

Examples

envGetVal("graphic" "markerDeleteSeverities")
envSetVal("graphic" "markerDeleteSeverities" 'string "info")
envSetVal("graphic" "markerDeleteSeverities" 'string "signedOffCriticalError")
envSetVal("graphic" "markerDeleteSeverities" 'string "fatalError")
