termDirection

layout termDirection cyclic { "input" | "output" | "inputOutput" | "switch" | "jumper" | "unused" | "tristate" }

Description

Specifies the default I/O type of a pin.

The default is inputOutput.

GUI Equivalent

Command:

Create – Pin

Field:

I/O Type

Examples

envGetVal("layout" "termDirection")
envSetVal("layout" "termDirection" 'cyclic "switch")
envSetVal("layout" "termDirection" 'cyclic "jumper")
envSetVal("layout" "termDirection" 'cyclic "tristate")
