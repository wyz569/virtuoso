Locked Group Arrays

To ensure design integrity, while creating group arrays, designers can be prevented from specifying spacing mode and values by enabling the copyGroupArrayLockXY environment variable. When copyGroupArrayLockXY is set to t, the options for specifying spacing mode and values for a group array appear disabled in the Copy form.

For an existing group array that has spacing defined, you can choose to lock the spacing settings so that users cannot modify them. You can lock an existing group array by selecting the Lock X,Y check box on the Attributes tab of the Edit Group Array Properties form. This disables the Mode options and X and Y fields, indicating that the spacing mode and spacing values for the group array cannot be edited.
