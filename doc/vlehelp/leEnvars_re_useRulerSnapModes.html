useRulerSnapModes

layout useRulerSnapModes boolean { t | nil }

Description

Defines whether snap modes specific to the ruler command are used when you invoke the command.

When set to t, ruler snap modes set in the rulerSnapMode and rulerSegSnapMode environment variables are used, and window snap modes are restored when you exit the ruler command. When set to nil, window snap modes are used for the ruler.

The default is nil.

GUI Equivalent

None

Examples

envGetVal("layout" "useRulerSnapModes")
envSetVal("layout" "useRulerSnapModes" 'boolean t)
