Capture and Replay Assistant

Use the Capture and Replay assistant to record a series of actions and later replay them in your designs easily as required. The capture and replay functionality is available in Layout EXL and higher tiers.

The following table describes the Capture and Replay assistant buttons and options.

Icon Command Description

Start recording

Lets you records the action that you perform in the current window.

Once you start recording, the term Recording is displayed on the status banner. Also, the button toggles to Stop recording .

Stop recording

Stops recording your actions and displays the Save Recording form.

Replay recording with saved points

Performs the actions from a recording at the exact same locations as saved in the recording.

Replay recording with user-entered points

Performs actions from a recording at the custom point that you specify as the starting point for the replay.

Group

Lists all the groups available for a set of recordings. You can specify a new group name or add the recording to an existing group using the Save Recording form.

If you select a group, it displays a list of recordings related to the group.

The default is All, which shows a list of all the recordings. You can change the default group using the carDefaultGroup environment variable.

When you open the Capture and Replay assistant in multiple windows, the Group field in one window remains synchronized with the Group field of another window. This means that the list of groups and the list of recordings are the same in multiple windows. However, you can select different groups for different windows.

Details

Displays the following details about a recording:

  • File name and location
  • Creation date and author
  • Command name and description
  • Name of the group under which the recording is categorized
  • SKILL function that you can use to replay the selected recording

You can access the Details option by right-clicking a recording.

Delete

Lets you delete a recording. You can access the Delete option by right-clicking a recording.

Rename

Lets you change the name, description, and group of a recording using the Rename Command Name form. You can access the Rename option by right-clicking on a recording.
