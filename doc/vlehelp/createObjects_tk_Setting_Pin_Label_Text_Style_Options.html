Setting Pin Label Text Style Options

To specify pin name characteristics:

  1. In the layout window, choose Create – Pin.
    The Create Pin form appears.
  2. Select the Create Label check box.
  3. Click the Options button.
    The Set Pin Label Text Style form appears.
  4. Select the font height and style.
  5. Select the text options.
    • Turning Drafting on prevents the label from being rotated more than 90 degrees.
    • The Overbar option determines how the underscore character is displayed.
  6. Select a valid layer name.
  7. Select a valid layer purpose.
    You must select a valid combination of layer name and purpose. In case, it is not a valid combination, an error message is displayed in the CIW while creating the text display.
  8. Select the justification for the pin name origin.
    The origin appears as a small square on the name when placed or selected.
  9. Select the orientation of the pin name.
    Once the pin is placed, an outline of the pin name appears.
