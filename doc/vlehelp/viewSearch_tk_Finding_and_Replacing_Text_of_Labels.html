Finding and Replacing Text of Labels

To search and replace labels in the current cellview:

  1. In the layout window, choose Tools – Find/Replace or press Shift+S.
    The Find/Replace form appears.
  2. Choose label from the Search for list.
  3. Click Add Criteria to add the search criteria.
    The first field in the criterion line shows text by default.
  4. To search for all the labels in the cellview, type * in the text field. You can specify multiple criteria to search for the required labels.
  5. Select the location where you want perform the search by choosing from the Search Hierarchy Range list. By default, the search is performed in the current edit cellview in the layout window but you can choose to perform the operation across different hierarchy levels.
  6. Click Find.
    The layout editor highlights all the labels in the cellview.
  7. In the Replace section, choose text from the first list.
  8. Choose one of the following from the second list:
    • Change text: Select this option and specify the text to replace the label text of all the labels matching the search criteria.
    • Add Prefix: Select this option and specify the text to be prefixed to all the labels matching the search criteria.
    • Add Suffix: Select this option and specify the text to be added at the end of all the labels matching the search criteria.
  9. To replace only one label, click Previous or Next until the label you want is highlighted and then click Replace.
  10. To replace all the highlighted labels, click Replace All.
