Splitting an Object

To split and stretch an object:

  1. On the layout canvas, select the object you want to split.
  2. Choose Edit – Advanced – Split.
  3. Click the points of the split line on the canvas.
    You can use the middle mouse button to toggle between the split segments. In the above example, the split will alternately highlight the path section interior to the split lines, and the two segments of the path exterior to the split lines.
    When the split shape is drawn clockwise, the Split command selects the parts of the object outside the split lines first. When the split shape is drawn counterclockwise, the command selects the parts of the object interior to the split lines first. In the case of a single split line, toggling with the middle mouse button will alternate between the sides of the object divided by the split line.
  4. Click the reference point for the stretch.
  5. Click the new location for the stretch.
