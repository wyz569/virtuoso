Object Reshaping

You can change the shape of a selected object by using the Reshape command. The following table describes the support available for reshaping objects.

Object Type Support for the Reshape Operation

ROD objects

Except for multipart paths and multipart rectangles, all ROD objects that can be reshaped remain ROD objects after a reshape operation. When you reshape any ROD object, all alignments to that object are deleted.

Rows

The Reshape command does not support rows.

Groups

The Reshape command does not support group objects.

Blockages

The Reshape command reshapes blockages in the same way it reshapes a polygon.

Boundaries

The Reshape command reshapes PR and cluster boundaries in the same way it reshapes a polygon, using points.

The Reshape command supports reshaping of IOBox and CoreBoxSpec when PR boundary is reshaped. The relative distance of IOBox and CoreBoxSpec from the PR boundary is maintained.

The Reshape command does not support snap boundary objects.
