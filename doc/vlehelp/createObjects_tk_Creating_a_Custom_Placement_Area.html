Creating a Custom Placement Area

You use a custom placement area for placement of standard cells and custom cells. You use the Create Custom Placement Area form to create a custom placement area. The custom placement area is automatically snapped to an appropriate grid depending on the state of the snapCpaToPlacementGrid environment variable.

To create a custom area for placement:

  1. In the layout window, choose Create – P&R Objects – Custom Placement Area.
    The Create Custom Placement Area form opens.
  2. Select the method for creating the custom placement area as Manual, Assisted, or Auto.
  3. Specify the attributes in the Definition section.
    • Type the name of the custom placement area in the Base Name field.
      When creating the custom placement area using Manual mode, you do not need to specify attributes such as the height, width, and orientation of the area.
    • Specify the Height and Width of the placement area, if using Assisted or Auto creation mode.
    • Specify Number of Placement Areas you want to create, if using Auto creation mode.
    • Specify the width between the placement areas in the Channel Width field.
    • Select the orientation of the placement area as Horizontal or Vertical.
    • Specify the flip and abut settings for the placement area. For example, you can set every other area to flip or set only the first area to flip, and specify the number of areas to abut or set to abut the first area.
  4. Specify the template you want to use, in the Templates section.
    Templates allow you to place rails and/or components automatically into the custom placement areas. Choose pre-existing rails, components, or both to create the custom placement area. If pre-defined rails or components exist, they are listed in the Pre-Defined box in the form. Select the ones you want to use for custom placement area creation one-by-one and move them to the Selected box by using the arrow icons.
    Template definitions are associated only with the cellview in which they are defined. Therefore, the rail and component templates available to one cellview are not available to another cellview.
  5. Draw the custom placement area on the canvas if you selected the Manual or Assisted create method or click on the canvas if you selected Auto.
