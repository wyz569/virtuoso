autoPinLayerFunctionsList

layout autoPinLayerFunctionsList string "list_of_layers"

Description

Specifies the list of layers for auto pin.

The default is an empty list.

GUI Equivalent

None

Examples

envGetVal("layout" "autoPinLayerFunctionsList")
envSetVal("layout" "autoPinLayerFunctionsList" 'string "metal1 metal2")
