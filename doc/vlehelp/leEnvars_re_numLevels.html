numLevels

layout numLevels int numberOfLevels

Description

Specifies the number of levels to copy in leYankFigs.

The default is 32.

GUI Equivalent

Command:

Edit – Basic – Yank

Field:

Yank Levels

Examples

envGetVal("layout" "numLevels")
envSetVal("layout" "numLevels" 'int 10)
