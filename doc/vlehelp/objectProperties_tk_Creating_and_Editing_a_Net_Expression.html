Creating and Editing a Net Expression

To create or edit a net expression:

  1. In the layout window, select the pin to whose terminal you want to assign a net expression.
  2. Choose Edit – Basic – Properties.
    The Edit Properties form for the pin opens.
  3. Click the Connectivity tab.
  4. Type the net expression override property name in the Net Expression field.
  5. Type the default net name in the Default field.
    The default net name is used when no override property is defined in the hierarchy above this point in the schematic view. If you do not enter a default net name, the terminal name is used.
  6. Click Apply.
    The pin label display changes to show the net expression property preceded by the @ symbol and the default net name preceded by the % symbol.

The Net Expressions field in the Display Controls section of the Display Options form must be enabled for the net expressions to display in your cellview.

Clearing the Net Expression field in the Edit Properties form deletes the net expression from your design.

If several pins in your design have the same terminal name, you need update only one pin and the rest will update automatically. You can see the edited text labels after you use View – Redraw.
