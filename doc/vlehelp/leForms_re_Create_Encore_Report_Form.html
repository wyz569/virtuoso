Create Encore Report Form

Use the Create Encore Report form to customize the report format to meet your requirements.

Field Description

Headers

Lets you customize the header information for your report.

  • >> button: Lets you move the selected elements from the left pane to the right pane.
  • Move Up and Move Down: Let you change the order of the elements in the right pane.
  • << button: Removes the selected elements from the right pane.

The available elements are:

  • Empty Line: Inserts a blank line.
  • Separator: Inserts a line of asterisks.
  • Title: Inserts the title “Pad Opening Locator”.
  • Created: Inserts the date and time the report was generated.
  • Libname: Inserts the name of the library.

Columns

Lets you specify what information is included for each pad shape in the report. The use model is the same as that described above for the Headers section. The available columns are:

  • Pad Opening Name: Specifies the label name (if any), otherwise NO_NAME.
  • X: Specifies the X coordinate of the center point of the pad shape.
  • Y: Specifies the Y coordinate of the center point of the pad shape.
  • Area: Specifies the area of the pad shape.
  • Width/Height: Specifies the width and height of the pad shape.

File Path

Lets you specify the location and file name for the generated report. Type the full path in the field provided, or use the Browse button to navigate to the required location on disk.

Filter

Lets you specify filters to limit the data shown in the report.

Click Set Filter to open the Pad Opening Info: Filter Form where you can specify the filters to be applied.
