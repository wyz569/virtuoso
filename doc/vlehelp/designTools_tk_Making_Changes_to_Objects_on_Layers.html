Making Changes to Objects on Layers

To enlarge copies of the objects selected:

  1. On the layout canvas, select the objects that you want to resize.
  2. Choose Tools – Layer Generation.
    The Layer Generation form opens.
  3. Fill out the Layer Generation form. Specify the input layer, Boolean operator, the resize value, and the output layer. The output layer can be same as the input layer.
  4. Click OK.
    Copies of the selected objects are enlarged.
