Moving an Object by Using the Move Command

To move an object using the Move command:

  1. In the layout window, choose Edit – Move or click the Move icon on the Edit toolbar.
    Alternatively, right-click at any free space on the design canvas to display the Layout shortcut menu and select Move.
  2. Press F3 to open the Move form.
  3. Select one or more objects. If you click a single object, the click point is taken as the reference point, which is the point from where the move starts. If you create a selection box or press Shift while selecting multiple objects, the Move command prompts you for a reference point.
  4. Click the reference point for the move operation, if prompted. Alternatively, you can specify the reference point for the move operation by using the Enter Points form.
  5. Click on the canvas where you want to move the objects. Alternatively, you can specify the destination point by using the Enter Points form.
