Flip Form

Use the Flip form to flip an object horizontally or vertically in the specified window..

Field Description

Specify reference and direction

Activates the options in the form for specifying the reference and the direction to be used for flipping an object.

Environment variable: useFlipReference

Reference

Specifies the point around which an object is flipped. The default is CenterCenter.

Environment variable: flipReference

X,Y

Specifies the X and Y coordinates around which an object is flipped. This field is available when you select Custom from the Reference list.

Flip around combined bbox

Flips multiple selected objects around the specified reference point of the combined bounding box of the objects.

Environment variable: flipAroundCombinedCenter

Direction

Specifies whether an object is flipped horizontally along the X axis or vertically along the Y axis.
