Making Hierarchy Editable

Use the Make Editable command to switch to edit mode for updating a hierarchy.

To make a hierarchy editable:

  1. In the layout window, choose Edit – Hierarchy – Make Editable.
    The Make Readonly/Editable form opens, listing master cellviews of all the instances in the current cellview.
  2. Use the Filter options if you want to specify a search criteria for listing only specific master cellviews.
    You can display the master cellviews of all or only the instances selected in the cellview. You can specify a name and type criteria for filtering master cellviews.
  3. Select a Readonly master cellview in the left pane and click Make Editable.
    Alternatively, select an Editable master cellview in the left pane and click Make ReadOnly. You can select multiple master cellviews and change their mode.
