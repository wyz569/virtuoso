Choose Object for Label Creation Form

Use the Choose Object for Label Creation form to choose a layer when clicking on a coordinate with objects on overlapping layers, if there are multiple overlapping shapes in the manual and auto label creation modes. You can select one or more objects for label creation.

Field Description

Label

Displays the label for the layer.

This field is available only if the label creation mode is auto.

Layer

Displays the layer for the label.

Purpose

Displays the purpose for the label.

Object

Displays the object.
