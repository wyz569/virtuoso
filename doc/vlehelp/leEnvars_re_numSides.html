numSides

layout numSides int numberOfSides

Description

Specifies the number of sides used to convert a conic to a polygon.

The default is 20.

GUI Equivalent

Command:

Options – Editor

Field:

Conic Sides

Examples

envGetVal("layout" "numSides")
envSetVal("layout" "numSides" 'int 100)
