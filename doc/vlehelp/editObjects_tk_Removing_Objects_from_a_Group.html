Removing Objects from a Group

To remove an object from a group:

  1. In the layout window, choose Edit – Hierarchy – Edit In Place.
    Alternatively, you can right-click the group to display the Group shortcut menu and select Edit In Place.
    You are prompted to point at a shape in the cellview to be edited in place.
  2. On the canvas, click the group you want to edit.
    The Edit – Group – Remove from Group remains grayed out until you have completed this step.
  3. Choose Edit – Group – Remove from Group.
    You are prompted to select the figures to be removed from the current group.
  4. Click the object you want to remove from the group.
    The object is removed from the group. The solid line surrounding the group changes to exclude the removed object. Similarly, you can continue to click other objects to remove them from the group.
  5. Click the Esc key to end the command.
  6. Choose Edit – Hierarchy – Return to Level or right-click anywhere on the design canvas and select Return from the Layout shortcut menu.
    The dotted line surrounds only the remaining objects when you move your pointer over the group.
