Drawing a Multipart Path in the Layout Window

You can you draw a multipart path (MPP) on your layout canvas based on the template selected or the values specified in the Create Multipart Path and ROD Subpart forms.

MPP templates do not contain net names because each MPP you create from the same template could be on a different net. When you choose an MPP template that has pin connectivity defined, the system displays a message at the bottom of the cellview window prompting you to enter a valid net name. If the MPP template does not have pin connectivity, and you want to associate the MPP with a net, you need to type the net name in the Net Name field for the master path or for any subpart, as desired.

When there is pin connectivity, you must enter the name of the net in the Net Name field for each part (master path and/or subparts) before you begin to create the MPP in the layout window. If you click in the layout window before entering the net names, the system beeps.

As you draw an MPP, you can change the values in the Create Multipart Path and ROD Subpart forms and see immediate changes to the MPP, until you double-click to enter the last point. After you enter the last point to complete an MPP, you can no longer change that MPP using the Create forms. You will need to use the Edit MPP Path Properties form to modify a completed MPP.

To draw an MPP in the layout window:

  1. Ensure you have specified the required values in the Create Multipart Path and ROD Subpart forms.
  2. Click on the canvas to enter the first point.
  3. Move the pointer to the next point and click on the canvas.
  4. Continue to move the pointer and click to enter points.
  5. To enter the last point of the path, either double-click on the canvas or press Enter.

You have completed drawing the MPP in your layout window.
