viaCompuEnclosures

layout viaCompuEnclosures boolean { t | nil }

Description

Allows you to automatically calculate enclosures.

The default is t.

GUI Equivalent

Command

Create – Via

Field

Enclosures: Compute

Examples

envGetVal("layout" "viaCompuEnclosures")
envSetVal("layout" "viaCompuEnclosures" 'boolean nil)
