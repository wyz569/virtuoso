instLibName

layout instLibName string "libraryName"

Description

Specifies the default name of the library for leHiCreateInst.

The default is none.

GUI Equivalent

Command:

Create – Instance

Field:

Library

Examples

envGetVal("layout" "instLibName")
envSetVal("layout" "instLibName" 'string "MyLib")
