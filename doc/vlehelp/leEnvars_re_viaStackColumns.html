viaStackColumns

layout viaStackColumns int numberOfColumns

Description

Specifies the number of columns in the top via of the via stack.

The default is 1.

GUI Equivalent

Command:

Create – Via; Mode: Stack

Field:

Top Columns

Examples

envGetVal("layout" "viaStackColumns")
envSetVal("layout" "viaStackColumns" 'int 10)
