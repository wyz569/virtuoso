remasterInstSearchCellText

layout remasterInstSearchCellText string { "any_cell_name }

Description

Specifies the name of the cell that contains the instances you want to use to re-master the instances specified by remasterInstUpdateCellText.

GUI Equivalent

Command:

Tools – Remaster Instances

Field:

Search for: CellName

Examples

envGetVal("layout" "remasterInstSearchCellText")
envSetVal("layout" "remasterInstSearchCellText" 'string "MyCell")
