Save Multipart Path Templates Form

Use the Save Multipart Path Templates form to save the values currently in the Create Multipart Path and ROD Subpart forms as a template with a unique name.

Field Description

Template List

Lists the existing templates.

ASCII Fie

Saves the template definition to an ASCII file.

The extension of the template file can be .il or you can use any other extension.

Technology Library

Saves the template definition to the selected technology library.
