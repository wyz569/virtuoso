pathBeginExt

layout pathBeginExt float pathExtension

Description

Specifies the beginning extension of paths with variable end type.

The default is 0.0 user units.

GUI Equivalent

Command:

Create – Shape – Path and Create – Wiring –Wire

Field:

Begin Extension and Begin

Examples

envGetVal("layout" "pathBeginExt")
envSetVal("layout" "pathBeginExt" 'float 1.0)
