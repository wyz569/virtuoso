Rotate Form

Use the Rotate form to change the orientation of geometric objects.

Field Description

Specify reference and angle

Activates the following options in the form for specifying the reference for rotating an object: Reference, X,Y, and Flip around combined bbox.

Environment variable: useRotateReference

Reference

Specifies the point around which an object is rotated. The default is CenterCenter.

Environment variable: rotateReference

X,Y

Specifies the X and Y coordinates around which an object is rotated. This field is available when you select Custom from the Reference list.

Rotate around combined bbox

Rotates multiple selected objects around the specified reference point of the combined bounding box of the objects.

Environment variable: rotateAroundCombinedCenter

Angle of Rotation

Sets the angle for rotating the object you select. Valid values are:

  • 90: Turns the object, instance, or label 90 degrees counterclockwise.
  • 180: Mirrors the object about the Y axis, or turns an instance or label 180 degrees counterclockwise.
  • 270: Mirrors the object about the X axis, or turns an instance or label 270 degrees counterclockwise.

Snap Mode

Specifies the snap mode for rotating the objects. Valid snap modes are anyAngle, diagonal, orthogonal, horizontal, or vertical.

Precision

Controls the precision of the angle of the snap.

  • 1 Degree: Lets you use whole numbers; for example, 10.
  • .1 Degree: Lets you use one decimal place; for example, 12.5.
  • Any: Lets you use three decimal places; for example, 15.125, assuming 1,000 database units/user units.
