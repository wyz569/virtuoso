probeCycleBool

graphic probeCycleBool boolean { t | nil }

Description

Specifies whether to cycle the probe layer-purpose pairs. If cycling, the current layer-purpose pair for the probe is picked from probeHiliteLppList.

The default is t.

GUI Equivalent

Command:

Options – Highlight

Field:

Cycle check box in the Display Layer section

Examples

envGetVal("graphic" "probeCycleBool")
envSetVal("graphic" "probeCycleBool" 'boolean nil)
