Route Status of Objects

To determine the route status of vias, paths, and pathsegs, you can use the following two attributes in the Property Editor assistant:

  • Route Status: It is editable and can have the following values:
    • Routed indicates normal routing status.
    • Fixed indicates a route that should not be edited automatically, though it can be edited manually.
    • Locked/Cover indicates a route that should not be edited either by automatic tools or manually. The route forms a part of the base array. To edit the route, change the routing status to Routed or Fixed.
  • Part of a Route Status: It is a non-editable attribute. It is set to FALSE if the selected via, path, or pathSeg is not part of a route and set to TRUE if the via, path, or pathSeg is part of a route.

The Create Wire command in Layout XL creates vias and pathSegs that are in route.
