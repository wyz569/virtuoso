Save/Restore Selection Set Form

Use the Save/Restore Selection Set form to save objects selected in the canvas as a selection set so that you can restore them later.

Field Description

Name

Enables you to specify the name of the selection set under which you want to save the objects selected in the canvas.

Store

Enables you to save the selection set information to a file for later retrieval.

If the Store button does not work, the selection set is empty or the name you entered in the Name field already exists. A warning is issued in CIW.

Stored Selection Sets

Lists the saved selection sets.

The selection set name you entered in the Name field is listed here once you click the Store button.

Select

Retrieves the stored information and displays it on the originally selected set.

You must click the stored selection set prior to clicking the Select button to bring up the stored selection set you want to display. Each selection you make adds additional selection to the display unless you click the Deselect button in the form before making another selection.

Deselect

Removes selection from the stored set from the objects in the stored selection set you have selected to display in the cellview.

Delete

Removes the set that you have selected in the Stored Selection Sets list box. If the deleted set is the current selection set, the selection is removed from the form, but remains displayed until you change the selection.
