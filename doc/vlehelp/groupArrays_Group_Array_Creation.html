Group Array Creation

You can create a group array in any of the following ways:

  • While copying objects by using the Copy form
  • While generating selected schematic instances and pins directly in a layout view by using the Generate Selected Components form
  • While generating clones by using the Generate Clones form
  • Using the lxCreateGroupArray SKILL function
