Descend Options Form

Use the Descend Options form to enable the option to be able to choose the view opened when descending into instances.

Field

Description

Prompt For View Name

Enables the Descend form to appear when you click an instance. You can then choose a different view to open. If there is only one view of the master cell, Descend does not prompt you for a view name.

By default, Descend descends into the same view name as the current cellview.

Open in

Lets you select to open the cellview in a new tab, the current tab, or a new window.
