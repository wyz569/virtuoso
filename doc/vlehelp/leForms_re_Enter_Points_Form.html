Enter Points Form

Use the Enter Points form to create an object at specific coordinates in the design window.

All point adjustment commands, such as DRD, Grid, Snap Mode, Acute Angle, and Gravity are disabled when the points are entered using the Enter Points form.

Field Description

XY

Enables you to specify the points for an object, such as a rectangle, a polygon, an arc, a path, a circle, an ellipse, a donut, geometric wires, instances, pins, labels, vias, multipart paths, fluid guard rings, MPP guard rings, slots, and PR objects. The object appears on the canvas at the points specified in this field. In this field you can also specify points for commands such as, Pan, Zoom, Move, Copy, Chop, and Edit In Place.

Use Relative XY

Indicates that the points specified in the XY field are the points relative to the last entered points of the command (delta X and Y).
Environment variable: useRelativeXY
