System Evaluation of Environment Variables

The system looks for the value of an environment variable as follows:

  • If no layout license is available, the system looks for the system-defined default settings in the following file:
    install_dir/etc/tools/graphic/.cdsenv
  • If a layout license is available, the system looks for the system-defined default settings in the following file:
    install_dir/etc/tools/layout/.cdsenv
  • The system loads any values in the local .cdsenv file in your home directory,
    ~/.cdsenv
    The layout variables supersede any matching graphic variables. Environment variables defined in your .cdsenv supersede any default values in the graphic or layout .cdsenv.
There is a precedence of window settings that might override environment variable settings. The settings are read in the following order: cellview properties, library properties, technology library properties, and environment variable values. Therefore, any override window properties stamped on a cellview will take precedence over a setting in the .cdsenv file.
