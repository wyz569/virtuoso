Edit Synchronous Clone Form

Use the Edit Synchronous Clone form to automatically replicate changes in all the clones in the synchronous figure group. If you make a change to the physical topology of one member of a synchronized family, that change is propagated to all the other clones in the family.

Field Description

Do not show me this dialog again

Prevents the Edit Synchronous Clone form from being displayed the next time you edit in place (EIP) a synchronous clone figure group.
