nPtsToTaper

layout nPtsToTaper int numberOfSegments

Description

Specifies the default number of segments in the taper. This is only used for exponentially tapered lines.

The default is 20.

GUI Equivalent

Command:

Create – Microwave – Taper

Field:

Resolution

Examples

envGetVal("layout" "nPtsToTaper")
envSetVal("layout" "nPtsToTaper" 'int 30)
