probeHiliteLppList

graphic probeHiliteLppList string "lppList"

Description

Lets you specify the layer-purpose list to be used for probe.

The default value is hilite drawing hilite drawing1 hilite drawing2 hilite drawing3 hilite drawing4 hilite drawing5 hilite drawing6 hilite drawing7 hilite drawing8 hilite drawing9.

GUI Equivalent

Command:

Options – Highlight

Field:

LPP drop-down list in the Display Layer section

Examples

envGetVal("graphic" "probeHiliteLppList")
envSetVal("graphic" "probeHiliteLppList" 'string "x y z")
