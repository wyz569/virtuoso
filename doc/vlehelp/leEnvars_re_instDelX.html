instDelX

layout instDelX float columnDistance

Description

Specifies the distance between the columns of an arrayed instance or contact measured from edge to edge. This field is grayed out if the number of columns specified is 1.

The default is 0.

GUI Equivalent

Command:

Create – Instance

Field:

Mosaic: Delta X

Examples

envGetVal("layout" "instDelX")
envSetVal("layout" "instDelX" 'float 1.0)
