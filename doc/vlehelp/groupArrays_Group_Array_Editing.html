Group Array Editing

While working with a group array, you can perform basic functions such as stretching, moving, chopping, or ungrouping the group array. The drag mode set in the groupArrayDragMode environment variable is used while stretching a group array.

You can edit a group array by using the Edit Group Array Properties form, the Property Editor assistant, or the lxUpdateGroupArrayParams SKILL function. You can modify the following attributes of a group array:

  • Number of rows and columns
  • Spacing mode and values
  • Orientation of individual cells
  • Holes in the pattern
  • Lock status of the group array, whether users can modify spacing mode and values
