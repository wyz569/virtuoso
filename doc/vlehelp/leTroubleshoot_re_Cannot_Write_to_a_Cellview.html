Cannot Write to a Cellview

If you have write access to a library but cannot open a cellview for editing, one of the following might be true:

  • You do not have write access for the cellview file.
  • Another user is editing the cellview and has locked it.

If the owner of the cellview set privileges to read only, you see a message like this:

The cellView could not be opened for edit. Do you want to open it for read?

To gain write access to the cellviews in a library, do one of the following:

    • Change the access permissions using the Access Permissions form. You can access the form by choosing Tools – Library Manager – Edit – Access Permissions in the CIW.
    • Use the UNIX command chmod to change the access permissions in the UNIX directory containing the library.
