Design Review Editor Form

Enables to review a design by using a checklist and adding defect markers.

Field Description

Name

Specifies the name of the review event.

Checklists

This section provides options to use checklists in reviews.

Checklists Drop-down list

Lists the user-defined checklist names.

Text field for Checks

Shows the list of checks available in the chosen checklist. You can select the check boxes displayed in the text field to indicate the checks being used or double-click them to indicate the checks being ignored.

Defects

This section provides options to add the defect markers and update their status.

Text field for Defect Name

Specifies the description of a defect.

Add Defect Marker

Creates defect markers in a design.

Text field for list of Defects

Displays the list of defects and lets you edit, add notes, or change status of the defects.

Review Status

This section provides options to specify and update review status.

Status Drop-down list

Shows the current review status.

Text field for Comments

Lets you add comments about the review status.
