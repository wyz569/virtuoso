Create Defect Form

Enables to review a design by using a checklist and adding defect markers.

Field Description

Reference Mode

Specifies various reference modes or points on the canvas to create defect markers. The default value is Object.

  • Point - Adds defect markers on specific coordinates.
  • Object - Adds defect markers on an object.
  • Rectangle - Adds defect markers on a rectangular region.
  • Polygon - Adds defect markers on a polygon region.

Defect Color

Specifies the color of defect markers.
