awvUpdateWindow

awvUpdateWindow(
w_windowID
)
=> t / nil

Description

Updates the display of all subwindows whose update statuses are turned on.

Arguments

w_windowID

Waveform window ID

Value Returned

t

Subwindows of the specified Waveform window are updated successfully.

nil

The specified Waveform window does not exist.
