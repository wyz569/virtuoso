Setup Library Task Assistant
Product Version IC23.1, November 2022

How do I create my setup in Data View using Setup Library?

Drag a specific sweeps or corners setup from the Setup Library assistant into the Data View assistant. The setup elements that you drag from the Setup Library assistant appear with a green background in the Data View assistant. 

Note: When you hover the pointer on sweeps and corners in the Data View assistant, a tooltip appears to display the name of the referenced setup from Setup Library.
