How do I synchronize Data View with Setup Library?

If you change the value of referenced sweeps or corners in Data View or Setup Library, they appear in bold in the Data View.

To synchronize the ADE Assembler setup to reflect the latest updates:

  • Right-click the Global Variables or Corners section in Data View assistant and choose Sync with Reference Setup.
