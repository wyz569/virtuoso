Tools Menu in the CIW

The Tools menu provides access to commands to start the Library Manager and your design software and utilities. The commands available on this menu varies with the binary you are running.

Menu Description

Library Manager

Opens the Library Manager.

Library Path Editor

Opens the Library Path Editor.

SystemVerilog

Opens the SystemVerilog Netlister form.

NC-Verilog

Opens the Virtuoso Verilog Environment for NC-Verilog.

VHDL Toolbox

Opens the VHDL Toolbox.

ADE Assembler

Opens ADE Assembler by creating a new view or opening an existing view.

ADE Explorer

Opens ADE Explorer by creating a new view or opening an existing view.

ADE Verifier

Opens ADE Verifier by creating a new view or opening an existing view.

VIVA XL

Opens Virtuoso Visualization and Analysis XL window to plot and analyze the simulation results.

AMS

Displays a submenu of commands for setting options and netlisting for the Cadence AMS simulator.

Technology File Manager

Opens the Technology File Tool Box window that lets you compile, store, and edit technology data.

Display Resource Manager

Opens the Display Resources Tool Box.

Abstract Generator

Opens the Abstract Generator form.

Print Hierarchy Tree

Displays a list showing the hierarchy of cellview instances in the selected cellview.

Set Cell Type

Displays the Set Cell Type form.

Check Pin Accessibility

Opens the Pin Accessibility Checker form.

Migrate

Displays a submenu of commands that operate on various Schematic and Layout Migration operations.

CDF

Displays a submenu of commands that operate on Component Description Format.

Diagnostic Center

Opens the Diagnostic Center form.

SKILL IDE

Opens the Cadence SKILL IDE window.

SKILL API Finder

Opens the SKILL Finder window.

Conversion Tool Box

Opens the Conversion Tool Box.

Uniquify

Displays the Uniquify form.

Virtuoso Multi Technology

Opens the Virtuoso Multi Technology Enablement form.
