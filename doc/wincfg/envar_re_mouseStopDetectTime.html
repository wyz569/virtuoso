mouseStopDetectTime

ui mouseStopDetectTime int time

Description

Measures the duration of a mouse stopped at a point long enough to classify it as inactive.

The default value is 100.

GUI Equivalent

None

Examples

envGetVal("ui" "mouseStopDetectTime")
envSetVal("ui" "mouseStopDetectTime" 'int 2)
