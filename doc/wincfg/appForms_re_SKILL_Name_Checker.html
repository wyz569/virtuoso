SKILL Name Checker

Use this form to check for name collisions in the loaded contexts.

Field Description

Select contexts for checking name conflicts

Specifies a valid context file to check the name conflicts.

  • Add: Moves the specified file name from the browser field to the list area.
  • Delete: Removes the selected file from the list area.
