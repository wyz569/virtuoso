Add Bookmark Form

Use this form to add bookmarks to your session window.

Field Description

Bookmark

Specifies either to create a new bookmark or add to already existing bookmarks.

Name

Displays the name of the current design cellview or cellviews. You can type a different name in this field to personalize the name of your bookmark.

Description

Provides a longer description for your bookmark. This text appears in the Description column of the Bookmarks Manager window.

Add to Bookmarks Toolbar

Adds the bookmark to the Bookmarks toolbar.

Bookmark All Tabs

Bookmarks all the open tabs in your session window as a composite bookmark.
