Diagnostic Center Form

Use this form to troubleshoot any Virtuoso or non-Virtuoso related issues in the current session.

Tab Description

Performance

Contains various debugging techniques to detect issues that might have caused the current session to slow down or freeze.

Data Integrity

Checks for inconsistencies in the contents of a library or cellview, and optionally repairs them.

Performance

The following table describes the fields available on the Performance tab of the Diagnostic Center form.

Field Description

Health Monitor

Monitors the performance and activites of Virtuoso, with an option to record callstacks for reporting the issues.

Check

Performs various checks to search for the root cause affecting the performance of the current session.

Library

Checks the design library access time with multiple pings.

Design

Checks specified designs or all designs in a library for better performance.

Environment

Checks system environment or environment variables for better performance.

SKILL

Performs SKILL profiling.

Data Integrity

The following table describes the fields available on the Data Integrity tab of the Diagnostic Center form.

Field Description

Check

Checks for inconsistencies in the contents of a library or cellview, and optionally repairs them.

Layout Integrity

Checks for layout integrity issues, such as missing instance masters or via definitions.

Constraints Integrity

Checks for constraint integrity issues, such as invalid or unreferenced constraints.

Database Integrity

Checks for data integrity issues using the oaScan features. It provides two options:

  • Scan Library
  • Scan Hierarchy
