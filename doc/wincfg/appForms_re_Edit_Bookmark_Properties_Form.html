Edit Bookmark Properties Form

Use this form to edit bookmark properties.

Field Description

Name

Displays the name of the current design cellview or cellviews. You can type a different name in this field.

Description

Specifies a description for your bookmark. This text appears in the Description column of the Bookmarks Manager window.

Add to Bookmarks Toolbar

Adds the bookmark to the Bookmarks toolbar.
