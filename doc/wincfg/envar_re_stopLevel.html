stopLevel

ui.thumbnails stopLevel int stopLevel

Description

Specifies the stoplevel of the thumbnail in the cellview window.

The default value is -1.

GUI Equivalent

None

Examples

envGetVal("ui.thumbnails" "stopLevel")
envSetVal("ui.thumbnails" "stopLevel" 'int 1)
