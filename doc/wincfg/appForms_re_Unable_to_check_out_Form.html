Unable to check out Form

When all available licenses have been checked out, this form lists the licenses that are currently in use, and also displays the list of active users.

Field Description

Product

Identifies the product number that cannot be checked out.

Licenses

Specifies how many licenses are available for the application.

User

Idenitifies current users who have a license checked out for the application.

Host

Specifies the host system where the application resides.

Product Description

Provides the name or short description of the application.
