Close Opened Cellviews Form

Use this form to view the currently open cellviews and allows you to save and close them.

Field Description

Library

Lists the name of the library for which you have opened cellviews for viewing or editing.

Cell

Displays the name of all the cells corresponding to the library that you have opened for viewing or editing.

View

Displays the names of the cellviews that you have opened.

Status

Displays the status of the opened cellviews. If the cellviews have not changed, the Status field remains blank. If the cellviews have been changed, the Status field displays modified.

Save modified cellviews

Allows you to save the opened cellviews before defragmentation. All modified cellviews are saved by default.
