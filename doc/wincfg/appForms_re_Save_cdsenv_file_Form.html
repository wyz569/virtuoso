Save .cdsenv file Form

Use this form to specify how modified environment variables are saved.

Field

Description

Save to filename

Specifies the .cdsenv file to which you want to save the changes.

Saved tools

Enables you to specify the tools for which you want to save the environment variables. You can select:

  • all possible Saves environment variable updates for all possible tools, includes both loaded and unloaded tools.
  • all loaded Saves environment variables updates for tools that have been loaded.
  • loaded by name Saves environment variables updates for only those tools that have been selected in the Loaded Tools tree.

Saved variable states

Specifies the variables you want to save.

  • modified Saves only those variables whose values have been edited.
  • modified and different from default Saves updates to only those environment variables that have been modified and their current value is different from the default value.
  • all Saves all environment variables in the selected tools.

Saved file status

Specifies the action taken when an existing .cdsenv file is updated:

  • overwrite Overwrites the existing file.
  • merge (update file with most recent values) Updates are merged in the existing file. Environment variables in the file are updated with the most recent value in the Cdsenv Editor.
  • retain (update file with additional information - retain file data) Updates are added to the existing file, however, environment variables in the file retain their values.

Loaded Tools

Displays the tree of tools that have been loaded. The tree becomes available when the loaded by name option has been specified under Saved tools. You can select the tools you want to save from this tree.

Expands all the supported tools in the Loaded Tools tree.

Collapses all the supported tools in the Loaded Tools tree.

Selects all the supported tools.

Deselects all the supported tools.

Preview

Displays a dialog box that allows you to preview the updates made. You can view the updates in a tree view or as plain text.
