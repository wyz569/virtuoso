File Menu in the CIW

Use this menu to perform various operations on CIW.

Menu Description

New

Displays a submenu of the following items:

  • Library displays the New Library form.
    Alternative New Library forms gets displayed dependent upon whether the form was accessed from the CIW or directly from the Cadence Library Manager.
  • Cellview displays the New File Form.

Open

Opens a cellview from the CIW.

Import

Displays a submenu of available translators for importing on your system.

Export

Displays a submenu of available translators for exporting on your system.

Refresh

Refreshes the design data and the CDF data.

Make Read Only

Lets you select one or more cellviews to make read-only.

Bookmarks

Displays a list of recently viewed files (design views).

Recently Viewed Design List

Displays a list of the most recently viewed designs for quick re-opening.

The format is to display in a library/cell/view format with two consecutive spaces between each to aid clarity.
