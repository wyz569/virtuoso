Check Environment Variables Form

Use this form to check the information related to Virtuoso and Unix environment variables.

Field Description

Report Environment Variables Affecting Performance

Reports the environment variables that are affecting the performance of the current session.

Modified Virtuoso environment variables

Specifies whether to report the modified Virtuoso environment variables that are affecting the performance of the current session.

Unix environment variables and settings

Specifies whether to report the modified Unix environment variables and settings that are affecting the performance of the current session.

Tool to Check

Specifies the tools to check. The options are:

  • All covered tools: Checks the tools listed in the Covered Tools section.
  • All loaded tools: Checks the tools listed in the Currently Loaded section.
  • Loaded tools by name (fill in Tool Names): Checks the tools specified in the Tool Names section.

Covered Tools

Lists all the covered tools.

Currently Loaded

Lists all the currently loaded tools.

Tool Names

Specifies the tool name to check the performance. This option is enabled, when Loaded tools by name (fill in Tool Names) is selected.

Windows to Check

Checks the layout window. The options are:

  • All layout windows: Checks all the available layout windows.
  • Current layout windows: Checks the currently opened layout window.

View Modified Virtuoso Environment Variables

Displays all the modified environment variables.

Public

Displays all the modified environment variables in the Cdsenv Editor.

Private

Displays all the private environment variables that were modified at the Virtuoso startup.
