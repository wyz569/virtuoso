Library Form

Use this form to check the design library access time with multiple pings.

Field Description

Library Information

Opens a log file containing all the library-related information, such as cdsinfo, cell count, and permissions.

Library Diagnostic

Scans all libraries and disk mount points for network latency. You can also access the same function from Library Manager – Help – Diagnostics.
