Email Tabulated List of Functions Form

Use this form to locate a valid mail command on your system (mail or mailx in /usr/bin or /bin) after you have provided all the details in the SKILL Tabulator Form.

Field Description

Email tabulated list of functions to Cadence for analysis

Specifies whether to send the output report file. The default is ./skillTab.out to Cadence Support for analysis.

Your email address to receive analysis results

Specifies the e-mail address that Cadence should use to send the analysis results back to you. By default, the e-mail address you type in the Customer Email Address field on the SKILL Tabulator form appears in this field.

Your Name

Specifies the name of the user.

Your Company Name

Specifies the name of the user company.

Your Company Location

Specifies the address of the user company.
