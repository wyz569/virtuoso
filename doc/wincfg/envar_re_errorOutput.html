errorOutput

ui errorOutput boolean { t | nil }

Description

Displays the error as an output in CIW. The default is t.

GUI Equivalent

None

Examples

envGetVal("ui" "errorOutput")
envSetVal("ui" "errorOutput" 'boolean nil)
