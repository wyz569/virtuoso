Set Log File Display Filter Form

Use this form to select the outputs you want to display in the CIW.

Field Description

Show

Indicates the items that gets displayed in the output area of the Command Interpreter Window.

Return values from typed-in commands

Displays return values from typed-in commands in the output area.

Error message output

Displays error message output in the output area.

Warning message output

Displays warning message output in the output area.

Standard message output

Displays standard message output in the output area.

Accelerated input

Displays accelerated input in the output area.

Return values from accelerated input

Displays return values from accelerated input in the output area.

Prompt output

Displays prompt output in the output area.
