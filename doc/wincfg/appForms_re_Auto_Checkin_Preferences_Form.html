Auto Checkin Preferences Form

Use this form to set preferences for the automatic check-in feature.

Field Description

Auto Checkin Preferences

Specifies whether or not you want to be prompted when you check in properties and files or cellviews if you have selected the automatic checkin feature.

When auto checking in properties and files

Specifies whether you want the checkin prompt to appear when you attempt to close and save a file that is not a cellview.

always ask me

Opens a checkin prompt form when you attempt to close and save a file that is not a cellview, such as a property file or data file, in a managed library.

never ask me

Ensures that the checkin prompt form is not displayed when you attempt to close and save a file that is not a cellview, such as a property file or data file, in a managed library.

When auto checking in cellViews

Specifies whether you want the checkin prompt to appear when you attempt to close and save a cellview in a managed library.

never ask me

Ensures that the checkin prompt form is not displayed when you attempt to close and save a cellview in a managed library.

always ask me

Opens a checkin prompt form when you attempt to close and save a cellview in a managed library.

never auto checkin

Ensure that the cellview is not checked in when you attempt to save and close a cellview in a managed library.

always auto checkin

Checks in the cellview when you save and close a cellview in a managed library.
