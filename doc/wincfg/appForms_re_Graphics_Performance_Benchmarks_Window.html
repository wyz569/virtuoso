Graphics Performance Benchmarks Window Form

Use this window to select the benchmark test you want to perform. The Graphics Performance Benchmarks window is displayed when you run the hiGraphicsBenchmark command from a shell terminal.

Field Description

File menu

Performs various actions on the Graphics Performance window.

Run Selected Benchmark

Runs the benchmark tests that are currently selected in the Benchmarks table. This is equivalent to using the Run button.

View Saved Results

Displays the Open form from where you can open and view previously saved results.

Tools menu

Choose the appropriate command to open the Task Viewer window or to run the Calibrator.

Options

Specifies the controls of the benchmark test.

Selections

Filters the benchmark test to be performed. For example, select All tests, or only those tests that are part of the Redraw Set.

Repeats

Specifies how many times the selected tests must be run. For example, if you have selected the REDRAW ON WINDOW group, and have set Repeats to Twice (the default), the test group then runs twice. Using Repeats can increase testing accuracy because the program keeps track of the number of times the tests are run and reports an average result.

Duration

Specifies the time for which each test must run. For example, if you set the Duration to 2 seconds (the default), each selected test runs for two seconds, performing the same check operation until the time set is reached. The higher the value that is set, the higher is the testing accuracy

Testing Window

Specifies the display settings of the window. The default is to display the window With Frame.

  • With a frame, you can perform operations such as moving the window and changing the window stack (through window management). The name of the current test also gets displayed in the frame to monitor test progress.
  • Without a frame, the testing window is always displayed on top, with window management playing no role, and consequently it does not interfere in the test.

Remember Settings

Saves the latest settings to the.benchmark file in your home directory. Saved values are used as the default values the next time the program is run

Benchmarks table

Displays information about all the available graphics performance tests (Name column) that can be run. Select the check box corresponding to a test if you want to run that test.

Tests are grouped into specific testing areas, such as REDRAW ON WINDOW, AREA COPY, and IMAGE OPERATION.

A Weight and Description can also be provided for each benchmark test.

To edit the weight (importance) to be applied to a test, double-click the value in the Weight column and type the new value, or use the up and down arrows.

Run

Runs the selected tests.

The Run button becomes available only if one or more specific benchmark tests are selected in the Benchmarks Table.

After testing starts, the Run button changes to a Stop button to enable you to interrupt the testing, if required.
