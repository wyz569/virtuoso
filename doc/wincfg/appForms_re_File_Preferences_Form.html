File Preferences Form

Use this form to set the file preferences for the Library Browser.

Field Description

File Preference

Specifies the maximum number of entries that can appear on the recently used file list.

Recently used file list

Specifies the maximum number of files you want to see on this list. The maximum is 20.

The program applies your settings immediately, you do not have to restart the software.

Auto Save Database After

Specifies the number of database edits required before an auto save is performed. The default is 500.

The minimum acceptable edit value is 100, and the maximum is 9999.

Open Browser Automatically For

Decides which forms you want the Library Browser to open up with.

Open Form

Selects if you want the Library Browser to open with the Open File form.

  • yes: Opens the Library Browser automatically, along with the Open File form, when you choose File – Open from the CIW.
  • no: No change takes place.

Create Instance Forms

Specifies if you want the Library Browser to open with the Create Instance forms.

  • yes: Opens the Library Browser automatically when you choose Add – Instance or Create – Instance from a schematic or layout window.
  • no: No change takes place.

Other Forms

Selects if you want the Library Browser to open with the other commands set in the .cshrc file.

  • yes: Opens the Library Browser automatically when you select other commands you set in your .cshrc file.
  • no: No change takes place.

CIW Preferences

Sets the display preferences for CIW.

Prompt On Exit

Selects if you want the Exit dialog box to get displayed automatically when exiting the CIW.

  • yes: Displays the Exit dialog box automatically each time you choose File – Exit from the CIW.
  • no: No change takes place.
