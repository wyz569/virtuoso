Edit Library Path Form

Use this form to modify the library path.

Field Description

Library

Specifies a library name and path to edit.

Name

Specifies name to the library.

Directory

Selects the directory to which you assign the library if you are not using design management.

Design Manager

Specifies the design management setup.

If you are using design management, you must put the library in the design-managed workarea from which you started the software.

Cadence DM

Opens your library under design management.

No DM

Provides no design management features, such as check in and check out, for your library.
