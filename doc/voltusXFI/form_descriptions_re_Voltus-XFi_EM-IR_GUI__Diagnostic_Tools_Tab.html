Voltus-XFi EM-IR GUI – Diagnostic Tools Tab

Lets you run DSPF Linter on a DSPF netlist file.

Field Description

Enable

Specifies to run DSPF Linter on the selected DSPF netlist file.

DSPF File

Specifies the DSPF file to be used for running the DSPF Linter utility.

Size

Specifies the size of the DSPF file.

Options

Specifies additional options for running DSPF Linter.

Run/Stop

Specifies to start or stop the DSPF Linter utility.
