includetCmdFile

Specifies the file with user-customized extract commands.

To set this variable in the .cdsinit file or CIW, use the call:

envSetVal("voltusXFi.ext" "includeCmdFile" 'string "file")

To set this variable in the .cdsenv file, use the call:

voltusXFi.ext includeCmdFile string "file"

Variable Type

string

Default Value

nil

Acceptable Values

Any string value

GUI Equivalent

Include Command File
