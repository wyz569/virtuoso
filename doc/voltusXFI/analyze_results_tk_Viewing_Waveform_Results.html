Viewing Waveform Results

To view the current and voltage waveforms in VIVA Waveform Viewer:

  1. On the Voltus-XFi window, choose Results from the task pane.
  2. Select an analysis, test, and corner for which you want to view the reports.
  3. Select the Plots tab.
  4. In the Layout View section, select the layout, cell, and view from the respective drop-down lists.
  5. Click the Open button. The Virtuoso Visualization & Analysis window opens in a new tab.
