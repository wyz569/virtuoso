vxfiSetLVSRunName

vxfiSetLVSRunName(
t_sessionName
t_runName
t_lvsrunName
)

Description

Specifies the LVS run name to be used for extraction.

Arguments

t_sessionName

Specifies the name of the Voltus-XFi session name.

t_runName

Specifies the run name used while running extraction.

t_lvsrunName

Specifies the name of the LVS run.

Value Returned

None

Examples

The following example specifies that the name of the LVS run name is DiffOpAmp:

vxfiSetLVSRunName(vxfiSession, extName, "DiffOpAmp")
