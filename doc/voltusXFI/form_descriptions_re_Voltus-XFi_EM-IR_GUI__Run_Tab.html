Voltus-XFi EM-IR GUI – Run Tab

Lets you start an extraction or analysis run.

Field Description

Job Status

Run Name

Specifies the name of the extraction or analysis run.

Analysis

Specifies the type of run: extraction or analysis.

Stop/Rerun

Provides the options to stop a run and restart a run. After the extraction run is complete, you must stop this run before starting the analysis run.

Progress

Displays the progress of the current run.

Status

Displays the status of the current run.
