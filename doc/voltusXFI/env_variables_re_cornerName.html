cornerName

Specifies the name of the corner at which extraction will be performed.

To set this variable in the .cdsinit file or CIW, use the call:

envSetVal("voltusXFi.ext" "cornerName" 'string "rcx_typical")

To set this variable in the .cdsenv file, use the call:

voltusXFi.ext cornerName string "rcx_typical"

Variable Type

string

Default Value

nil

Acceptable Values

Any string value

GUI Equivalent

Corner
