Generating a DSPF

  1. Click the Run tab from the task pane.
  2. In the Job Status section, click the Start Run ( ) button next to the extraction run (ext_1).
    The extraction log appears at the lower section of the Voltus-XFi window. After the extraction run is complete, the log window displays the message “Finished Extraction”. The status changes from Running to Complete.
    You can click the Stop Run ( ) button to stop extraction.
  3. You can check the log file for the extraction run from within the Voltus-XFi interface by right-clicking the extraction run name and selecting the Open Log File option.
  4. You can open the Quantus output directory from within the Voltus-XFi interface by right-clicking the extraction run name and selecting the Open Terminal Here option.
