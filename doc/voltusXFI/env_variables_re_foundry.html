foundry

Specifies the foundry.

To set this variable in the .cdsinit file or CIW, use the call:

envSetVal("voltusXFi.project" "foundry" 'string "foundry_name")

To set this variable in the .cdsenv file, use the call:

voltusXFi.project foundry string "foundry_name"

Variable Type

string

Default Value

Generic

Acceptable Values

Generic or foundry name

GUI Equivalent

Foundry
