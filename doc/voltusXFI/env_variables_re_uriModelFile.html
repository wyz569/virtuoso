uriModelFile

Specifies a file containing all ECO changes for a specific what-if run. This variable is supported only for solver method 2-Stage.

To set this variable in the .cdsinit file or CIW, use the call:

envSetVal("voltusXFi.ext" "uriModelFile" 'string "file")

To set this variable in the .cdsenv file, use the call:

voltusXFi.ext uriModelFile string "file"

Variable Type

string

Default Value

nil

Acceptable Values

Any string value

Miscellaneous - ECO File
