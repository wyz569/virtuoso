Rerunning EM Analysis with Different EM Settings

To rerun the EM analysis with modified EM settings, do the following:

  1. On the Voltus-XFi window, choose Results from the task pane.
  2. Right-click a history item, test, or corner for which you want to rerun EM analysis.
  3. Select the Rerun EM analysis option.
    Depending on your selection, the shortcut option allows you to run EM analysis for all tests and corners of an analysis, all corners of a test, or a specific corner.
    An EMIR results database file (.wdb) is generated with the new EM settings that can be loaded in the Voltus-XFi Result Browser. The .wdb file is saved in the rerun (for example, 00_rereun_em, 01_rereun_em, and so on) folder in the psf directory.
