Data Requirements for Voltus-XFi

The following diagram illustrates the inputs required for EM-IR analysis in Voltus-XFi:

The following inputs are required for EM-IR analysis in Voltus-XFi:

  • xDSPF – parasitic netlist of the design
  • EM Rules
  • Power and Signal Nets
  • Type of analysis to be performed (peak, avg, rms, absAvg)
