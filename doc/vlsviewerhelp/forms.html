Layout Viewer Forms

The following table describes the forms that are available in Layout Viewer.

Form

Description

Background Cellview Form

Use the form to display two superimposed cellviews in the same window.

Create Measurement Form

Use the form to create rulers.

Custom Show Layer/Purpose Form

Use the form to define the criteria to filter layers based on the values of specific columns in the Layers panel.

Delete Configuration Form

Use the form to delete a selected configuration.

Descend Options Form

Use the form to enable the functionality that allows you to choose the view to be opened when descending into instances.

Display Options Form

Use the form to control the appearance of objects and the behavior of commands in a cellview.

Dynamic Display Form

Use the form to control the information displayed about objects in information balloons and during dynamic measurement.

Dynamic Selection Assistant Options Form

Use the form to specify how selected objects are displayed in the Dynamic Selection assistant and the mode used when opening a cellview from the context menu of the assistant.

Find Form

Use the form to search for objects with specific attributes or property values.

In Layout Viewer, the Replace functionality is not available.

Highlight Options Form

Use the form to control the objects highlighted when a net is selected or probed. You can choose to selectively display or hide probes for specified nets.

Magnifier Options Form

Use the form to magnify or zoom in a specific part of the design display area instead of zooming in the entire design display area.

Palette Options Form

Use the form to set local and global options for the Palette assistant.

Print Hierarchy Tree Form

Use the form to specify details of the cellview for which you want to print the hierarchy of cellview instances.

Restore View Form

Use the form to choose the view that you want to restore. You can choose from the last three saved views in a session.

Save/Restore Selection Set Form

Use the form to save the objects selected in the canvas as a selection set so that you can restore them later.

Save Configuration Form

Use the form to save the configuration for displaying information about shapes and objects.

Select Area Shape Type Form

Use the form to specify the type of shape within which you want to select objects. You can choose to draw a rectangle or a polygon to define the area for selecting objects.

Selection Options Form

Use the form to set controls for object selection.

Selection Protection Highlight Options Form

Use the form to specify the protection highlight options during the selection of objects.

Set Default Application Form

Use the form to set the default application for opening the current cellview and all cellviews with the same view type as the current cellview.

Show Selected Set Window

Use the form to view information about each type of object that is selected in the current cellview.

Summary Window

Use the window to view the statistics related to the current cellview, such as the instance, layer object, wire, and blockage statistics.

Transparency Options Form

Use the form to control the transparency of tracks, FinFET/WSP shapes, regions, and wire editor tracks.

Tree Form

Use the form to set the levels of instances hierarchy to be listed for the current cellview.
