Grids Panel

Use the Grids panel to manage the visibility and selectability of tracks and grid elements.

The Grids panel user interface can be categorized into the following major components: columns, column header context menu, and toolbars.

Grids Panel Columns

The following table describes the columns displayed in the Grids panel.

Column Description

Grids

Displays elements in a hierarchical tree structure.

V (Visibility)

Lets you control the visibility of elements. To make an element visible, select the corresponding check box in the V (Visibility) column.

Grids Panel Column Header Context Menu

The following table describes the options available on the column header context menu of the Grids panel. You can access the context menu by right-clicking a column header in the panel.

Command Description

Columns

Lets you selectively hide or display columns in the Grids panel. The Hide All and Show All commands hide or display the V (Visibility) column. The Grids column is not hidden from view when you choose the Hide All command.

Sort By

Lets you sort the elements displayed in the Grids panel by element name and visibility.

  • Grids: Sorts elements in the alphabetical order by name.
  • Visibility: Sorts elements by visibility. The elements for which the visibility check box is deselected are listed first, followed by the elements for which the check box is selected.
  • Default: Sorts elements in the default order, which is Tracks and then Grids.
  • Reverse: Reverses the order in which the elements are currently displayed.

Hide This Column

Hides a column in the Grids panel. To hide a column, right-click the column header and choose Hide This Column.

Grids Panel Toolbars

The following table describes the toolbars available in the Grids panel. For more information about the functionality available in these toolbars, see Layers Panel Toolbars.

Toolbar Description

Search

Lets you search for specific grids and tracks in the Grids panel.

Control Buttons

Lets you manage the visibility and selectability of grids and tracks in the Grids panel.

By default, the Grids panel does not display toolbars. You can display a toolbar by selecting it from the Show Tools submenu in the Palette context menu.
