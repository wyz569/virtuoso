Selecting Vias in a Via Stack

To select all the vias in a via stack, use any of the following methods:

  • Select a via in the via stack on the canvas and press Ctrl+O to select the remaining vias.
  • Click and drag to create a selection box around the via stack on the canvas.
  • Switch on via stack selection mode and then click a via on the canvas. You can turn on via stack selection mode by selecting the Via Stack check box in the Selection Options form or by using the Via Stack icon on the Options toolbar. You can also choose the Via with matching sizes between common layers check box to select only those vias that have the same size on common layers.

Full and partial selection modes do not have an impact on via stack selection. You can select a complete via stack even when partial selection mode is on.
