Exporting a Layer Set

To export a layer set:

  1. From the Palette context menu, choose Layer Set – Export.
    The Export Layer Set form appears. The Layer Set repository field in the Origin section displays the repository in which the layer set is stored. This is a non-editable field.
  2. Select the layer set that you want to export, by choosing from the Layer Set name list.
  3. Specify the layer set name in the File name field of the Destination section. The layer set is saved with this name.
    Specify .layerSet as the file extension along with the layer set name.
  4. Click Export.
    A new layer set is created in your present working directory.
