Layout Viewer Assistants

Layout Viewer provides various assistants to enable you to work on your designs effectively and efficiently. The following table provides an overview of the assistants.

Assistant Description

Palette Assistant

Lets you manage the display context and control the visibility and selectability of layer-purpose pairs, objects, grids in your designs.

Dynamic Selection Assistant

Identifies and selects specific objects of interest from a densely-populated design window that has many overlapping or hierarchical objects.

World View Assistant

Displays a complete picture of your entire design and marks the part of the design that is currently displayed in the drawing area. It acts as a navigation tool, especially useful in large designs.

Navigator Assistant

Provides features to view objects across the design hierarchy using a tree representation. It provides a convenient method of selecting sets, expanding, viewing, and interacting with the objects in it, such as instances, nets, and pins across a schematic hierarchy.

Property Editor Assistant

Provides a flexible framework for defining a collection of objects and their associated properties. It lets you view object property values based on the components in your design.

Search Assistant

Provides a wide range of design search features, including dynamic context search categories, fast, keyword-directed searching with immediate feedback, and an intuitive user interface that makes searching for design data simple and productive.
