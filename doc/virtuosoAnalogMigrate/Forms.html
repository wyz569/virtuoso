Select Libraries that have Cells to Map Form

Use this form to select the preferred library and extract the CDF parameters you need in order to create a CSV file for all the source devices that need to be mapped. It is accessed using the Migrate – Schematic – Utilities – Create Source-side of Migration Map command.

Field Description

List of Libraries

Lists the libraries that are available in the current design. You can select one or more libraries that contain the CDF parameters to be captured into the reuse file.

Use Shift + click to select a range of (consecutive) libraries.

Use Ctrl + click to select multiple libraries individually.

Select a Cell to Inspect Form

Use this form to review and delete instance properties carried over by copies of cellviews. It is accessed using the Migrate – Schematic – Utilities – Inspect Schematic Instance Properties command.

Field Description

List of instance properties

Lists all instance properties in the current design. You can select an instance property and click OK to open the corresponding cellview.

Schematic Mapping Settings Form

Use this form to define the basic settings for schematic migration. This form can be accessed using the Migrate – Schematic – Mapping – Settings command.

Field Description

Configuration File

Shows the path to the configuration file with the settings used for the migration.

Running Directory

Specifies the running directory (pre-selected).

Mapping File

Selects the mapping file to be used for the migration from a browser window.

Technology Libraries

This section lets you specify the libraries for the migration process.

Source

Defines the library from which the source schematic data must be captured.

Target

Defines the library to which the captured source schematic data is to be applied.

Virtuoso Schematic Mapping Form (Part 1)

Use this form to save data from the source schematic. It is accessed using the Migrate – Schematic – Mapping – Save Source Schematic Data (Part 1) command.

Field Description

Source

This section lets you specify the source library, cell, and view for the migration.

Library

Lets you select the source library.

Cell Name

Lets you select the source cell.

View Name

Defines the source view name.

  • Process Hierarchy: Migrates the selected cellview and the underlying hierarchy.
  • Process Library: Migrates all cells in the source library.
  • Process all views of each cell: Migrates all views of each cell.
  • Process all reference libraries: Migrates all reference libraries.

Target

This section lets you specify the target library and cell for the migration.

Library

Lets you select the target library or create a new library. To create a new library, select Create New from the drop-down list. The New Library form opens.

Cell Name

Lets you select the target cell.

View Name

Lets you select the target view name (pre-selected from source).

Name Options

A string field that can be used when Use name options below is selected for cell name.

  • Prefix: The name in the Name Options field will be a prefix of the cell name.
  • Replace: The name in the Name Options field will replace the cell name.
  • Suffix: The name in the Name Options will be a suffix of the cell name.

More Options

This section lets you specify additional options for the migration.

Mapping File

Specifies the mapping file.

Design Library Mapping

Specifies the path to the file that contains mappings of layers, devices, and parameters between the source and the target.

Exclude Library List

Specifies a list of libraries to be ignored during migration.

Exclude View List

Specifies a list of views to be ignored during migration.
