Export Color Constraint File

Generates the color constraint file to perform Layout Versus Schematic (LVS) coloring checks.

Field

Description

Color A Layer Name(s)

Specifies the name of the color A layer.

Color A Layer Name(s)

Specifies the name of the color B layer.

Export CDF color

Specifies that the CDF color must be exported.

Log file name

Specifies the name of the log file.
