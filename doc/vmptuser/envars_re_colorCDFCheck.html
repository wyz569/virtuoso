colorCDFCheck

mpt colorCDFCheck boolean { t | nil }

Description

Specifies whether CDF color is exported. The default is t.

GUI Equivalent

Command

File – Export Color Constraint File

Field

Export CDF color

Examples

envGetVal("mpt" "colorCDFCheck")
envSetVal("mpt" "colorCDFCheck" 'boolean nil)
