Multiple Patterning Layer Default Color

Changes the default color for each colorable layer.

Field

Description

Layer Name

Specifies the name of the colorable layer.

Layer Default Color

Sets the color for the colorable layer. You can choose from the following layer default color options for each colorable layer: grayColor, mask1Color, mask2Color, mask3Color, mask4Color, alternating.
