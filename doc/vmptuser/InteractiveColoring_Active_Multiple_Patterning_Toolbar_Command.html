Active Multiple Patterning Toolbar Command

The status banner Cmd field in the bottom-right corner of the layout window shows the active command for post-select actions initiated from the Multiple Patterning toolbar. For information on these post-select actions, see Post-Select Lock and Unlock and Removing Color from Shapes.

You use the Multiple Patterning toolbar for the following functions:

Icon Command Description

Congestion Analyze

Runs different methods of congestion analysis on the design.

Color Engine Switch

Turns the multiple patterning color engine on and off.

Show/Hide Color

Shows and hides color

Layer Default Color

Updates layer default color

Shift Color

Shifts colors

ReColor

Recolors selected colors

Lock Color

Locks and unlocks colors

Delete Colors

Removes color from shapes

Stitch/Unstitch

Stitches and unstitches

Markers to Mask

Converts markers to mask colors

Checks

Checks colors

Last Operation Status and Color Query

View the last operation status and probe the color source

Multiple Patterning Toolbar Task Assistant

Displays multiple patterning toolbar task sssistant

Multiple Patterning Options

Opens multiple patterning options
