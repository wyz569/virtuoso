How do I filter by snap defs?

To show only WSSPDefs and SPDefs matching an expression, type the expression in the Snap Def field.

As you type, the WSSPDefs and SPDefs in the Layer/Def column of the table is dynamically updated to include only those matching the expression. Expressions are case-sensitive and can include the special characters shown in the table below:

Special CharactersMeaning
? Matches any single character.
* Matches any sequence of zero or more characters.
[chars] Matches any single character in chars. If chars contains a sequence of the form a-x, then any character between a and x (inclusively) will match.
{a, b,…} Matches any of the strings a, b, … listed within the braces.
