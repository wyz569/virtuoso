Save State Form

Use the Save State form to save the settings from the current setup as a state. The form contains the following fields and lists:  

Field Description

Library

Specifies the name of the library that contains the design.

Cell

Specifies the name of the cell in the selected library for which the state needs to be saved.

State

Specifies the name of the state that must be saved.

Cells

Lists the names of all cells in the selected library.
