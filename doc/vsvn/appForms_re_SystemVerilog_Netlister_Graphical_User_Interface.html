SystemVerilog Netlister Graphical User Interface

The graphical user interface of SystemVerilog Netlister can be categorized into the following main sections:

Section

Description

Menus

Displays the Commands and Help menus.

Design

Specifies the library, cell, and view of the top-level design.

You can select only configuration views.

Setting

Specifies the options and design variables required for netlist generation.

Actions

Specifies the netlisting actions. You can generate or regenerate a netlist to view the netlisting results.

Status bar

Displays the current status of the selected item.
