Virtuoso Variation Option Forms

This section describes the forms used in Virtuoso Variation Option.

  • The Monte Carlo form:
    The Monte Carlo form lets you set up options to run various Monte Carlo methods.
  • The Improve Yield form:
    The Improve Yield form lets you specify the options for running Improve Yield mode.
  • The AOP Options form:
    The AOP Options form lets you specify the options to run advanced optimization.
