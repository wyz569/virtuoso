Full Evaluation and Conditional Evaluation

The optimization options in the Improve Yield mode provides the following evaluation types:

  • Full Evaluation
    Full evaluation runs all tests for each new design point to compare against the current lowest-cost point (best point).
  • Conditional Evaluation
    Conditional evaluation first simulates only the tests or corners that failed at the previous point. If the results of those tests and corners are worse than they were at the previous point, the remaining tests and corners are skipped for that design point. This evaluation saves time for optimization over many tests and corners.
