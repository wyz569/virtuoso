Toolbar of the Mismatch Contribution Tab

The toolbar of the Mismatch Contribution tab of the Sensitivity Analysis-MonteCarlo.N form contains the following commands.

Icon Command Name Description

N/A

Tests

Selects the test for which you want to view the mismatch contribution results.

The mismatch contribution table can display data for only one test at a time. By default, it shows the data for the first test in the setup.

N/A

View

Selects one of the following views:

  • Hierarchical: Shows mismatch contribution results per instance.
  • Flat: Shows mismatch contribution results per parameter. This is the default view.
  • Flat - Show Parameters: Shows each statistical parameter in the instance_name:parameter_name format. As there can be multiple parameters for a device, each parameter is shown in a separate row.
  • Variance: Shows the variance values for each instance or parameter.
  • Percentage: Shows the contribution of each instance or parameter to variance in percentage.

Ascend parameter hierarchy

Ascends the design hierarchy one level.

Descend parameter hierarchy

Descends the design hierarchy one level.

Display numerical values

Displays the mismatch contribution results in numerical values.

N/A

Displays vertical bar graphs to compare data in the rows.

N/A

Displays the horizontal bar graphs to compare data in the columns.

N/A

Corners

Selects the corner for which you want to view the mismatch contribution results.

N/A

Plot Mode

Selects one of the plotting modes to plot the mismatch contribution results:

  • Replace: Replaces the graph in the active window with a new graph.
  • Append: Appends the result in the active graph window or subwindow.
  • New SubWin: Plots the result in a new subwnidow within the existing graph window.
  • New Win: Plots the result in a new graph window.

N/A

Max Contribution

Hides the rows for the instance or parameters whose contribution to the variance is greater than or equal to the value specified by the Max Contribution filter.

Use this filter to focus on the devices that have high contribution to the variance of the circuit.

N/A

Yield

Hides the columns for the specifications whose yield exceeds the value specified by the Yield filter.

Use this filter when you need to focus on the outputs that are not achieving the required specification yield.

Reset

Resets the Max Contribution and Yield filters to initial values.

Open the Mismatch Tuner

Opens the Mismatch Tuner, which you can use to see how a change in the size of any device instance impacts the variation of circuit output.

Close the Mismatch Tuner

Closes the Mismatch Tuner.
