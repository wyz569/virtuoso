blockAnnotationShrinkFactor

layoutXL.floorplan blockAnnotationShrinkFactor float shrink_factor

Description

Specifies the factor by which each edge of the block annotation must be shrunk.

The default value is 1.

GUI Equivalent

Command:

Floorplan – Block Annotations

Plan – View – Block Annotations (Virtuoso Layout Suite EXL)

Field:

Shrink Factor

Examples

envGetVal("layoutXL.floorplan" "blockAnnotationShrinkFactor")
envSetVal("layoutXL.floorplan" "blockAnnotationShrinkFactor" 'float 3)
