Pin Creation Commands

The Pin Tool lets you create new pins and customize their attributes. The Create menu lists the commands to create pins from various sources.

Command

Description

Create Pin

Creates pins in the layout canvas.

See Creating Pins.

Create Pin From Label

Creates pins from existing labels in the cellview.

Create Pin From Net Shapes

Creates pins from existing net shapes in the cellview.

(Virtuoso Layout Suite EXL) Create Pin Template

Creates missing pins and renames, repositions, and constrains existing pins in the layout view as per the source view.

Promote Pins

Promotes pins from a lower level in a design hierarchy to a higher level.

Auto-Create Pins

Searches for net shapes in a design and create boundary and buried pins automatically on these net shapes.
