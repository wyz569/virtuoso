Label Update Form

Use the Label Update form to update pin labels and text displays for clearer visualization.

Field Description

Mode

Specifies the pins on which the command has to run.

  • All: Runs the command on all soft block pin labels and text displays in the current design. Soft blocks that are marked as read-only are made editable and a warning message is displayed in the CIW.
  • Selected: Runs the command only on the selected pin labels and text displays. You can select:
    • Top-level and level-1 pins
    • Individual soft blocks
    • Individual top-level pin labels and text displays

Adjust Label Orientation

This section makes labels and text displays perpendicular to the nearest PR boundary edge.

Orientation remains unchanged for labels and text displays that are already perpendicular to the nearest PR boundary edge.

Make Label Perpendicular To Nearest Edge

Makes labels and text displays perpendicular to the nearest PR boundary edge. The nearest PR boundary edge is determined based on the minimum distance of the label’s parent pin center from the PR boundary.

Adjust Label Position

This section includes the options for adjusting label positions.

Pull Label Inside P&R Boundary

Places labels and text displays inside the PR boundary after performing different permutation and combinations for justification and orientation. The label or text display origins remain unaltered.

If all labels and text displays could not be successfully pulled inside the PR boundary, an appropriate report is displayed in the CIW.

Move Label Origin

Specifies whether the origins of labels and text displays are to be changed when running the Label Update command.

  • Pin Edge Nearest to Boundary: Moves all labels and text displays to the center of the nearest pin edge irrespective of their initial origins.

  • Pin Center: Moves all labels and text displays to the center of the pin.

Environment variables: moveLabelTo, moveLabelOrigin

Adjust Label Size

This section includes options to resize labels.

Update Label Size

Specifies the label size in the adjoining box.

Update Label Size does not run in the following situations:

  • On floating labels and text displays that are not associated with any parent pin.
  • When the label or text display parent is not pin.
  • When labels or text displays for the instance pins are present on the top-level instead of within the same cellview as the pin.

In the above situations, an appropriate message is printed in the report.

Auto Reduce Label Size

Computes any existing label overlaps and reduces the size of the labels or text displays so as to remove any remaining overlaps.

Choose this option when label or text display overlaps exist even after making the above settings. The size of labels or text displays is reduced up to a minimum threshold value as specified in the adjoining text box.

Minimum Label Size

Specifies the minimum threshold value for reducing the size of labels.
