Add Pin On Different Purposes Form

Use the Add Pin On Different Purposes form to create pins with two or more purposes in a single click.

Field Description

Search

Lets you search for specific purpose in the design.

Purpose

Specifies the layer purpose on which you want to create a new pin. You can select multiple purposes at the same time.
