pinResizeCurrentEstimator

layoutXL.floorplan pinResizeCurrentEstimator string "current_estimator"

Description

Specifies the area estimator function that is to be displayed as the selected area estimator function in the Resize function drop-down list.

The default value is "".

GUI Equivalent

Command

Floorplan – Pin Tool – Place – Resize Pins

Plan – Pin Planning – Pin Tool – Place – Resize Pins (Virtuoso Layout Suite EXL)

Field

Resize function

Examples

envGetVal("layoutXL.floorplan" "pinResizeCurrentEstimator")
envSetVal("layoutXL.floorplan" "pinResizeCurrentEstimator" 'string "functionA")
