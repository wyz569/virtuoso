Create Stacked Pin Form

Use the Create stacked pin form to create a stack of pins and vias on the selected metal layers.

Field Description

Connectivity

Specifies whether multiple pins are created as strongly or weakly connected.

  • Strong: Lets you create multiple pins as strongly connected. You can add multiple pin figures to the selected pin, to define all of them as strongly connected. This option is selected by default.
  • Weak: Lets you create multiple pins as weakly connected. A separate pin is created for every pin figure in the stack.

Layer and Vias

From and To: Specifies the range of metal layers that contain the pins to be combined into a pin stack.

Stacked pins support only metal and poly layers for the From and To layer fields.

Create vias

Creates a via between the From and To metal layers based on the geometry of the pin.

Vias can be viewed in the layout by selecting the Via Stacks field in the Edit Via Stack Properties form.
