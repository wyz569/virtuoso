Pin Spacing Advanced Options Form

Use the Pin Spacing Advanced Options form to define pin grids. A pin grid defines the points at which pins need to be placed. Each layer has a separate pin grid.

Field Description

Choose width for Grid

Specifies whether the width of the grid should correspond to the Width of the Smallest Pin in the design, or to the value that is picked from the technology file (default). Use this option only when the pins on a layer have different pin widths. Choose width for Grid is available for both Auto and Custom pin spacing modes.

For layers that do not have any pins, the layer minWidth value is considered the minimum pin width for creating the pin grid.

Environment Variable: widthForPinGrid

Pin Grid Adjustment Factor

Specifies the value by which the pin grid needs to be scaled for each layer. For example, if you specify the Pin Grid Adjustment Factor as 0.5 for a layer, then the width of the grid is reduced to half its current value. Pin Grid Adjustment Factor is available for both Auto and Custom pin spacing modes.

Define Pin Grid Adjustment Factor

Specifies that the pin grid adjustment factor can be specified for individual layers.

Environment variables: perLayerAdjustmentFactorList, pinGridAdjustmentFactor

Custom Pin Grid

Specifies the absolute values for the width of the pin grid for each layer. Pin spacing values, as specified in the Pin Spacing Options form, are applied. Custom Pin Grid is available only for the Custom pin spacing mode.

Define Custom Pin Grid

Specifies that the custom pin grid can be specified for each layer individually.

Environment variables: customPinGrid, perLayerCustomPinGridList

Defaults

Changes the form values to the default values, which correspond to the original factory settings.
