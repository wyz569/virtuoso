Import Label Info from File Form

(Virtuoso Layout Suite EXL) Use the Import Label Info from File form to generate the labels that are defined in the label file into the current cellview.

Field Description

Import File

Specifies the name of the file from which label information is to be imported.

Environment variable: ptPinPurposeNames

Mode

Specifies the mode to generate labels.

  • Replace: Deletes all existing labels before generating new labels.
  • Update: Retains existing labels in the design and generates new labels.

Environment variable: ptImportFileMode
