Product Version IC23.1, November 2023

Prerequisites for Virtuoso Floorplanner

This section is aimed at developers and designers of integrated circuits and assumes that you are familiar with:

  • The Virtuoso Studio design environment and application infrastructure mechanisms designed to support consistent operations between all Cadence® tools.
  • The applications used to design and develop integrated circuits in the Virtuoso Studio design environment, notably, the Virtuoso Layout Suite, and Virtuoso Schematic Editor.
  • The Virtuoso Studio design environment technology file.

To use Virtuoso Floorplanner, ensure that the following requirements are met:

  • Technology Requirements
    Virtuoso® Floorplanner requires technology-specific information about your design to be stored in a technology file for the design library. You can create and edit technology files. You can use also use Cadence SKILL functions to query or update the technology file.
  • Constraint Group Lookup Precedence
    Virtuoso® Floorplanner requires technology file rules to work correctly.
  • Licensing Requirements
    In IC 23.1, the floorplanning commands require one of the following licenses:
    • A Virtuoso_Layout_Suite_XL license combined with 4 GXL flexible license tokens
    • A Virtuoso_Layout_Suite_EXL license

    In ICADVM20.1, floorplanning commands require either of the following:
    • A Virtuoso_Layout_Suite_XL license combined with 4 GXL flexible license tokens
    • A Virtuoso_Layout_Suite_EXL license

    In IC6.1.8, floorplanning commands require one of the following licenses combined with 4 GXL tokens:
    • Virtuoso_Layout_Suite_XL
    • Virtuoso_Layout_Suite_GXL
    • Virtuoso_Layout_Suite_EAD
