astPinMoveMatchLabelLayerLPP

layoutXL.floorplan astPinMoveMatchLabelLayerLPP string "LPP"

Description

(Virtuoso Layout Suite EXL) Specifies the layer and purpose to which labels are to be moved when running the assisted move command. This setting is valid only when astPinMoveMatchLabelLayerChoice is set to Custom LPP.

GUI Equivalent

Command:

Plan – Pin Planning – Pin Tool – Tools – Options

Field:

Match Label – Custom LPP

Examples

envGetVal("layoutXL.floorplan" "astPinMoveMatchLabelLayerLPP")
envSetVal("layoutXL.floorplan" "astPinMoveMatchLabelLayerLPP" 'string "M2 drawing")
