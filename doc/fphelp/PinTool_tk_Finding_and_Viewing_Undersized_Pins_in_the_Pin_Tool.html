Finding and Viewing Undersized Pins in the Pin Tool

To find and view violating pins:

  1. Choose Plan – Pin Planning – Pin Tool. The Pin Browser is displayed.
  2. Choose Tools – Electrically Aware Pin Sizing.
    The Electrically Aware Pin Sizing form appears.
  3. Select one of the following Run On Pins options to specify the pins on which the command is to be run. Choose All (default) to run the command on all top-level and level-1 pins in the design or choose Selected to run the command on the selected pins.
  4. Select Find and View UnderSized Pins for violation detection.
  5. Choose Select Pin to select all the detected undersized pins.
  6. Choose Highlight Pin to highlight the undersized pins in brown for better visibility. Any prior highlights are removed.
  7. Choose Highlight Violating Edge to highlight only the violating edge of undersized pins.
  8. Select Show Edge Dimension (Actual vs Desired) to display the current edge dimension and the desired electrically-aware dimension of the violating edge.
  9. Click OK.

All violating pins are listed in the Annotation Browser.

After viewing the violating pins, you can use the options in the Resize UnderSized Pins section of the Electrically Aware Pin Sizing form to re-size these pins and fix the violations.
