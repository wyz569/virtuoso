pinSpacingUpdateConstrs

layoutXL.floorplan pinSpacingUpdateConstrs boolean { t | nil }

Description

Specifies whether pin constraints are to be updated when applying a spacing value using the Pin Spacing form.

The default value is nil.

GUI Equivalent

Command:

Floorplan – Pin Tool

Plan – Pin Planning – Pin Tool – Place – Pin Spacing (Virtuoso Layout Suite EXL)

Field:

Update Constraints

Examples

envGetVal("layoutXL.floorplan" "pinSpacingUpdateConstrs")
envSetVal("layoutXL.floorplan" "pinSpacingUpdateConstrs" 'boolean t)
