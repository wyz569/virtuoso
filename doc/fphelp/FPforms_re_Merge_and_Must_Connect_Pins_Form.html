Merge and Must Connect Pins Form

Use the Merge and Must Connect Pins form to merge two or more overlapping rectangular, rectilinear, or path pin shapes into a single rectilinear shape.

Field Description

Mode

Specifies the pins to be selected for merge.

  • All: Applies to all pin shapes.
  • Selected: Applies to the selected pin shapes.

Environment variable: mergePinsSelMode

Pin Connectivity

Specifies the connectivity settings for the merged pin.

  • Must Connect Pins Without Creating New Nets: Specifies that the connectivity of the pin shapes is to be retained. The must connect all pins setting is applied to the existing terminals. This option is selected by default.
  • Must Connect Pins Using New Nets: Specifies that new nets, terminals, and pins are to be created for each pin shape island. The new terminals are connected and are placed together in a must connect group.
  • Strongly Connect Pins: Specifies that a single connection point on a highly conductive metal is to be used to provide the required connectivity.

Environment variable: mergePinsConnMode

Suffix String

Specifies the suffix to be added to the new nets, terminals, and pins that have been created on the merged pin islands. The default value is _MJ, which is applied when Must connect pins using new nets is selected.

Environment variable: mergePinsSuffix
