blockAnnotationAreaUnits

layoutXL.floorplan blockAnnotationAreaUnits int area_units

Description

Specifies a value for the total block area for the block annotation.

The default value is 1.

GUI Equivalent

Command:

Floorplan – Block Annotations

Plan – View – Block Annotations (Virtuoso Layout Suite EXL)

Field:

Area

Examples

envGetVal("layoutXL.floorplan" "blockAnnotationAreaUnits")
envSetVal("layoutXL.floorplan" "blockAnnotationAreaUnits" 'int 2)
