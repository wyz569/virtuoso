Sorting Pins in the Pin Tool

By default, pins in the Pin Browser of the Pin Tool are sorted based on their assigned edges (Edge column).

To sort by any other column:

  1. Choose Plan – Pin Planning – Pin Tool. The Pin Browser is displayed.
  2. Click the relevant column header.
  3. Click again to reverse the sort order.

To sort a selected set of pins:

  1. Right-click and choose Sort on selection from the shortcut menu.
  2. Choose Reverse order of pins to reverse the sort order. Sorting in reverse order works only for pins with ordered edge constraints.
