Compare Pin Group Guides Form

Use the Compare Pin Group Guides form to compare the pin group guides in two different cellviews.

Field Description

Library Name

Specifies the source library that contains the required pin group guide for comparison.

Cell Name

Specifies the source cell that contains the required pin group guide for comparison.

View Name

Specifies the source view that contains the required pin group guide for comparison.
