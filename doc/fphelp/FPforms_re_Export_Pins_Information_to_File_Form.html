Export Pins Information to File Form

(Virtuoso Layout Suite EXL) Use the Export Pins Information to File form to export pin information from the current cellview into a pin file.

Field Description

File name

Specifies the name of a pin file.

Files of type

Specifies that only files in text format type can be chosen for exporting.

Choose

Exports pin information from the current cellview and stores it in the specified pin file.
