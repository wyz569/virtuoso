Adding Pins On Different Purposes in the Pin Tool

To add pins on specific purposes for any existing pin:

  1. Choose Plan – Pin Planning – Pin Tool. The Pin Browser window is displayed.
  2. Select the required pin in the Pin Browser window.
  3. Right-click and choose Add Pin On Different Purposes.
    The Add Pin On Different Purposes form is displayed.
  4. Select a purpose in the Add Pin On Different Purposes form. For example, select pin1 in Pin Browser and choose pin and slot purposes in the Add Pin On Different Purposes form.
  5. Click OK to reflect the changes in the layout canvas. When you click OK, two additional pins are created below pin1 on the same metal layer with Metal1slot and Metal1pin LPPs.

This functionality can be run on all top-level and level-1 layout pins.
