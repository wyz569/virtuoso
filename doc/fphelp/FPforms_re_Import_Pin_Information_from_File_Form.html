Import Pin Information from File Form

(Virtuoso Layout Suite EXL) Use the Import Pin Information from File form to generate the pins that are defined in the pin file into the current cellview.

Field Description

Import File

Specifies the name of the file from which pin information is to be imported.

Mode

Specifies the mode to generate pins.

  • Replace: Deletes all existing pins before generating new pins.
  • Update: Retains existing pins in the design and generates new pins.

Environment variable: ptImportFileMode
