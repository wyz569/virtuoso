Resizing Undersized Pins in the Pin Tool

Use the options in the Electrically Aware Pin Sizing form to resize pins that do not meet the electrical requirements of the design. You can either first find and view the violating pins or directly resize them.

To resize violating pins:

  1. Choose Plan – Pin Planning – Pin Tool. The Pin Browser is displayed.
  2. Choose Tools – Electrically Aware Pin Sizing.
    The Electrically Aware Pin Sizing form appears.
  3. Select one of the following Run On Pins options to specify the pins on which the command has to run. Choose All (default) to run the command on all top-level and level-1 pins in the design. or choose Selected to run the command on the selected pins.
  4. Select Resize UnderSized Pins to fix the pin violations.
  5. Select Violating Edge Only to resize only the violating pin edges to the desired electrically-aware width. All other edges remain unaffected.
  6. Select Change to All Edges (Square Pin) to resize all edges to create square pins.
  7. Select Retain Pin Selection to retain the selection of pins after they are resized.
  8. Select Retain Highlight on Original Pin to retain the highlights on the pins that were selected for resize.
  9. Select Maintain Abutment to PR Boundary to ensure that pins that are initially abutted to the PR boundary retain the abutment even after they are resized.
  10. Select Show in the Preview section to see a preview of the pins after they are resized before updating the layout.
  11. Verify that the changes are as per your requirements.
  12. Click Hide to close the preview.
  13. Click OK.

Pins in the design are resized as per your specifications.
