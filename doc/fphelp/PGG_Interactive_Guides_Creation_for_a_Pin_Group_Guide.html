Interactive Guide Creation for a Pin Group Guide

You can use the options on the GUIDES tab of the Pin Group Guide form to interactively create guides.

If you try to create a pin group guide constraint with no pins, then an error message is displayed in the CIW, and the guide is not created.

You can create a pin group guide along the following edges:

  • Partial Edge
  • Full Edge
  • Box
  • Free Area
