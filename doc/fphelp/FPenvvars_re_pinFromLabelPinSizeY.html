pinFromLabelPinSizeY

layoutXL.floorplan pinFromLabelPinSizeY float pinHeight

Description

Specifies a custom height to be applied to all pins created by the Create Pin from Label command.

The default value is 0.0, which implies zero height.

GUI Equivalent

Command

Floorplan – Pin Optimization

Plan – Pin Planning – Pin Tool – Create – Create Pin From Label (Virtuoso Layout Suite EXL)

Field

Pin Size – Custom – Y

Examples

envGetVal("layoutXL.floorplan" "pinFromLabelPinSizeY")
envSetVal("layoutXL.floorplan" "pinFromLabelPinSizeY" 'float 0.2)
