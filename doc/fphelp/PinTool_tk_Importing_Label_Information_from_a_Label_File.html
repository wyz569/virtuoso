Importing Label Information from a Label File

To import label information from a label file into your current design in the Pin Tool:

  1. Choose Plan – Pin Planning – Pin Tool. The Pin Browser is displayed.
  2. Choose Tools – Label File – Import.
    The Import Label Info from File form appears.
  3. Specify the required label file name in the Import File field. Alternatively, click Browse to choose the required text file.
  4. Select the Mode in which the labels are to be created.
    • Replace: All the existing labels in the design are deleted and new labels are generated based on definitions in the label file.
    • Update: The existing labels in the design are retained and new labels are generated in the design.
  5. Click OK to generate labels based on the definitions in the selected label file.
