Master Properties in Floorplan Property File

The master properties are defined in the CPH window when you select the soft block option. The properties that are defined are as displayed in the following figure.

The Master properties are defined as follows:

  • Binding properties in the Configure Physical Hierarchy window
  • Global variables for soft block generation
