pinTemplateCreatePGPins

layoutXL.floorplan pinTemplateCreatePGPins boolean { t | nil }

Description

(Virtuoso Layout Suite EXL) Specifies whether power and ground pins are to be created in the layout.

The default value is nil.

GUI Equivalent

Command:

Plan – Pin Planning – Pin Tool – Create – Create Pin Template

Field:

Create PG Pins

Examples

envGetVal("layoutXL.floorplan" "pinTemplateCreatePGPins")
envSetVal("layoutXL.floorplan" "pinTemplateCreatePGPins" 'boolean t)
