Creating and Deleting Pin Labels in the Pin Tool

The Has Label column of the Pin Browser of the Pin Tool specifies whether pins have labels. For pins that do not have a label, you can create labels directly in the Pin Tool. Before creating pin labels, ensure that pin label settings are configured in the Pin Tool Options form.

To create pin labels for one or more pins:

  1. Choose Plan – Pin Planning – Pin Tool. The Pin Browser is displayed.
  2. Select the required pins in the Pin Browser.
  3. Right-click to display the shortcut menu.
  4. Select Label – Create Label.
  5. Check whether Has Label is reset to Yes.

Pin labels are created.

To delete pin labels:

  1. Choose Plan – Pin Planning – Pin Tool. The Pin Browser is displayed.
  2. Select the required pins in the Pin Browser.
  3. Right-click to display the shortcut menu.
  4. Select Label – Delete Label.
  5. Check whether Has Label is reset to No.
