Set Cell Type Form

Use the Set Cell Type form to set the cell type for multiple instances using the Tools – Set Cell Type command from the CIW menu bar.

Field Description

Filter Cellviews

Lets you specify the library, cell, or view name to be filtered.

Library Name

Lets you specify a library name from the drop-down list.

Cell Name

Lets you specify a cell name by which you want to filter in the Cell Name field.

View Name

Lets you specify a view name by which you want to filter in the View Name field. For performance reasons, this field does not accept regular expressions.

Cell Type

Lets you filter cells based on the selected cell type from the drop-down list.

Filter

Filters the list to contain only those cells that match the library, cell, and view name.

Select All

Lets you select all the cellviews displayed in the list.

Set Cell Type

Lets you set the cell type of your soft block to softMacro, blockBlackBox, cover, coverBump, blockRing, core, coreSpacer, or coreAntenna as required.
