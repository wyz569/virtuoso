eaPinResizePinAbutToPRB

layoutXL.floorplan eaPinResizePinAbutToPRB boolean { t | nil }

Description

(Virtuoso Layout Suite EXL) Specifies whether the pins that are abutted to the PR boundary must remain abutted after they are resized.

The default value is nil.

GUI Equivalent

Command:

Plan – Pin Planning – Pin Tool

Field:

Tools – Electrically Aware Pin Sizing – Maintain Abutment to PR Boundary

Examples

envGetVal("layoutXL.floorplan" "eaPinResizePinAbutToPRB")
envSetVal("layoutXL.floorplan" "eaPinResizePinAbutToPRB" 'boolean t)
