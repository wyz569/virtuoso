Pin Spacing Options Form

Use the Pin Spacing Options form to define pin spacing values for each layer.

Field Description

Spacing type

Defines pin spacing values for the various layers in the current design.

  • line-line: Uses the minimum spacing between two wires on a layer for spacing pins. Pin Optimizer reads appropriate spacing rules from the technology file to identify the spacing value. Use this option if you are using a single layer. In multi-layer designs, line-line spacing can cause congestion when vias are inserted, because of the extra jogs that you add to wires to allow for vias.
  • line-via: Uses the minimum spacing between a wire and a via on a layer for spacing pins. In addition to the minimum spacing rules, Pin Optimizer reads valid via definitions and minimum extension rules from the technology file to calculate the spacing value.
  • via-via: Uses the minimum spacing between two vias on a layer for spacing pins. Similar to the line-via option, Pin Optimizer reads valid via definitions, minimum spacing rules, and minimum extension rules from the technology file to calculate the spacing value.

  • specify: Enables you to enter pin spacing values for each layer in the current design manually. By default, the pin spacing values that are retrieved from the technology file are displayed in the individual fields. You can specify the same or different spacing values for each layer.
    Environment variables: spacingType, spacingValue

Update All

Assigns the same pin spacing value for all layers.

Define Pin Spacing (Edge to Edge)

Lets you define the pin spacing values for each layer in the design. Values specified here are the minimum values. These values are scaled up as per the table spacing rules if Use Table Spacing is selected.

The options in the Define Pin Spacing (Edge to Edge) section are not editable in line-via, via-via, and line-line modes. However, the layer spacing values that are retrieved from the technology file are displayed in the individual fields.

Environment variable: perLayerSpacingList
