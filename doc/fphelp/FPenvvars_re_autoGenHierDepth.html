autoGenHierDepth

layoutXL.floorplan autoGenHierDepth int hierarchyDepth

Description

Specifies the level up to which the hierarchy must be generated.

The default value is 31, which is the maximum value.

GUI Equivalent

Command:

Floorplan – Auto Generate Hierarchy

Plan – Generate – Auto Generate Hierarchy (Virtuoso Layout Suite EXL)

Field:

Generate Hierarchy to Level

Examples

envGetVal("layoutXL.floorplan" "autoGenHierDepth")
envSetVal("layoutXL.floorplan" "autoGenHierDepth" 'int 25)
