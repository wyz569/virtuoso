Database Preparation for IO Planning and Placement

In database preparation, IO Placer requires information about the technology file, site pattern, and cell type for pads, fillers, and corners.

In the technology file, ensure that at least one site definition exists and is of a PAD type.

  • Site Pattern: Lets you define a site pattern for each instance, that is, pad, corner, filler. A site pattern is a set of site definitions. The site definition is read from the technology file.
