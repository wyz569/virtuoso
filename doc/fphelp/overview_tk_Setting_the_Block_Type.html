Setting the Block Type

To select the required block:

  1. Right-click in the layout canvas to display the shortcut menu, and select Block.
  2. Select one of the block types: Digital, Custom, Any, or As is.
    When you select multiple blocks of different block types, and then display the shortcut menu, option As is selected by default. As is is not a block type. It signifies that there are multiple block with different types in the selection set.
  3. If you select Any, you can use the Floorplan Global Options form to specify the pin grid to be used for placing pins during pin optimization.
