Inserting Filler Cells

To insert filler cells:

  1. Define the design area. It could either be an entire design or a selected region of a design. After defining the design area, select the filler cells from the libraries that are to be used.
  2. Choose Floorplan – I/O Placer – Insert Filler.
    In Layout EXL and higher tiers, choose Plan – I/O Placer – Insert Filler.
    The Insert Filler Cells form is displayed.
  3. In the Region Selection Mode section, choose if the filler needs to added to the entire design or a selected region.
    1. Click Draw to select the area on the canvas on which the operation must be performed. Choose from Entire Design or Selected Region. This automatically populates the coordinates field.
    2. Define the coordinates of the selected region in the coordinates field. You can manually edit the coordinates.
    3. Click Zoom to for zooming in the selected region.
  4. Enable selection of few cells from entire range of libraries/cells/views using the Select Filler Cells section. The few selected cells would be used by the engine as filler cells.
  5. To select a lib:cell:view for the filler cell, you can type the library, cell, and view names in the corresponding fields or perform the following steps:
    1. Select the library from the Library Name drop-down list and click Filter.
      The value of the selected lib:cell:view name along with the site name is displayed in the Library Name, Cell Name, View Name, and Site Name columns, respectively.
    2. Click the right arrow button to add the selected library/cell/view to the list of fillers in the right pane. You can add multiple lib:cell:view from the Library Name, Cell Name, View Name, and Site Name table and add them to the filler list in the right pane using a single click.
      You can click the left arrow button to delete the selected lib:cell:view name as a filler.
  6. Define the instance name prefix and the placement status of the filler cell in the Filler Instance Attribute section.
    1. Specify instance name prefix, which is pre-pended to the inserted physical only filler instances in Instance Name Prefix. This will help you recognize the filler inserted in a specific row or location for deleting them or doing some edits.
      The physicalOnly attribute is automatically added to the new cells that are inserted.
    2. Select Mark Fillers as Fixed field to specify the placement status of inserted cells as Fixed. By default, the check box is not selected. When the check box is deselected, the placement status of the inserted cells is set to Placed.
  7. Click OK or Apply.

Fillers are inserted according to their site patterns and row as shown in the following image:

The filler cells are then placed from the biggest to the smallest in order to minimize the number of inserted filler cells. Filler cells with height less than that of the row site are not considered for placement because if their heights are lesser than the row site, there will be discontinuity in the rails.
