Creating a Box Pin Group Guide

You can use the Box option in the GUIDES tab of the Pin Group Guide form to draw one or more areas in the layout window.

To create a box pin group guide:

  1. Choose Floorplan – Pin Group Guides – Create.
    In Layout EXL and higher tiers, choose Plan – Pin Planning – Create/Edit Pin Group Guide.
    The Pin Group Guide form appears.
  2. Specify the name of the new pin group guide in Group Name field or click New to generate a unique group name.
    You can edit the automatically-generated group name.
  3. Draw the pin group guide on the layout canvas using the options on the GUIDES tab.
  4. Create pin group guides on the PR boundary edges that intersect the selected boxes.
  5. Press Esc when you are done.
  6. Select Split into edge guides during create check box, top-level and soft block edges that intersect the box are created as the partial- or full-edges for creating pin group guides.
  7. Deselect the Split into edge guides during create check box, a bounding box is created for the selected area. Pin edges within the bounding box are selected for placing pins.
Whenever the PR boundary of a digital block is modified though stretch or move operations, the area pin group guides on these blocks are automatically snapped to the PR boundary.
