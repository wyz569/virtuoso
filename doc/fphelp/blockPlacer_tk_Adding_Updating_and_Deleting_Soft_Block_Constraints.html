Adding, Updating, and Deleting Soft Block Constraints

You can add, update and delete the side constraints for soft blocks using the Constraints tab of the Block Placer form.

To modify the side constraints for soft blocks:

  1. Choose Floorplan – Block Placer command to display the Block Placer form.
    In Layout EXL and higher tiers, choose Plan – Block Planning – Place Blocks.
  2. In the Constraints tab, choose Soft Block Constraints sub-tab.
  3. Select the row for the soft block that you want to modify.
    The Area, Minimum AR, and Maximum AR fields below the table show the values for the specified soft block.
  4. Modify the values in the Area, Minimum AR, and Maximum AR fields. The valid values are positive numbers, AS IS, and empty field. If a field value is AS IS, then the corresponding value in the table is not updated.
  5. Click Update Selected to apply the changes to the selected soft block as shown in the image. You can also update multiple soft blocks simultaneously by selecting the corresponding rows.
  6. If you select multiple rows, the Area, Minimum AR, and Maximum AR fields display the values if they are same for all the selected soft blocks, else it displays AS IS as shown in the image.
  7. Click Update Selected to update all the selected rows with the new values.
  8. Click Update All to update all rows.
