pinFromLabelSetCustomPurpose

layoutXL.floorplan pinFromLabelSetCustomPurpose string "layerPurpose"

Description

Specifies the purpose of the layer in which pins are to be created by the Create Pin from Label command. This environment variable is valid only when pinFromLabelSetPinLayers is set to Same Layer. Custom Purpose.

GUI Equivalent

Command:

Floorplan – Pin Tool – Create – Create Pins From Label

Plan – Pin Planning – Pin Tool – Create – Create Pins From Label (Virtuoso Layout Suite EXL)

Field:

Pin Layer – Same Layer, Custom Purpose

Examples

envGetVal("layoutXL.floorplan" "pinFromLabelSetCustomPurpose")
envSetVal("layoutXL.floorplan" "pinFromLabelSetCustomPurpose" 'string "drawing")
