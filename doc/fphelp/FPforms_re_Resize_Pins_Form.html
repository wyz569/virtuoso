Resize Pins Form

Choose Tools – Resize Pins to display the Resize Pins form. Use the Resize Pins form to register, run, or deregister the pin resize estimator SKILL functions.

Field Description

Run On

Lets you specify whether the procedure must be run on All or Selected pins. In case of Selected pins, you must select the pins before launching the Resize Pins form.

Environment variable: pinResizeMode

Resize function

Selects the Resize function to be run. Resize function lists all registered pin resize functions.

None: Indicates that there are no registered functions.

Environment variables: pinResizeAllEstimators, pinResizeCurrentEstimator

Register/DeRegister

Selects the pin resize estimator SKILL function to be registered or deregistered from the Deregister drop-down list.

File Path

Specifies the file path of the SKILL hook procedure file that contains the pin resize definitions.

Environment variable: pinResizeFiles

Load

Loads the procedure.

Browse

Selects the required file and closes the window.

Register/DeRegister – Function Name and Nickname

Lets you register and deregister the procedures. To do this, select the corresponding function name from the Register/DeRegister drop-down list.

Add

Registers the function. The function is listed in the Resize function list, from where you can run it.

Delete

Deletes the function. The function is listed in the DeRegister list, from where you can delete it.

Apply

Runs the selected function. The pins in the design view are resized.
