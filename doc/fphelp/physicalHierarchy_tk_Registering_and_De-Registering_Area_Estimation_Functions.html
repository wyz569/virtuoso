Registering and De-Registering Area Estimation Functions

You can use the Register and Deregister Area Estimators form to register or deregister an area estimator function, for estimating the area of the PR boundary, or that of a virtual hierarchy block.

To register an area estimator function:

  1. Select Floorplan – Auto Generate Hierarchy.
    The Auto-Generate Hierarchy form is displayed which lets you quickly generate a physical hierarchy by applying common parameters.

  2. Click Add beside the Use Area Estimator list.
    The Register and Deregister Area Estimators form appears.
  3. Expand the Register section.
  4. Specify the File Path of the area estimator file that contains the boundary definitions. Alternatively, click Browse to select the required file and close the window.
  5. Click Load to load the area estimator function.
  6. Specify a Function Name and Alias.
  7. Click Add to register the area estimator function.
  8. Click Close.
  9. Select the area estimator function that you registered from the Use Area Estimator list.
  10. Click Apply to run the area estimator function.

To de-register an area estimator function:

  1. In the Register and Deregister Area Estimators form, select the area estimator function from the Deregister – Alias list.
  2. Click Deregister.
  3. Click Close.
