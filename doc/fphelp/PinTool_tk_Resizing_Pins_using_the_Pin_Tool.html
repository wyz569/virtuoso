Resizing Pins using the Pin Tool

Use the Resize Pins form to register, run, or deregister the pin resize estimator SKILL functions.

To run a pin resize estimator SKILL function:

  1. Choose Plan – Pin Planning – Pin Tool. The Pin Browser is displayed.
  2. Choose Tools – Resize Pins to display the Resize Pins form.
  3. Specify whether the procedure must be run on All or Selected pins.
    In case of Selected pins, you must select the pins before launching the Resize Pins form.
  4. Select the Resize function to be run. Resize function lists all registered pin resize functions.
    Select None to indicate that there are no registered functions.
    Click Apply to run the selected function. The pins in the design view are resized.
  5. In the Register/Deregister section, you can register a pin resize estimator or SKILL function. Specify the File Path of the SKILL hook procedure file that contains the pin resize definitions. Alternatively, click Browse to select the required file and close the window.
  6. Click Load to load the procedure.
  7. Specify a Function Name and Nickname in the Register field.
  8. Click Add to register the function. The function is listed in the Resize function list, from where you can run it. Click Apply to run the procedure.
  9. Optionally, select the pin resize estimator SKILL function to be deregistered from the Deregister drop-down list.
  10. Click Delete.
  11. Close the Resize Pins form.
  12. Click Refresh in the Pin Tool to update the pin attributes in the Pin Browser.
