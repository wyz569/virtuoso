astPinMoveMatchLabelLayerPurpose

layoutXL.floorplan astPinMoveMatchLabelLayerPurpose string "Purpose"

Description

(Virtuoso Layout Suite EXL)Specifies the purpose to which labels are to be moved when running the assisted move command. This setting is valid only when astPinMoveMatchLabelLayerChoice is set to Same Layer.Custom Purpose.

GUI Equivalent

Command:

Plan – Pin Planning – Pin Tool – Tools – Options

Field:

Match Label Layer– Same Layer.Custom Purpose

Examples

envGetVal("layoutXL.floorplan" "astPinMoveMatchLabelLayerPurpose")
envSetVal("layoutXL.floorplan" "astPinMoveMatchLabelLayerLPP" 'string "drawing")
