Block Annotations Options Form

Use the options in the Block Annotations Options form to customize the information displayed in the block annotations. The Block Annotations form is available for both edit and read-only cellviews.

Field Description

Annotations

This section provides the information to be included in the block annotations.

Display Mode

Specifies whether the block annotations are to be displayed for All or Selected blocks.

Environment variable: blockAnnotationSelMode

Instance name and Master LCV

Displays relevant information in the block annotations.

Environment variables: blockAnnotationInstName, blockAnnotationTextLPP

Area

Displays the total block area within the PR boundary. Select the required units:

  • um^2: Square micrometers
  • mm^2: Square millimeters
  • du^2: Square database unit

Environment variables: blockAnnotationArea, blockAnnotationAreaUnits

Shrink Factor

Specifies the factor by which each edge has to be shrunk.

Environment variable: blockAnnotationShrinkFactor

Rotation

Specifies the block orientation information.

Environment variable: blockAnnotationRotation

Cell Type

Displays the cell type of your selected soft block.

Environment variable: blockAnnotationCellType

Display Options

This section lets you customize the text settings.

Text Size

Sets the text size of block annotations to one of the following values:

  • Auto: Calculates the annotation size such that all annotations are inside the instance boundary.
  • Custom: Provides a slider to drag to the required value.

Environment variable: blockAnnotationTextSize

Text Layer

Lets you select a text layer based on which the block annotation text is to be formatted.

Environment variable: blockAnnotationTextLPP
