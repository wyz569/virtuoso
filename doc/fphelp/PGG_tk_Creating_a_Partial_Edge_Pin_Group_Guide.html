Creating a Partial Edge Pin Group Guide

To create a pin group guide that cover a partial edge:

  1. Choose Floorplan – Pin Group Guides – Create.
    In Layout EXL and higher tiers, choose Plan – Pin Planning – Create/Edit Pin Group Guide.
    The Pin Group Guide form appears.
  2. Specify the name of the new pin group guide in Group Name field or click New to generate a unique group name.
    You can edit the automatically-generated group name.
  3. Draw the pin group guide on the layout canvas using the options on the GUIDES tab.
  4. Choose Partial Edge.
  5. Click Create Guide.
  6. Click the required edge to indicate the first point of the pin group guide in the layout design.
  7. Click to indicate the last point of the pin group guide. If you are creating multiple pin group guides, continue to click the edges to create the additional pin group guides.
    A pin group guide is not created if the length of the guide is too small for placement of the selected pins, that is if the guide is smaller than the sum of the minWidth, keep out, and spacing values for metal1. A suitable warning message is displayed in the CIW.
  8. Press Esc when you are done.
  9. Choose one of the options Constraints Only, Preview, or Optimize (default).
  10. Choose a Mode.
  11. Click Apply to submit the updates you made and view the pin group guide in the layout design.

Area boundaries are created and the specified pins are placed within the pin group guide that you created.

Width of the pin group guide equals the unit width area boundary.

If you enter a top-level or soft block corner as the first point of a pin group guide, then, by default, you are prompted to create a vertical guide (along the y-axis).

Creating a Horizontal Guide

To create a horizontal guide:

  1. Press F3 to display the Create Guide popup box.
  2. Choose Horizontal from the Snap Mode drop-down list.
  3. Click Hide to close the Create Guide popup window.
  4. Proceed to create a horizontal pin group guide.

You can use this method to create orthogonal pin group guides.
