pinTemplateGroundNet

layoutXL.floorplan pinTemplateGroundNet string "ground_net"

Description

(Virtuoso Layout Suite EXL) Specifies the net on which pins with signal type set to Ground are to be created.

The default value is nil. On running the Create Pin Template command, the tool displays a message indicating that the value must be specified.

GUI Equivalent

Command:

Plan – Pin Planning – Pin Tool – Create – Create Pin Template

Field:

Ground Net

Examples

envGetVal("layoutXL.floorplan" "pinTemplateGroundNet")
envSetVal("layoutXL.floorplan" "pinTemplateGroundNet" 'string "net1")
