pinTemplateCreatePGRailsLPPList

layoutXL.floorplan pinTemplateCreatePGRailsLPPList string "list_of_LPPs"

Description

(Virtuoso Layout Suite EXL) Specifies the layer-purpose pairs on which the power and ground rails are to be created.

The default value is "", in which case the list is loaded from the technology file with the LPPs of all metal and poly layers that were available when the form was initially opened.

GUI Equivalent

Command:

Plan – Pin Planning – Pin Tool – Create – Create Pin Template

Field:

Create PG Pins

Examples

envGetVal("layoutXL.floorplan" "pinTemplateCreatePGRailsLPPList")
envSetVal("layoutXL.floorplan" "pinTemplateCreatePGRailsLPPList" 'string "(Poly drawing) (metal1 drawing)")
