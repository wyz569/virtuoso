Importing Pin Information from a Pin File

To import a pin file in the Pin Tool:

  1. Choose Plan – Pin Planning – Pin Tool. The Pin Browser is displayed.
  2. Choose Tools – Pin File – Import.
    The Import Pin Info from File form appears.
  3. Specify the required pin file name in the Import File field. Alternatively, click Browse to choose the required text file.
  4. Select the Mode in which the pins are to be created.
    • Replace: All the existing pins in design are deleted and new pins are generated based on definitions in the pin file.
    • Update: The existing pins in the design are retained and new pins are generated in the design.
  5. Click OK to generate pins based on the definitions in the selected pin file.
