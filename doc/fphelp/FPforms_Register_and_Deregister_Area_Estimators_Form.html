Register and Deregister Area Estimators Form

Use the Register and Deregister Area Estimators form to register, run, or deregister the area estimation procedures.

Field Description

Register

Lets you register the procedures.

File Path

Specifies the file path of the area estimator file that contains the boundary definitions.

Load

Loads the area estimator function.

Browse

Selects the required file and closes the window.

Function Name

Lets you specify a function name.

Alias

Lets you specify an alias.

Add

Registers the function. The function is listed in the Register function list, from where you can run it.

DeRegister

Lets you deregister the procedures.

Alias

Lets you specify the corresponding function name.

DeRegister

De-registers the function.
