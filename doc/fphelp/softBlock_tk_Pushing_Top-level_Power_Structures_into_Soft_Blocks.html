Pushing Top-level Power Structures into Soft Blocks

To push the top-level power structures to the block level:

  1. Select the soft block to which the overlapping power geometries are to be pushed.
  2. Choose Floorplan – Soft Block Update – Push Into Blocks.
    In Layout EXL and higher tiers, choose Plan – Block Planning – Push Into Blocks.
  3. If soft blocks were not selected before invoking the form, select the required soft block instances. The Push Into Blocks form appears.
    The preselected soft blocks are listed in the tree structure.
  4. (Layout EXL and Higher Tiers) To add soft blocks:
    1. Click Browse.
    2. Select the required cellview from the Library Browser.
    3. Click Close to close the Library Browser.
    4. Click Add in the Push Into Blocks form.

    Blocks from the selected cellview are listed in the tree structure.
    To delete soft blocks from the tree:
    1. Select the required cell.
    2. Right-click and choose Delete Cell.

    Blocks from the selected cellview are deleted from the tree.
  5. Edit the Target Lib, Target Cell, and Target View values directly in the table as per your requirement.
  6. Specify which soft blocks need to be pushed into blocks. By default, Push is set to Yes for all listed soft blocks. Change the value to No for the soft blocks you do not want to push.
  7. Specify the target cellview in the Target View section. If the form is invoked after selecting a soft block, the library, cell, and view names are populated by default.
  8. Select the required options from the Options section.
  9. Click OK.
