analyzeConnCheckLogicalConn

layoutXL.floorplan analyzeConnCheckLogicalConn boolean { t | nil }

Description

Checks connectivity information in the schematic view for the objects selected in the layout when at least one of the selected objects is unconnected in the layout view.

The default value is nil.

GUI Equivalent

Command:

Connectivity – Analyze – Connectivity

Field:

Check – Logical Connectivity

Examples

envGetVal("layoutXL.floorplan" "analyzeConnCheckLogicalConn")
envSetVal("layoutXL.floorplan" "analyzeConnCheckLogicalConn" 'boolean t)
