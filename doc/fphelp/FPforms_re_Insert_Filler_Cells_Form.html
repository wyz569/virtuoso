Insert Filler Cells Form

Use the Insert Filler Cells form to insert filler cells in the empty spaces between PADs in IO rows.

Field Description

Region Selection Mode

This section specifies the mode for running the command.

Entire Design

Enables you to select the entire design for inserting filler cells. This means that filler cells will be inserted in all the IO rows of the design.

Selected Region

Enables you to select a region for inserting filler cells. This means that only few IO rows need to be used for filler insertions.

Draw

Lets you select an area of the design in the canvas. The values are automatically populated in the Coordinates field.

Coordinates - llx, lly, urx, ury

Defines the coordinates of the selected region. You can manually edit the values.

Zoom to

Lets you zoom the selected region.

Select Filler Cells

This section specifies the cells to be used as filler cells.

Library Name, Cell Name, and View Name

Specifies the cellview that contains the required cells.

Filter

Lets you choose the lib:cell:view to be used as fillers based on the values entered in the Library Name, Cell Name, and View Name fields.

Adds the selected lib:cell:view name as a filler and displays it in the filler table. You cannot add duplicate values in the filler table.

Deletes the selected lib:cell:view from the filler table.

Filler Instance Attribute

Defines the instance name prefix and the placement status of filler cells.

Instance Name Prefix

Specifies a prefix for the physical-only filler instances.

Mark Fillers as Fixed

Specifies the placement status of the filler cells as Fixed.
