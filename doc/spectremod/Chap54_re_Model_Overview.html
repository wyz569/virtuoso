Model Overview

Advanced SPICE model for GaN HEMTs, ASM-GaN-HEMT is a physics-based model for these devices. The model can be structured into following three parts:

  • Core channel-charge or surface-potential calculations
  • Core terminal current and charge model, and
  • Models for real device effects in the device.
