Product Version 23.1, June 2023

Model Version and Development

MMSIM supports DIODE_CMC model version 1.0.0 and 2.0.0. The current available version for DIODE_CMC is version 2.0.0.

Version Update and Enhancement

Version 2.0.0 Enhancement

Following enhancements have been made in version 2.0.0:

  • Internal node AIK has been added
  • NQS recovery mode has been added.
  • Following parameters have been added for NQS recovery mode: corecovery, njh, njdv, ndibot, ndigat, ndisti, inj1, inj2, nqs, tau, wi, depnqs, tnom, taut, injt, and qrr.
