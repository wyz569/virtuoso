Wire RC Model

If you specify the capacitance for a two terminal resistor, the model is a Wire RC model. If you do not specify the capacitance explicitly, it is computed from the physical length and width of the resistor using Cj and Cjsw. You can use the model parameter cratio to allocate the parasitic capacitance of the wire element between the model’s input capacitor and output capacitor.
