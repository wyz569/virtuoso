TMI 2.0 Model

The TMI 2.0 model feature is released in MMSIM10.1, provides the following new capabilities:

Flexible device topology with customized model formulations

With MMSIM 10.1 release, additional component(s) and/or internal node(s) can be added on
top of existing standard public domain model topology. This allows the simulator to evaluate
(during runtime), the customized model as well as the standard core model formulations,
gather/scatter device information and conduct matrix stamping for various type of circuit
analysis.

Support for reliability aging modeling and analysis for simulators with and without aging capability

With this feature, the user can add customized aging algorithms for all age integration, extrapolation and mapping. The Safe Operating Areas (SOA) check is also supported.
