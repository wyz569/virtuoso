Linear Voltage Controlled Voltage Source (vcvs)

Current through the voltage source is calculated and is defined as positive if it flows from the positive terminal to the negative terminal through the source.

This device can also model ideal digital gates.

This device is supported within altergroups.

Sample Instance Statement

e1 (out1 0 pos neg) vcvs gain=10

Instance Syntax

Name  p  n  ps  ns ... vcvs parameter=value ...

Instance Parameters

m=1

Multiplicity factor.

type=vcvs

Type of the source. Possible values are vcvs, and, nand, or, or nor.

delta=0

Smoothing parameter. This parameter can lead to circuit convergence. Its value should be between 0 and 0.5, both ends included. The smaller the delta is, the sharper the corner is.

Linear source parameters

gain=0 V/V

Voltage gain.

td=0.0 s

Time delay.

min (V)

Minimum output voltage.

max (V)

Maximum output voltage.

abs=off

When abs is set to on, the output voltage is an absolute value. Possible values are off or on.

PWL source parameters

file

Name of the file that contains the voltage or voltage pairs that define the PWL transfer function.

pwl=[...]

Vector of the voltage or voltage pairs that defines the PWL transfer function. The format of the vector is [in1 out1 in2 out2 ... ].

scale=1

Scale factor for the PWL output voltage.

stretch=1

Scale factor for the PWL controlling voltage.

Temperature effects parameters

tc1=0 1/C

Linear temperature coefficient.

tc2=0 C-2

Quadratic temperature coefficient.

trise=0.0K

Temperature rise from ambient.

The Smoothing Factor

The parameter delta is the smoothing factor. When its value is non-zero, a curve is introduced to smooth the corner. The value of delta defines the starting and ending points of the curve that is used to smooth the corner. The minimum value, 0, means no smoothing curve is introduced. The maximum value, 0.5, means the distance between the starting point of the curve to the corner point is half of the shortest distance. The shortest distance is the length of the shortest segment of the piecewise linear function. Or, when there is no shortest segment in the controlling function, that is, only abs=on is specified, Spectre uses 1 as the shortest distance.
