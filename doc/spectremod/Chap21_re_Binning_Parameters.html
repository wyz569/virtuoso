Binning Parameters

The table below lists geometry sensitivity factors.

The value of a parameter is calculated from the following equation:

If binunit =1,

P = p0 + 1e-6 * wp0 / Weff + 1e-6 * lp0 / Leff + 1e-12 * pp0 / ( Weff * Leff )

else

P = p0 + wp0 / Weff + lp0 / Leff + pp0 / ( Weff * Leff )

where P is the parameter name.
