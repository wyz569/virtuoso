HISIM_HV Model (hisim_hv)

HiSIM HVMOS (HiSIM_HV) is an extension of HiSIM model that is a surface potential based MOSFET model. HiSIM_HV adds high voltage device specific feature, like drift region resistance, and so on. It can be used to model both laterally-diffused MOS (LDMOS) and high-voltage MOS (HVMOS) types of device structures. HiSIM_HV is selected as CMC’s standard high voltage device model. The latest model version is HiSIM_HV 2.50.
