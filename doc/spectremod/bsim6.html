BSIM6 Model (bsim6)

BSIM6 is the new Bulk MOSFET model from the BSIM Group. The model provides excellent accuracy compared to measured data in all regions of operation. It features model symmetry valued for analog and RF applications while maintaining the strong support and performance of the BSIM model valued for all applications. The available versions for the bsim6 model are 6.00(6.0.0), 6.10(6.1.0), 6.11(6.1.1), 106.2(Bsimbulk 106.2.0), 107.0(Bsimbulk 107.0), and 107.10(Bsimbulk 107.1.0).

Starting with version 106.2, BSIM6 has been renamed to BSIM-BULK. Spectre supports both master names bsim6 and bsimbulk.
