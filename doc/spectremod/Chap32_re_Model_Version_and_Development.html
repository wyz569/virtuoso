Model Version and Development

Spectre/APS asupports the following versions of HiSIM2 model: 2.50, 2.51, 2.60, 2.61, 2.70, 2.80, 2.90, 2.91, 3.10(3.1.0), 3.11(3.1.1), and 3.20(3.2.0). Different model versions can be obtained by setting different model parameter VERSION = 2.50/2.51/2.60/2.61/2.70/2.80/2.90/2.91/3.0.0/3.1.0/3.1.1/3.2.0. 2.50 is set as the default version. The parameter cannot be parameterized, its value must be constant.
