TMI 1.0 Model

TMI, TSMC Model Interface, is supported in MMSIM7.1.0.

In order to support customized model equations for advanced technologies, TMI is proposed to provide an alternative approach for this request in addition to adding more macro models.

TMI provides both good flexibility as macro model and good simulation efficiency as build-in standard model at the same time.
