.param

.param param1=value1 [param2 = value2 ... paramn = valuen]

or 

.param func_name='expression'

Description

Defines the parameters and user-defined functions.

Examples

.param vcc=2.5
.param half_vcc='0.5*vcc'
.param g(x)='5*x+0.5'
.param f(x)='g(x)+5*x+0.5'
