SPICE Options Supported by Spectre FX

Spectre FX supports the following SPICE options that you can use with the .options or .option SPICE command:

  • autostop
  • geoshrink
  • gmin
  • measdgt
  • parhier
  • scale
  • scalm
  • search
  • tnom
