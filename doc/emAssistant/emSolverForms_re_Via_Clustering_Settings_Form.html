Via Clustering Settings Form

Use the Via Clustering Settings form to configure options for via clustering. Via clustering requires the maximum spacing and maximum number of cuts for each via layer.

Field Description

Cut Layer

Specifies the names of via layers found in the layout

Max Spacing

Specifies the spacing (in microns) between cuts. Any cuts that are within the given spacing are considered for clustering. Set this value to greater than or equal to cut spacing to cluster the vias on a layer.

Max Cuts

Specifies the maximum array size for a cluster. For example, a value of 50 would create a single shape up to a 50 x 50 array. This is typically set to a large number so that all cuts in a via are clustered.
