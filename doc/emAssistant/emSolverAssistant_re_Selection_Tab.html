Product Version IC23.1, November 2023

Selection Tab

The following table describes the columns available in the Selection tab of the Electromagnetic Solver assistant.

Field/Command Description

Object Selection

Displays the instances, nets, and reference models that you select for inclusion in a model.

This list is not available when you select the Extract Full Cellview check box.

Add selected instances, nets, or reference models

Enables extraction of a full cellview for a model.

Remove instances, nets, or reference models

Enables extraction of a full cellview for a model.

Include devices between nets

Includes any schematic instance, such as LVS resistors and net splitters, where all pins are connected to nets in the model automatically.

This command is only for LVS models.

SVDB Directory

Specifies a directory to load the SVDB data into memory.

This command is only for LVS models.

Excluded Cells

Specifies a list of cells to be excluded when you are using a full cellview for a model.

This list is available only when you select the Extract Full Cellview check box.

Edit the list of excluded cells

Lets you edit the Excluded Cells list.

This list is available only when you select the Extract Full Cellview check box.

Highlight shapes inside the cutting boundary except the excluded cells

Highlights the selected shapes when you are using a full cellview for a model.

Cutting Boundary

Shows the coordinates of the cutting boundary that you choose for a model.

This command is not available for LVS models.

Select cut type

Lets you specify a shape on the layout canvas.

This command is not available for LVS models.

Draw Rectangle

Changes the mouse pointer and lets you select a rectangular boundary.

Draw Polygon

Changes the mouse pointer and lets you select a rectangular boundary.

Show Cutting Boundary

Highlights the cutting boundary in the layout canvas.

This command is not available for LVS models.

Config File

Specifies the path to the config file that contains the detail of the material file to be used for a package layout.

This field is visible for only a package layout.
