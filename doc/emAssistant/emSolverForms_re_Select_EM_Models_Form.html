Select EM Models Form

Use the Select EM Models form to choose a reference model for an instance selected for another model in a cross-fabric design..

Field Description

EM Models for Selected Instance

Displays a list of all the models available for the instance selected on the Selection tab of the Electromagnetic Solver assistant.

OK

Successfully completes the selection of a reference model.

Cancel

Closes the form without selection of a reference model.
