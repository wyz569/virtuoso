Virtuoso Electromagnetic Solver Assistant User Guide
Product Version IC23.1, November 2023

Menus

The 3D Viewer contains five menus described in this section.

Menu Command Icon Description

File

Close

Close All

Closes the .vmesh currently displayed in the canvas.

Closes all .vmesh files currently displayed in the canvas.

View

Zoom Fit

Zoom In

Zoom Out

Zoom to Area

Fits the mesh in the given canvas area.
Bindkey: F

Magnifies the mesh by a factor of 2.
Bindkey: Ctrl + Z

Reduces the mesh by a factor of 2.
Bindkey: Shift + Z

Zooms the mesh view to a selected area.

To select the rectangular area, move the pointer to the top-left corner of the area to zoom and right-click or press the Z bindkey. Drag the pointer to the bottom-right corner and release the mouse button to complete the selection.

View Top

Displays the top view of the model.

View Bottom

Displays the bottom view of the model.

View Left

Displays the left side view of the model.

View Right

Displays the right side view of the model.

View Front

Displays the front view of the model.

View Back

Displays the back view of the model.

Isometric SW

Displays the 3D view as if you are viewing it from the lower-left corner.

Isometric SE

Displays the 3D view as if you are viewing it from the lower-right corner. This is the default view.

Isometric NE

Displays the 3D view as if you are viewing it from the upper-right corner.

Isometric NW

Displays the 3D view as if you are viewing it from the upper-left corner.

Z-Scale

Scales the mesh on the z-axis by the selected factor. This command provides eight predefined scaling factors to choose from. The scaled z-axis allows you to clearly see the thin layers or to view the mesh between the layers.

Options

Display Orient Axes

Displays or hides the orientation axes in the canvas.

Related environment variable: displayOrientAxes

Display Rotation Axes

Displays or hides the rotation axes in the canvas.

Related environment variable: displayRotationAxes

Display Wire Frames

Displays or hides the edges of the mesh. Hide the edges when you want to view the solid metal.

Related environment variable: displayWireframes

Display Surfaces

Displays or hides the surfaces. Hide the surfaces for a transparent mesh view with only wire frames.

Related environment variable: displaySurfaces

Display Ports

Displays the ports in the mesh. Port numbers help you identify the ports corresponding to the ports in the Ports tab of the Electromagnetic Solver assistant. The selected ports are highlighted in a color to recognize them easily.

Related environment variable: displayPorts

Light

Activates lights thrown on the canvas. This command is helpful in identifying edges when you hide wire frames.

Related environment variable: lightEnabled

Provides commands to show or hide toolbars.

Help

This is the standard Virtuoso Help menu. For details, see Getting Help.
When you use the commands to pan, zoom, rotate, scale, or change the view type, the tool updates the display of the model. It also saves the previous view of the model. You can use the W and Shift + W bindkeys to open the previous or next view, respectively. Use the viewStackMaxCapacity variable to specify the maximum number of views the tool can save in the memory stack.
