VEM_PDF_VIEWER

Sets the full path to the PDF viewer in which the layer stackup for EMX models is displayed. It is not sufficient to provide only the path to the installation hierarchy. By default, this variable is set to the path for the Evince PDF viewer.

Example

setenv VEM_PDF_VIEWER path-to-pdfViewer
