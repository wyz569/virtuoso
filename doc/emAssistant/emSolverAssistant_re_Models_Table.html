Models Table

The following table describes the columns available in the Models table of the Electromagnetic Solver assistant.

Column Description

Name

Name of the model.

Simulator

Simulator selected to run EM simulations for the model.

Corner

Name of the process corner.

Status

Status of the model.

Possible values are:

  • pending: When you are configuring a model and electromagnetic simulation is not run.
  • running: When the electromagnetic simulation is in progress.
  • done: After the simulation is complete.
  • exported: After you export the S-parameter file to a schematic or S-param view.
