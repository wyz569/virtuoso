Create S-Parameter View Form

Use the Create S-Parameter View form to save a model as an S-parameter cellview.

Field Description

Library

Specifies the name of the library in which you want to save the extracted view.

Related environment variable: exportLibName

Cell

Specifies the name of the cell in which you want to save the extracted view.

Related environment variable: exportCellName

View

Specifies the name of the new S-parameter view to be saved.

Related environment variable: exportSparamViewName

Create Symbol

Specifies whether to save a symbol view in addition to the S-parameter view in the same library and cell.
