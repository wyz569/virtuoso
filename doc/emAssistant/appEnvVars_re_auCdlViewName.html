auCdlViewName

vem.cpd auCdlViewName string aucdlViewName

Description

Generates customizable auCdl view for custom passive device during device signature generation. The default value is aucdl.

GUI Equivalent

None

Examples

envGetVal("vem.cpd" "auCdlViewName")
envSetVal("vem.cpd" "auCdlViewName" 'string "auCdlViewName")
