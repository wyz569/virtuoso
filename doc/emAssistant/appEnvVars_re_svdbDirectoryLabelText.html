svdbDirectoryLabelText

vem svdbDirectoryLabelText string dirName

Description

Specifies the user-defined value of the SVDB Directory label. The default value is SVDB Directory.

GUI Equivalent

Command

Window – Assistants – Electromagnetic Solver

Field

SVDB Directory

Examples

envGetVal("vem" "svdbDirectoryLabelText")
envSetVal("vem" "svdbDirectoryLabelText" 'string "LVS Ouput Folder")
