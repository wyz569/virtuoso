Simplify Layout for EM Simulation Form

Use the Simplify Layout for EM Simulation form to specify the rules for shape simplification.

Field Description

Skip all shapes on LPP

This section provides options to create shape simplification rules based on their fill size to skip dummy metal fills from one or more layer-purpose pairs.

List of rules

Displays all the rules created in this section

Lpp

Specifies the layer-purpose pairs for which you want to create a rule. You can choose from the drop-down list that displays all the layer-purpose pairs from the design.

Fill size

Specifies the metal fill size to be used for the rule

Add

Adds a new rule to the list

Delete Selected

Deletes the selected rules from the list

Clear

Removes all the rules from the list

Remaster Instances

This section provides options to create rules to change the master for instances in the layout. For example, you may want to run a simulation by using a manually simplified version of a cellview instead of the actual cellview used in the design.

List of rules

Displays all the rules created in this section

From

Contains three drop-down lists to specify the names of the current library, cell, and view master of the instances

To

Contains three drop-down lists to specify the names of the current library, cell, and view to which the instances belong

Add

Adds a new rule to the list

Delete Selected

Deletes the selected rules from the list

Clear

Removes all the rules from the list

Remove dangling shapes

This section provides options to create rules based on the layer size to remove dangling connected shapes from one or more layer-purpose pairs in the layout

List of rules

Displays all the rules created in this section

Lpp

Specifies the layer-purpose pairs for which you want to create a rule. You can choose from the drop-down list that displays all the layer-purpose pairs from the design.

Min Layer Size

Specifies the minimum size of shapes you want to retain on the layer. The tool will remove any shape smaller than the specified size.

Direction

Specifies the direction in which the minimum size of shapes is measured.

Add

Adds a new rule to the list

Delete Selected

Deletes the selected rules from the list

Clear

Removes all the rules from the list

Merge shapes

This section provides options to create rules to merge shapes that are placed on the same layer or on the layers above or below it if they are within the given distance.

List of rules

Displays all the rules created in this section

Lpp

Specifies the layer-purpose pairs for which you want to create a rule. You can choose from the drop-down list that displays all the layer-purpose pairs from the design.

Spacing

Specifies the minimum allowed distance between the devices

Add

Adds a new rule to the list

Delete Selected

Deletes the selected rules from the list

Clear

Removes all the rules from the list

Stripe shapes

This section provides options to create rules to convert layers into stripes of the given width. This helps in converting fine mesh into a coarse mesh that would contain less number of unknowns to be passed to the solver.

List of rules

Displays all the rules created in this section

Spacing

Specifies the spacing to kept between each stripe

Width

Specifies the width for stripes

Lpp

Specifies the layer-purpose pairs for which you want to create a rule. You can choose from the drop-down list that displays all the layer-purpose pairs from the design.

Add

Adds a new rule to the list

Delete Selected

Deletes the selected rules from the list

Clear

Removes all the rules from the list

Smoothing shape steps

This section provides options to create rules to smoothen out small steps or edges after the layers are striped. This also helps in reducing the number of unknowns in the layout.

List of rules

Displays all the rules created in this section

Lpp

Specifies the layer-purpose pairs for which you want to create a rule. You can choose from the drop-down list that displays all the layer-purpose pairs from the design.

Step size

Specifies the minimum allowed step size. All the steps with size less than the given limit are smoothed.

Add

Adds a new rule to the list

Delete Selected

Deletes the selected rules from the list

Clear

Removes all the rules from the list
