Optimize Analysis (optimize)

Description

The optimize analysis automatically generates model parameters and component   values from a set of electrical specifications or measured data.

Syntax

Name optimize parameter=value ...

Parameters

Optimize Parameters

title

Analysis title.

annotate

title

Degree of annotation. Possible values are no, title, sweep and status.

Passfail parameters

Example

                             
Example:
                                       .tran 1n 3n sweep optimize=optimize1 result=max_i model=optimizemod
