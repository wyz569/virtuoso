Setting for Simulink-MATLAB co-simulation (cosim)

Description

Setting for Simulink-MATLAB co-simulation.

Syntax

Name cosim parameter=value ...

Parameters

server

MATLAB/Simulink server name.

port

38520

Co-simulink listen port.

timeout

60 s

Socket timeout in seconds. Default is 60s.

inputs

[...]

Array of input names.

outputs

[...]

Array of output node names.

design

MATLAB/Simulink design name. If the design name is specified, MATLAB is launched automatically.

silent

yes

Launch MATLAB with parameter -nodesktop. Possible values are no and yes.

ratio

0

Hold time ratio between two sample points. Default is 0.
