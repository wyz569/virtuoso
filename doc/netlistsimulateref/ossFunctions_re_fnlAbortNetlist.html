fnlAbortNetlist

fnlAbortNetlist(
)
=> nil

Description

Aborts netlisting. When the formatter detects an error during netlisting, it calls this function to inform the netlister to abort netlisting.

Arguments

None

Value Returned

nil

Always returns nil.

Examples

The following example aborts netlisting. 

fnlAbortNetlist()
=> t
