Euclidean and Manhattan Spacing Constraints

By default, spacing constraints apply within the Euclidean distance around a shape; the area in which the constraint applies, or the spacing halo, has rounded corners. For some spacing constraints, you can also specify that the distance applied be Manhattan instead; the Manhattan spacing halo, which has squared corners, allows a larger spacing area at the corners, as is required for some process rules or constraints. Throughout this manual, constraints for which you can specify manhattan spacing have an optional oaDistanceMeasureType parameter that can be set.

Value Description

0

Euclidean (rounded corners)

1

Manhattan (squared corners)
