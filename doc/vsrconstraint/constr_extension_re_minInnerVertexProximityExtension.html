minInnerVertexProximityExtension

Specifies the minimum extension for a cut that is near an inner vertex of the metal shape that encloses it.

minInnerVertexProximityExtension

Constraint Type

Layer pair

Value Types

Value

Database Types

Design, Technology

Scope

design, foundry

Category

Extension

Value Type

Value

Specifies in user units the required extension for cuts that are positioned near the inner vertex of an enclosing metal layer.

Required Parameters

width

Specifies that the constraint applies only if the width of the enclosing metal shape is less than or equal to this value.

Type: Value

jogHeight

Specifies that the constraint applies only when the height of the jog that forms the inner vertex is greater than this value.

Type: Value

Optional Parameters

cutClass

Specifies that the constraint applies only to cuts of the specified width and length.

Type: DualValue

withinRange

Specifies that if the cut is within this range of the inner vertex, then it must meet the extension specified by constraint value.

Type: RangeValue

otherExtension

Specifies that the constraint applies only if the extension on the opposite side of the inner vertex is less than this value.

Type: Value
