validRoutingVias

Specifies the list of vias that can be used when routing.

validRoutingVias Quick Reference

Constraint Type

Simple

Value Types

ViaDefArrayValue

Database Types

Design, Technology

Scope

net, route, Term, instTerm, net group, design, foundry

Category

Routing

Value Type

validRoutingVias constraints have a ViaDefArrayValue that represents the list of vias that can be used when routing.

Examples

This example sets the valid routing vias to via1 and via2.

Format Example

Tcl

 
set_constraint -constraint validRoutingVias \
-ViaDefArrayValue {via1 via2}

Virtuoso

 
interconnect(
             (validVias ("via1" "via2"))
            )
