Move Cells Form

Use the Move Cells form to move selected cells from one component type to another or to the No component type folder.

Field Description

Select new component type

Lets you choose which component type should be set for the selected cells. Select the component type from the pull-down list and click OK.

No component type

Moves the selected cells to the No component type folder, which means that the selected cells are not assigned to any component type.
