netTracerConstraintGroup 

layoutXL netTracerConstraintGroup string { "virtuosoDefaultExtractorSetUp" | "customCG" }

Description

Specifies the constraint group that defines the via layer, connected layer, and stop layers to be used for creating traces.

The default is virtuosoDefaultExtractorSetUp, which is also the default constraint group used by the Layout XL connectivity extractor.

GUI Equivalent

Command:

Connectivity – Net Tracer – Add/Remove

Field:

Source (Net Tracer Options Form)

Examples

envGetVal("layoutXL" "netTracerConstraintGroup")
envSetVal("layoutXL" "netTracerConstraintGroup" 'string "virtuosoDefaultExtractorSetUp")
envSetVal("layoutXL" "netTracerConstraintGroup" 'string "customCG1")
