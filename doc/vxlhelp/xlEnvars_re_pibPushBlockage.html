pibPushBlockage

layoutXL pibPushBlockage boolean { t | nil }

Description

Pushes blockages inside their instances.

The default value is nil, in which case blocks are not pushed.

GUI Equivalent

Command

Floorplan – Soft Block Update

Field

Push Into Blocks – Push Blockage(s) (Push Into Blocks)

Examples

envGetVal("layoutXL" "pibPushBlockage")
envSetVal("layoutXL" "pibPushBlockage" 'boolean t)
envSetVal("layoutXL" "pibPushBlockage" 'boolean nil)
