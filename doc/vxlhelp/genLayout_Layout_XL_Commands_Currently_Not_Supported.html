Layout XL Commands Currently Not Supported

Mosaics do not currently support the following Layout XL features:

  • Auto Abut
  • Auto Permute
  • Auto Mirror/Rotate

Although mosaics do not automatically support the operations listed above, you can manually perform the following operations on mosaics:

  • Interactive Device Permutation
  • Interactive Pin Permutation
  • Interactive Mirroring/Rotation

Mosaics do not support abutment at all—neither automatically nor interactively.
