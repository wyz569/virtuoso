initAreaFuncName

layoutXL initAreaFuncName string "area_estimation_method"

Description

Sets the method of area estimation—PR Boundary Based or BBox Based—to be used during layout generation.

The default is "PR Boundary Based".

Arguments

None

GUI Equivalent

Command:

Connectivity – Generate — All From Source

Field:

PR Boundary based (PR Boundary tab)

Command:

Connectivity – Generate — All From Source

Field:

BBox based (PR Boundary tab)

Examples

envGetVal("layoutXL" "initAreaFuncName")
envSetVal("layoutXL" "initAreaFuncName" 'string "BBox Based")
