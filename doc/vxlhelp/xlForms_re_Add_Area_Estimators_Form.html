Add Area Estimators Form

Use the Add Area Estimators form to register an area estimator function for estimating the area enclosed by a PR boundary or a virtual hierarchy block.

Field Description

Function name

Specifies the name of the SKILL function to be used for estimating the area of the PR boundary or a virtual hierarchy block.

Nickname

Specifies a custom name for the SKILL function. After the function is registered, the name specified in this field is used to list the SKILL function in the area estimator drop-down list.

Type

Specifies whether the area estimator function being registered is used to estimate the area enclosed by a PR boundary or a virtual hierarchy block. Choose a value from the drop-down list:

  • PR boundary
  • Area boundary

Parameters

Specifies a SKILL list of name-value pairs, where the name specifies the parameter name and the value is the parameter value to be used when estimating the area.

Run mode

Specifies whether the area estimation function is applied immediately or after all the related information is available.

  • deferred: Applies the area estimation function in delayed mode after all the other related information is available. This ensures the estimated area is more accurate as the area estimation calculation considers the related area values to derive the final value.
