Add Soft Pin Form

Use the Add Soft Pin form to create additional soft block pins to improve the connectivity of a design.

Field

Description

Soft block list

Lists all the soft blocks that were selected when you opened the form. You can create soft pins for any of the blocks that are listed in this field. The Lib, Cell, and View fields confirm the currently selected block.

Use

Allows you to select either All Pins or Pins in Selected Set.

Name

Specifies a name for the pin you want to create.

Filter

Lets you filter the terminal names that are displayed by typing in the text box and filtering either All Pins or Pins in Selected Set. The terminal list is updated to display only selected set of pins.

Terminal name

Specifies the name of the terminal on which you want to create a soft pin. The pins are created as strong pins by default and are listed in the table under the Terminal name list. Choose the terminal you want from the drop-down list, which by default, contains all the terminals in the selected block.

Name

Specifies a name for the pin you want to create.

Layer

Specifies the layer purpose on which the pin is to be created. Choose the layer purpose you want from the list of extractable layers in the design.

Width and Height

Specifies the width and height of the new pin. These fields are prepopulated with the minWidth spacing defined in the technology file for the selected pin layer.

Side

Specifies the sides of the PR boundary on which the pins can be created. If you specify more than one side, the Pin Optimizer can place the pins on any of the specified sides.

Create Strong Pin

Lets you specify the parameters using the controls to create a pin for the selected terminal.

Update

Lets you update currently selected pins with the new parameter values.

Delete

Removes the selected pins from the list.
