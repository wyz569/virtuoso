lxcuCreateHTMLReport

layoutXL lxcuCreateHTMLReport  boolean { t | nil }

Description

Specifies whether to create an HTML report.

The default is t.

GUI Equivalent

Command:

Connectivity – Check and Update

Field:

Create HTML Report

Examples

envGetVal("layoutXL" "lxcuCreateHTMLReport")
envSetVal("layoutXL" "lxcuCreateHTMLReport" 'boolean t)
envSetVal("layoutXL" "lxcuCreateHTMLReport" 'boolean nil)
