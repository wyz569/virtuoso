autoZoomFactor

ab autoZoomFactor int integer

Description

Controls the Auto Zoom scale of the Annotation Browser, which determines the level of zoom applied on the selected markers on a scale of 1 to 10.

  • When Auto Zoom is set to Fixed mode, the autoZoomFactor environment variable fully controls the zoom level applied on the canvas.
  • When Auto Zoom is set to Minimal mode, the Annotation Browser limits the change in zoom level on the canvas if the selected marker is visible on panning. Else, the zoom level is modified as in Fixed mode.

The default is 4.

For more information on the Annotation browser Auto Zoom, see the section on the toolbar in Annotation Browser Graphical User Interface.

GUI Equivalent

Command:

Windows – Assistants – Annotation Browser

Field:

Auto Zoom (Toolbar button)

Examples

envGetVal("ab" "autoZoomFactor")
envSetVal("ab" "autoZoomFactor" 'int 2)
envSetVal("ab" "autoZoomFactor" 'int 10)
