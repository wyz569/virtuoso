Checking Process Rules Using Batch Checker

Process rule checking offers a set of options that allow you to specify the portions of the design or the types of rules to verify.

The Process Rules check is selected by default when the Batch Checker form is opened.

To perform process rule checking:

  • Click the DRD Options button in the Check Process Rules group box.

The DRD Options form opens showing the Batch page, in which you can set the various Batch mode options. For information, see Batch Check Mode in the Virtuoso Design Rule Driven Editing User Guide.
