netTracerHaloFadeoutShapes

layoutXL netTracerHaloFadeoutShapes boolean { t | nil }

Description

Controls whether a faded out halo is created around shapes when displaying net traces.

The default is t.

GUI Equivalent

Command:

Options – Net Tracer

Field:

Fadeout shapes (Net Tracer Options Form)

Examples

envGetVal("layoutXL" "netTracerHaloFadeoutShapes")
envSetVal("layoutXL" "netTracerHaloFadeoutShapes" 'boolean t)
envSetVal("layoutXL" "netTracerHaloFadeoutShapes" 'boolean nil)
