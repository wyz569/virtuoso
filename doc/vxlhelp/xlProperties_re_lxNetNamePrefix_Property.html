lxNetNamePrefix Property

lxNetNamePrefix string t_prefix

Description

Adds a unique prefix to internal nets generated in the layout using Layout XL conventions.

lxNetNamePrefix can be used only with devices which have the lxCombination property or sfactor parameter.

Example

Using the lxCombination Property property to generate layout devices from the R device in the schematic below generates three internal nets with the following names.

  • |R|net.ms1
  • |R|net.ms2
  • |R|net.ms3

Using the lxNetNamePrefix property, you can add the prefix X to the internal net names. The names change to

  • |R|Xnet.ms1
  • |R|Xnet.ms2
  • |R|Xnet.ms3
