Saving All Visible Traces

You can save all the traces that are visible in a design to a cellview. You can mark traces as visible by using the Trace Manager form.

To save all the visible traces in your design to a cellview:

  1. In the layout window, choose Connectivity – Net Tracer – Save All Visible Traces.
    The Save Traced Nets View form opens.
  2. Specify the name of the library, cell, and view in which you want to save the visible traces.
  3. Optionally, select the method to be used for saving all visible traces.
  4. Click OK or Apply.
    The visible traces are saved to the specified cellview and can be loaded in another cellview.
