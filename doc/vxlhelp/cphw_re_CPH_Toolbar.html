CPH Toolbar

Lets you access certain Configure Physical Hierarchy commands directly without opening any menus. As with all toolbars, you can use the handle to reposition it anywhere within the Configure Physical Hierarchy window. r

Icon

Description

File – New

Opens the New Physical Configuration form.

File – Open

Opens the Open Physical Configuration form.

File – Open Schematic

Opens the schematic window.

File – Save

Saves changes to the physical configuration of the design.

Choose Mode

Opens the Configure Physical Hierarchy window in the selected mode: Hierarchy Configuration, Component Types, or Soft Block.

Toggle Occurrence Mode

Switches between Instance and Occurrence modes in CPH.

Set Stop Level

Sets a global stop level at which to stop the elaboration of the logical design hierarchy. The stop level specified here overrides the value specified using the cphStopLevel environment variable.
