netTracerStopLevel

layoutXL netTracerStopLevel int integer

Description

Sets the hierarchy level at which tracing is stopped for a design.

The default is 0.

This environment variable takes effect only when netTracerRange is set to “User defined”.

GUI Equivalent

Command:

Connectivity – Net Tracer – Add/Remove

Field:

Hierarchy Range – User Defined – Stop

(Net Tracer Options Form)

Examples

envGetVal("layoutXL" "netTracerStopLevel")
envSetVal("layoutXL" "netTracerStopLevel" 'int 20)
envSetVal("layoutXL" "netTracerStopLevel" 'int 3)
