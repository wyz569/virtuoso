Connection Requiring an Extra Layer

A stdViaDef is used to define a connectivity model in which an extra layer is required to connect to the diffusion layer.

("MD"   "OD"   ("MD") "NP" "RECOGNITION" "NW" )

(layer1 layer2   cut implant1 implant2 well/substrate)

The extra layer is considered only if it is a recognition layer for a connection between diffusion and well or substrate. In the syntax, the slots implant1 and implant2 are interchangeable.
