Array Generation

Layout XL lets you generate selected or unplaced schematic instances as an XL-compliant array in the layout view. You can create an array of instances with the same or different master, parameters, or connectivity.

You can choose to create an array of ungrouped instances, a modgen, a mosaic, or a group array.

During array generation, preselected objects are automatically squared in the array. To square postselected objects, use the button.

Array placement mode cannot be used to place pins.
