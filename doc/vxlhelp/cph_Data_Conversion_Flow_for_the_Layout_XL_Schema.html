Data Conversion Flow for the Layout XL Schema

The figure below shows how legacy data is converted to use the Layout XL schema.

Data is converted in two separate operations.

  • A schematic library conversion (indicated by the top three arrows in the picture above).
    You can run it manually using the utility described in Converting Schematic Libraries and Designs.
  • A layout design conversion (indicated by the bottom two arrows in the picture above).
    This runs automatically when you load your design into Layout XL, or you can run it manually using the cphUprevDesign SKILL function.
