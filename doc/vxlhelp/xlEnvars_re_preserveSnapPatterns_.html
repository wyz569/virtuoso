preserveSnapPatterns 

layoutXL preserveSnapPatterns boolean { t | nil }

Description

Specifies that any existing snap patterns and width spacing patterns are preserved when you run the Generate All From Source command.

Width spacing pattern constraints are always preserved.

The default is nil.

For more information on snap pattern shapes, see Snapping Objects to Local Snap Pattern Shapes in the Virtuoso Layout Suite XL: Basic Editing User Guide. For more information on width spacing patterns, see Using Width Spacing Patterns in Virtuoso Width Spacing Patterns User Guide.

GUI Equivalent

Command:

Connectivity – Generate – All From Source (Floorplan tab)

Field:

Preserve Floorplanning Objects – Snap Patterns

(Generate Layout Form)

Examples

envGetVal("layoutXL" "preserveSnapPatterns")
envSetVal("layoutXL" "preserveSnapPatterns" 'boolean t)
envSetVal("layoutXL" "preserveSnapPatterns" 'boolean nil)
