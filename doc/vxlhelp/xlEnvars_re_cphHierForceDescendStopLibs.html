cphHierForceDescendStopLibs

layoutXL cphHierForceDescendStopLibs string "lib1 lib2"

Description

Determines where the hierarchical force descend will stop when the Physical stop library list is added.

For maximum flexibility, hierarchal force descend can be added at the top level of any node.

GUI Equivalent

None

Examples

envGetVal("layoutXL" "cphHierForceDescendStopLibs")
envSetVal("layoutXL" "cphHierForceDescendStopLibs" 'string "lib1 lib2")
