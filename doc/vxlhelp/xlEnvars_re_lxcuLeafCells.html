lxcuLeafCells

layoutXL lxcuLeafCells string "name_of_cells"

Description

Lists the cells to be treated as leaf cells. Only the terminals of leaf cells are checked when the All cells in library or All cells in library option is selected.

The default is "".

GUI Equivalent

Command:

Connectivity – Check and Update

Field:

Leaf Cells

Examples

envGetVal("layoutXL" "lxcuLeafCells")
envSetVal("layoutXL" "lxcuLeafCells" 'string "(STDLIB1 \\* \\*)" "(STDLIB2 NAND\\* \\*)")
