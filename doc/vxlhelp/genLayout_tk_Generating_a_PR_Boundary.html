Generating a PR Boundary

When you initialize a layout, Layout XL automatically generates a PR boundary for the design.

To set the default size for the PR boundary, from the layout window menu bar:

  1. Choose Connectivity – Generate – All From Source.
    The Generate Layout form is displayed.
  2. On the Generate tab, select the PR Boundary option.
  3. On the PR Boundary tab, set the options to generate the shape and size of boundary you need.
  4. When you click OK to generate the layout, the system draws the specified boundary in the layout canvas.

To create a design boundary manually:

  • Use the Create – P&R Objects – P&R Boundary command.

To stretch, move, or delete the PR boundary.

  • Use the layout editor Edit commands.
