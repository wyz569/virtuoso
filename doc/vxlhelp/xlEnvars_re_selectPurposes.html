selectPurposes

layoutXL.wspCheck selectPurposes string "purpose_names"

Description

Specifies the layer for the shapes to be checked.

purpose_names specifies one or more purpose names.

The default is drawing.

GUI Equivalent

Command

Verify – Design

Field

Purposes ( WSP | SP Active )

Examples

envGetVal("layoutXL.wspCheck" "selectPurposes")
envSetVal("layoutXL.wspCheck" "selectPurposes" 'string "drawing")
