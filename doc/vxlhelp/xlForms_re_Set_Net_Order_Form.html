Set Net Order Form

Use the Set Net Order form to create an ordered net constraint on a group of nets. An ordered net constraint is honored by the pin optimizer engine, which optimizes the nets while maintaining the order specified by the constraint.

Field Description

Net Order Name

Specifies a name for the ordered net constraint. If you do not specify a name, the system generates one for you.

Selected Nets

Lists the nets that are currently selected in the layout canvas.

Nets Name

Finds nets in the list of selected nets. Type in a string and click Filter to see only the nets you specified.

Ordered Net Members

Lists the names of the nets in the constraint.

Add

Adds one or more selected nets to the constraint.

Remove

Moves one or more selected nets out of the constraint.
