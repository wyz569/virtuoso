selectLayer

layoutXL.wspCheck selectLayer string "layer_name"

Description

Specifies the layer for the shapes to be checked.

  • layer_name specifies the layer name.
  • All specifies all the layers in the active WSP.

The default is All.

GUI Equivalent

Command

Verify – Design

Field

Layer ( WSP | SP Active )

Examples

envGetVal("layoutXL.wspCheck" "selectLayer")
envSetVal("layoutXL.wspCheck" "selectLayer" 'string "All")
envSetVal("layoutXL.wspCheck" "selectLayer" 'string "Metal1 Metal2")
