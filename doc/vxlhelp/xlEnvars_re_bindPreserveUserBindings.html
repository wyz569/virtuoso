bindPreserveUserBindings

layoutXL bindPreserveUserBindings boolean { t | nil }

Description

Preserves user bindings.

The default is t.

GUI Equivalent

Command:

Connectivity – Update – Binding

Field:

User-defined bindings (Update Binding Form)

Examples

envGetVal("layoutXL" "bindPreserveUserBindings")
envSetVal("layoutXL" "bindPreserveUserBindings" 'boolean nil)
