extractToLevel Property

extractToLevel int { 0 | 1 | 2 | ... | 32 }

Description

Overrides the extractStopLevel environment variable for an individual instance or instance master to increase the effective extraction stop level.

If the value set for this property is greater than that specified by the extractStopLevel environment variable, the effective extraction stop level is increased for the instances in question. For example, if the property value is set to 1 on a particular instance master, the shapes at hierarchical depth 1 will be seen for instances of this master, even if the extractStopLevel environment variable is set to 0. This is the case for Pcells also, even if the Extract through pcells option on the Connectivity tab of the Connectivity form is disabled.

You set the extractToLevel property in the Add Property form.

Layout instance Add Property form

Name

extractToLevel

Type

Int

Value

Integer value between 0 and 32 inclusive
