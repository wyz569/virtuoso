Design Summary Form

Use the Design Summary form to specify the types of statistics you want to view in the design summary report of the current cellview.

Field Description

Select All

Selects all the options in the Design Summary form. When all options are selected, the design summary report displays the basic summary for the current cellview and the statistics based on nets, signal types, and layers.

Basic Summary

Displays the basic summary for the current cellview, which includes details such as statistics related to layer objects, routing grids, and instances.

Net Statistics

Displays in the design summary the number of terminals, instance terminals, guides, and steiners for each net in the current cellview. For each constituent figure of a net, the report lists the layers, wire length, horizontal and vertical wire length, vias, routes, and wire segments.

Connectivity Statistics

Displays in the design summary the connectivity statistics for the current cellview based on signal types, such as signal, power, ground, and clock.

Routing Statistics

Displays in the design summary the routing statistics for the current cellview based on layers.
