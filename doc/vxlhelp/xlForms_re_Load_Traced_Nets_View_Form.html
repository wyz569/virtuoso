Load Traced Nets View Form

Use the Load Traced Nets View form to load the previously saved traces from a design cellview to your current layout cellview. The Load Traced Nets View form is available through the Connectivity – Net Tracer – Load Physical Traces command.

Field Description

Library

Displays the name of the design library from which the saved traces will be loaded.

Cell

Displays the name of the cell from which the saved traces will be loaded.

View

Displays the view name from which the saved traces will be loaded. The loaded traces are added along the already displayed traces in the current cellview.

Replace Traced Nets

Replaces the already displaying traces in the current layout cellview with the traces from the saved cellview.
