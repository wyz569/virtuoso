extractShortLocatorOtherShapesColor

layoutXL extractShortLocatorOtherShapesColor string "color"

Description

Specifies the color of the halo around the shapes that are neither common shapes nor shapes of the two shorted nets.

The default is "white".

GUI Equivalents

Command:

Connectivity – Short Locator – Preferences (tab)

Field:

Colors – Other shapes on net (Short Locator)

Examples

envGetVal("layoutXL" "extractShortLocatorOtherShapesColor")
envSetVal("layoutXL" "extractShortLocatorOtherShapesColor" 'string "red")
