checkPermute

layoutXL checkPermute boolean { t | nil }

Description

Controls whether the lxCheck SKILL function (or the Connectivity – Check – XL Compliance command) should issue information messages to report cells that may be missing permute rules.

The default is t.

GUI Equivalent

None

Examples

envGetVal("layoutXL" "checkPermute")
envSetVal("layoutXL" "checkPermute" 'boolean t)
envSetVal("layoutXL" "checkPermute" 'boolean nil)
