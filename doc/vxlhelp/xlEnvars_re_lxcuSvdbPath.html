lxcuSvdbPath

layoutXL lxcuSvdbPath string "path_to_the_directory"

Description

Specifies the path to the PVS svdb directory containing:

  • xyz.ixf instance cross-reference file.
  • xyz.net extracted layout CDL netlist file.

The default is "".

GUI Equivalent

Command:

Connectivity – Check and Update

Field:

LVS svdb path

Examples

envGetVal("layoutXL" "lxcuSvdbPath")
envSetVal("layoutXL" "lxcuSvdbPath" 'string "./svdb")
