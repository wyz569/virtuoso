Update Layout Device List Form

The Update Layout Device List form displays all the devices in the design. Use it to select specific devices when using the Update Layout Parameters and Update Schematic Parameters commands.

Field Description

Search

This section lets you find and select the names of devices you specify. Specify a device name, part of a device name, or a regular SKILL expression using part of a device name in the field to the right of the Search button.

A device name

Highlights the device and scrolls the list to make the device name visible.

Part of a device name (for example, IN)

Highlights all the devices with IN as an element in the name (IN3, IN6, and so on) and scrolls the list to make the first example visible.

A regular SKILL expression using part of a device name (for example, *mos)

Highlights all devices with the string *mos (for example, nmos, pmos) and scrolls the list to make the first example visible.
