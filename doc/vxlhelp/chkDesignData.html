Design Data Checks

This topic summarizes how to use the Virtuoso® Layout Suite XL (Layout XL) to check your design as you work.

  • Selecting and Deselecting Nets in Navigator - Select a net in the Navigator assistant and all the shapes associated with that net in the design.
  • Design Probing - Select an instance, net, or pin in the layout or schematic window to highlight the corresponding element in the other window.
  • Tracing Nets - Trace the physical and logical connectivity of nets in a design.
  • Checking Shorts, Incomplete Nets, and Illegal Overlaps - Check the number of incomplete nets, shorts, and illegal overlaps in your design using Annotation Browser.
  • Design Check Against Source - Check mismatches between the schematic and layout.
  • Design Check for Layout XL Compliance - Check if your design fulfills the compatibility criteria that allow it to fully leverage the connectivity-driven features of Virtuoso Layout Suite XL
  • Design Check for Manufacturability - Check manufacturability of a design for optimum device yield and performance.
  • Batch Checker - Use the Batch Checker tool to check for routability, process rules, connectivity, specialty routing, placement, fabrication, and WSP | SP Active.
