Finding Violations Using Annotation Browser

The Annotation Browser assistant is used for viewing and managing violation markers in your design.

To view and manage violation markers:

  1. Choose Window – Assistants – Annotation Browser or click the Annotation Browser icon.
  2. Click the tab in the Annotation Browser assistant for the markers that you want to view.
