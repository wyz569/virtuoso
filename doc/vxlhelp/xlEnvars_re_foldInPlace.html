foldInPlace

layoutXL foldInPlace boolean { t | nil }

Description

Creates device folds at the same location on the layout canvas as the original device.

The default is nil.

GUI Equivalent

Command:

Connectivity – Generate – Folded Devices

Field:

Create In Place

Examples

envGetVal("layoutXL" "foldInPlace")
envSetVal("layoutXL" "foldInPlace" 'boolean nil)
