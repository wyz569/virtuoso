Create Group Array Form

Use the Create Group Array form to create a group array with the specified parameters.

Field Description

Include Clone Source

Includes the clone source in the group array.

The clone source is the lower-left cell of the group array. The clone targets the other cells of the group array.

If the clone source is already a synchronous clone or contains a synchronous clone, it cannot be included in the group array. In this case, the Include Clone Source option is disabled.

Number of Clones in the array

Specifies the number clones that are included in the group array.

Mode

Controls the type of spacing for the group array.

  • Pitch: Lets you specify the distance that is measured between the same-edge corners of cells in the Delta X,Y fields. For example, the distance between the left edges of two cells.
  • Spacing: Lets you specify the distance that is measured between the adjacent or closest edges of cells in the Spacing X,Y fields. For example, the distance between the top edge of a cell and the lower edge of another cell.

Spacing X,Y

Specifies the spacing in the Spacing X,Y fields

Reference LPP

Specifies the layer-purpose pair used for computing the bounding box of the cells in a group array. The combined bounding box of the geometries on the specified reference layer-purpose pair is used for calculating the spacing between cells.

Array Pattern

Displays the Array Pattern form to let you specify the orientation pattern for the group array.
