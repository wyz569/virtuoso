hardBlockColor

layoutXL hardBlockColor string "hilite_layer hilite_purpose"

Description

Specifies the highlight color for hard blocks in the layout canvas.

The default is "hilite drawing".

Arguments

hilite_layer hilite_purpose

A list of parameter names separated by a space. The list must be enclosed in quotation marks; for example, "hilite_layer hilite_drawing".

GUI Equivalent

Command:

Floorplan – Floorplan Global Options

Field:

Hard block color

Examples

envGetVal("layoutXL" "hardBlockColor")
envSetVal("layoutXL" "hardBlockColor" 'string "hilite drawing")
