pibPinPurposeName

layoutXL pibPinPurposeName string "Purpose"

Description

Specifies the purpose of the layer on which pins are to be created when the top-level power structures are pushed to the block level. This environment variable takes effect when pibCreatePinLPPChoice is set to Custom Purpose.

The default value is drawing.

GUI Equivalent

Command

Floorplan – Soft Block Update

Field

Push Into Blocks – Create Pin Layer Selection – Custom Purpose (Push Into Blocks)

Examples

envGetVal("layoutXL" "pibPinPurposeName")
envSetVal("layoutXL" "pibPinPurposeName" 'string "drawing")
