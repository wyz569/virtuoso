Net Tracer Neighbors Form

Use the Net Tracer Neighbors form to specify the settings that the Net Tracer uses for finding the neighboring shapes of a traced shape.

The form is available through the Connectivity – Net Tracer – Neighbors command.

Field Description

Neighbors

Specifies the layer on which the Net Tracer finds the neighboring shapes of a traced shape.

  • On same layer as traced shape: Finds neighboring shapes on the same layer as the traced shape.
  • On any extractable layer: Finds neighboring shapes on any extractable layer.

Neighboring shapes need not exist on the same purpose as the traced shape.

Environment variable: netTracerNeighborsOnLayer

Distance

Specifies the distance from a traced shape within which the Net Tracer finds the neighboring shapes of a traced shape.

Compute net name

Layout EXL or a higher tier, enables net name computation for the neighboring shapes of a traced shape.

Environment variable: netTracerComputeNeighborNet

Compute From

Layout EXL or a higher tier, specifies the method used for computing net names for the neighboring shapes of a traced shape.

  • Logical: If a neighboring shape already has a net name, the same net name is used. In a hierarchy, the top-most net is used as the net name for a neighboring shape.
    If a neighboring shape does not already have a net name, no net name is assigned to the shape.
  • Logical and Physical: If a neighboring shape already has a net name assigned, the same net name is used. Otherwise, the neighboring shape is physically traced until a shape with a net name is found, and that net name is used for the neighboring shape.

Environment variable: netTracerComputeNeighborNetMethod
