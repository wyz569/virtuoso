Net Tracer Toolbar

Use the Net Tracer toolbar buttons to launch the Net Tracer, to manage the generated traces, and to control how traces are displayed in the layout canvas.

Icon Name Description

Add/Remove trace

Adds or removes a physical, logical, or step trace.

Press F3 after clicking the toolbar button to launch the Net Tracer Options Form.

Delete all traces

Deletes all traces in the design.

See Removing All Traces.

Show Traces

Opens the Trace Manager Form.

Zoom to fit visible traces

Zooms to display all the visible traces in the layout canvas.

Auto zoom to selected traces

Automatically zooms into traces that are selected in the Trace Manager.

Net Tracer EIP

Edits in place the instances of the traced figure.

Press F3 after clicking the toolbar button to launch the Net Tracer Edit In Place dialog box.

Display Depth

Sets the hierarchy depth up to which the corresponding shapes for a traced net are displayed.

By default, Display Depth is in sync with the Display Levels – Stop value in the Display Options form.

  • When the Display Depth toolbar button is clicked and a value specified in the associated text field, the specified value is used. The Display Depth drop-down in the Net Tracer Display Options form also reflects the same value.
  • When the Display Depth toolbar button is not clicked, the value set in the Display Levels – Stop field in the Display Options form is used. The Display Depth drop-down in the Net Tracer Display Options form is set to Same as stop display.

See Net Tracer Display Options Form.

Display Mode

Lets you choose if a trace is displayed:

  • in true color and with a halo
  • only in true color
  • only with a halo

See Net Tracer Display Options Form.

Display halo on visible shapes only

When selected, displays a halo only around visible shapes.

When deselected, which is the default setting, displays a halo around all the shapes on a trace, even if the shapes are not visible.

See Net Tracer Display Options Form.

Pins

When selected, pins have the same halo thickness as the shapes.

When deselected, which is the default setting, creates a wider halo around the pin shapes, making it easier to locate the pins in the canvas.

See Net Tracer Display Options Form.

Current Step

Displays the current step value of the trace.

Tail Value

Displays the value of the tail of a trace.
