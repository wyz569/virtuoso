placeCloneAsArray

layoutXL placeCloneAsArray boolean { t | nil }

Description

Places the clones selected from the Clones Found list in the Generate Clones form as a row, column, or as an array.

The default is nil, which means the selected clones are placed one-at-a time.

See Clone Placement as an Array.

GUI Equivalent

Command:

Connectivity – Generate Clones

Field:

Place Clone as Array (Generate Clones Form)

Examples

envGetVal("layoutXL" "placeCloneAsArray")
envSetVal("layoutXL" "placeCloneAsArray" 'boolean t)
envSetVal("layoutXL" "placeCloneAsArray" 'boolean nil)
