abutAddUniqueDummyNet

layoutXL abutAddUniqueDummyNet boolean { t | nil }

Description

Enables chaining and interactive abutment of unconnected dummy devices—devices that have no source or drain instance terminals. Parameter updates performed on abutted Pcell instances are preserved.

The default is t.

When set to nil, unconnected dummy devices cannot be chained or interactively abutted because no automatic net assignment is supported.

GUI Equivalent

None

Examples

envGetVal("layoutXL" "abutAddUniqueDummyNet")
envSetVal("layoutXL" "abutAddUniqueDummyNet" 'boolean nil)
