Trace Manager Form

Use the Trace Manager form to control the display of traces for various nets. Also use the form to change the trace color for a net, or to delete a trace.

Field Description

Search net name

Allows you to search for a net name or choose a net from the associated drop-down list to create a trace for the net.

Trace

Creates a trace for the selected net and displays the traced net and the associated traced shapes in a hierarchical structure in the Trace Manager window.

If you type a net name that does not exist in the cellview and click the Trace button, no trace is created.

Net

Displays the name of the traced net and all the corresponding island trace IDs in a tree structure. Selecting a net in the Trace Manager halos the net in the layout canvas even when the trace is set as not visible in the Trace Manager using the Visibility icon.

  • The Net Tracer creates one physical trace per island, whether or not the From net option on the Net Tracer Options Form is selected.
  • If the Auto zoom to selected traces button is clicked in the Net Tracer Toolbar, the Net Tracer automatically zooms to the selected trace.

Visibility icon

Controls the display of traces in a design. Click the icon in the column header to display (or hide) all the traces in one go. Alternatively, select the check box corresponding to each net to display (or hide) the trace for the net.

Color

Lets you choose a trace color for the selected net.

Type

Displays the icon specific to the type of trace created:

  • Physical –
  • Logical –
  • Step –

Remove all traces

Lets you delete all or specific net traces. Click the icon in the column header to delete all traces. Alternatively, click the delete icon associated with specific nets to delete the corresponding traces.

Expand All

Expands the hierarchy under the traced nets to display the names of all the shapes on each traced net.

Collapse All

Collapses the hierarchy under the traced nets to display only the traced net names.
