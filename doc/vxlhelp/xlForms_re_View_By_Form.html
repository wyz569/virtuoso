View By Form

Use the View By form to specify how the annotations are organized in categories and how the categories are sorted in the browser pane.

To open the form, right-click in the browser pane and choose View By.

Alternatively, you can open the View By form by right-clicking in the Annotation Browser column header and choosing View By – Advanced. For more information, see Annotation Browser Shortcut Menus.

Field

Description

View items by

Specifies a category by which to view items.

  • Ascending: Sorts the items by the lowest number or in alphabetical order
  • Descending: Sorts the items by the highest number or reverse alphabetical order.

Then by

Specify category by which to view items for second, third, and fourth-level groupings.

Defaults

Restores the default grouping for the tab (for example, the default grouping for the Connectivity tab is Type/Net)

Apply

Applies changes to the markers currently displayed in the Annotation Browser.

OK

Applies the changes and to closes the View By form.
