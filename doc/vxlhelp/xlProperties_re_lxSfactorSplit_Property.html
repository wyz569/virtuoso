lxSfactorSplit Property

Description

This property is automatically converted to the new Layout XL schema underlying physical hierarchy configuration.

To control whether Layout XL places a schematic device with the sfactor Property property as series-connected devices in the layout,

  1. From the layout window menu bar, choose Launch – Configure Physical Hierarchy.
  2. In the Instances table, select the logical instance for which you want to generate series-connected devices.
  3. In the Generation tab, switch on the Split sfactored devices option.
    This overrides the global setting on the Generation of the Connectivity form.
    • Split sfactored devices is grayed out if the selected instances do not have the sfactor property.
    • The shell environment variable "CPH_USE_SCHEM_PROP=1" must be set, if CPH must read the lxSfactorSplit Property property. Else, the property is upreved to CPH and driven from there.
