enableCloneColoringSupport

layoutXL enableCloneColoringSupport boolean { t | nil }

Description

Enables the color state and the color of a “color locked” shape in a synchronous clone to be carried to the corresponding shapes in other synchronous clones.

The default is t.

  • When set to t, and Editing In Place a clone, the color state and the color of the “color-locked” shapes inside the clone are automatically synchronized.
    • Color editing of shapes inside a synchronous clone by using the Multiple Patterning Toolbar is not supported.
  • When set to nil, the color state and color of shapes inside the clone are not synchronized.
    • The Multiple Patterning Toolbar can be used to color edit the shapes in a synchronous clone.

GUI Equivalent

None

Examples

envGetVal("layoutXL" "enableCloneColoringSupport")
envSetVal("layoutXL" "enableCloneColoringSupport" 'boolean t)
envSetVal("layoutXL" "enableCloneColoringSupport" 'boolean nil)
