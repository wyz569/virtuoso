netTracerHaloTransparency

layoutXL netTracerHaloTransparency int integer

Description

Controls the extent of transparency for a halo. This environment variable takes effect only when the netTracerHaloFadeoutShapes environment variable is enabled.

The default is 50.

GUI Equivalent

Command:

Options – Net Tracer

Field:

Transparency (Net Tracer Options Form)

Examples

envGetVal("layoutXL" "netTracerHaloTransparency")
envSetVal("layoutXL" "netTracerHaloTransparency" 'int 22)
envSetVal("layoutXL" "netTracerHaloTransparency" 'int 100)
