Updating VPLGen Pcell Instances

VPLGens support all the standard Layout XL update commands, meaning that changes in the schematic view can be propagated to the layout view.

For... Use...

Changes to a parameter or property value in a schematic instance.

For example, after the layout is generated, if you want to change any original parameter values, such as change the length parameter from l = 1u to l = 2u, the parameter updates must be made in the schematic view and an Update Layout Parameters command run to reflect the change in the layout view.

Update Layout Parameters to propagate the changes to the VPLGen Pcell instances in the layout view.

There is no need to use Update Components And Nets because only parameter or property values have been updated.

New or copied instances added to the schematic view.

Generate Selected From Source to add the new VPLGen Pcell instance to your layout, while re-using any VPLGen default layout already defined.

Changes you make in the layout view cannot be propagated back to the schematic view.
