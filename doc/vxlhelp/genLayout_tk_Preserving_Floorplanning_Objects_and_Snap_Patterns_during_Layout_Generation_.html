Preserving Floorplanning Objects and Snap Patterns during Layout Generation

Use the controls on the Floorplan tab on the Generate Layout form to preserve existing floorplanning objects during layout generation.

  • On the Floorplan tab, check the objects you want to preserve.
    Click All to select all the items in the list; None to deselect them all.

You can also preserve any Snap Patterns and Width Space Pattern (WSP) regions already defined in the layout view. The following components and definitions related to a WSP data model are also preserved in the cellview:

  • widthSpacingPatterns
  • widthSpacingSnapPatternDefs
  • widthSpacingPatternGroups
  • relatedSnapPatternDefs
  • constraints for the global grid setup for snapPatternDefs and widthSpacingSnapPatternDefs
  • the global grid offset reference for snapPatternDefs and widthSpacingSnapPatternDefs

To preserve the predefined patterns:

  • Select Snap Patterns in the Floorplan tab of the Generate Layout form.

Alternatively, you can choose to preserve the predefined patterns by using the preserveSnapPatterns environment variable.
