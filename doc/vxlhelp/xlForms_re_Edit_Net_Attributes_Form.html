Edit Net Attributes Form

Use the Edit Net Attributes form to modify the criticality and signal types of existing nets in the design.

Field Description

Net Name

Shows you which net you are currently editing.

If you select multiple nets, use the Previous and Next buttons to cycle through the selected nets

Common

Specifies that the current settings are to be applied to all the selected nets. When switched off, the settings are applied only to the net in the Net Name field.

Criticality

Sets the criticality of the specified net (or all nets).

The default value is 10. The specified value must be in the range -128 to 128.

Signal Type

Sets the signal type of the specified net. Choose one from signal, supply, ground, clock, tieOff, tieLo, tieHi, analog, scan and reset.
