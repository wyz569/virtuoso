fingerSplit

layoutXL fingerSplit boolean { t | nil }

Description

Controls whether each finger of a schematic device will be placed in the layout as a separate device.

The default is nil.

GUI Equivalents

Command:

Options – Connectivity– Generation

Field:

Split fingered devices (Connectivity Form)

Examples

envGetVal("layoutXL" "fingerSplit")
envSetVal("layoutXL" "fingerSplit" 'boolean t)
envSetVal("layoutXL" "fingerSplit" 'boolean nil)
