backAnnotateXSpacing

layoutXL backAnnotateXSpacing float instanceSpacing

Description

Specifies the horizontal spacing between instances.

The default is 0.625.

GUI Equivalent

Command:

Connectivity – Backannotate – Dummy Instances – F3

Connectivity – Generate – Selected From Layout – F3

Field:

X, Y Spacing

Examples

envGetVal("layoutXL" "backAnnotateXSpacing")
envSetVal("layoutXL" "backAnnotateXSpacing" 'float 0.1)
