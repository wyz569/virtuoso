Incomplete Nets Commands

Command

Description

Show/Hide All

Show and hide all the incomplete nets in your design. You can also access this command using:

  • The Shift+Ctrl+h bindkey on your keyboard.
  • The lxToggleShowAllIncompleteNets SKILL function.

If you run the Show/Hide All command with the Annotation Browser closed, the command fails. You must open the Annotation Browser before you run the command.

To automatically display all the incomplete nets in the design when the layout is launched in Layout XL, set the environment variable showIncompleteNets to t.

Show/Hide Selected

Show and hide the incomplete nets associated with selected objects in your design. You can also access this command using

  • The Show/Hide Selected Incomplete Nets button in the Layout XL toolbar. (This is the method used in this document.)
  • The Shift+h bindkey on your keyboard.
  • The lxShowHideIncompleteNets SKILL function.

When you start the command, the Annotation Browser is automatically initialized if required.

The command does not support blockage, boundary, or row objects. These objects are silently ignored unless a preselected set contains only unsupported objects, in which case you see a message informing you of this fact.

The command behaves differently depending on whether you launch it with objects already preselected or postselected.

See Showing Incomplete Nets in Preselection Mode and Showing Incomplete Nets in Postselection Mode

Environment variable: showIncompleteNets

Show/Hide Current

Change the visibility of the currently visible incomplete nets in the design canvas. You can also access this command using:

  • The h bindkey on your keyboard.
  • The lxToggleShowIncompleteNets SKILL function.

See Showing and Hiding Current Incomplete Nets

Show/Hide Common For Selected Instances

Display incomplete nets that are common between selected instances.

  • Show/Hide Common For Selected Instances button in the Layout XL toolbar.
  • The h bindkey on your keyboard.
  • The lxShowCommonIncompleteNetsForSelectedInsts SKILL function.

See Showing Common Incomplete Nets for Selected Instances
