Save Traced Nets View Form

Use the Save Traced Nets View form to save all the visible traces and their locked mask colors, where available, to a layout cellview. To mark the traces as visible, use the Trace Manager form.

Field Description

Library

Specifies the name of the design library to which the visible traces in the current layout library cellview will be saved.

Cell

Specifies the name of the cell to which the visible traces in the current layout library cellview will be saved.

View

Specifies the name of the view to which all the visible traces in the current layout library cellview will be saved.

Method

Lets you specify the method used for saving traces.

  • One view per trace: Saves each visible trace as a separate view. This helps to optimize the performance in case of a large trace that has lots of figures.
  • One view for all traces: Saves the visible traces into a single view.
  • Auto: Lets the system to automatically determine the fastest method to be used for saving traces.
    • If only one trace is visible, the One view per trace method is used.
    • If at least one visible trace is large, the One view per trace method is used.
    • If all visible traces are small, the Saving one view for all traces method is used.

Environment variable: netTracerSaveMethod
