updateParamCopyFolds

layoutXL updateParamCopyFolds boolean { t | nil }

Description

Specifies whether the Update Layout Parameters and Update Components and Nets commands generate a new fold with same parameter as the existing device.

The default is nil.

GUI Equivalent

None

Examples

envGetVal("layoutXL" "updateParamCopyFolds")
envSetVal("layoutXL" "updateParamCopyFolds" 'boolean t)
