Assign Layout Instance Terminals Form

Use the Assign layout instance terminals form to manually assign layout nets to unbound instance terminals in selected layout instances. This form is also useful when the Define Device Correspondence command is unable to assign all the instance terminals automatically. For example, if the instance masters in a one-to-one correspondence do not match, or if you create a one-to-many, many-to-one, or many-to-many correspondence.

Field Description

Select layout instance terminal to assign

Lists the Instance, Terminal, and the Net name to which the terminal is currently connected.

The instances are listed in non case-sensitive order, first ordered by instance name and then by the terminal name.

Select layout net

Lists global nets and the nets that connect to the instances being bound.

You can select one or more instance terminals and assign them to a layout net. The nets are listed in non case-sensitive order.

Assign instance terminal

Lets you choose the net you want to assign to the selected instance terminals and make the assignment.
