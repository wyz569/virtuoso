generateYSpacing

layoutXL generateYSpacing float floating_point_number

Description

Controls the vertical spacing between the instances generated as an array using the Generate Selected From Source command.

The default is 0.0.

GUI Equivalent

Command:

Connectivity – Generate – Selected From Source

(Generate Selected Components Form)

Field:

Placement Mode – Array

Array Type – Instances

Examples

envGetVal("layoutXL" "generateYSpacing")
envSetVal("layoutXL" "generateYSpacing" 'float 0.3)
