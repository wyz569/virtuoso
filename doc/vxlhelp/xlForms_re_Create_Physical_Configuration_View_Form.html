Create Physical Configuration View Form

Use the Create Physical Configuration View form to create a new physical configuration view to be used in a Layout XL session.

Field Description

Library Name

Specifies the name of the library in which the physical configuration view resides.

Cell Name

Specifies the name of the cell to which the physical configuration view relates.

View Name

Specifies the name of the physical configuration view.

Use Template

Specifies whether a template or file containing predefined values is to be used for the new physical configuration view.

Template

Lets you select a predefined template file from the drop-down list.

From File

Lets you specify a template file stored in another location.

If you do not specify a template or file, Layout XL uses the default template and any overriding values from the corresponding environment variables.

Open CPH

Opens the Configure Physical Hierarchy Window automatically after you click OK in this form.

Environment variable: cphLayoutSync
