electricalPinSetup

layoutXL electricalPinSetup string "constraint_group_name"

Description

Specifies the name of the constraint group to use for determining the layers available for generating the electrical pins.

The default is "virtuosoDefaultElectricalPinSetup".

If the virtuosoDefaultElectricalPinSetup constraint group is not defined, the validLayers defined in the default Layout XL constraint group, virtuosoDefaultSetup, are used for pin generation.

GUI Equivalents

None

Examples

envGetVal("layoutXL" "electricalPinSetup")
envSetVal("layoutXL" "electricalPinSetup" 'string "virtuosoDefaultElectricalPinSetup")
