File Commands

Tcl is a general purpose, popular scripting language that can be used to automate and control Virtuoso Space-based Router. The Tcl core commands are supported by the built-in Tcl interpreter. Tcl commands can be issued interactively via the Virtuoso Space-based Router command line and by using scripts. These commands are available when running Virtuoso® Space-based Router except as noted in the description for the command.

This topic lists the file Tcl commands that are available in Virtuoso Space-based Router.
