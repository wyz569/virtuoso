icopy

icopy
[ -once | -repeat ]

Description

Sets the mouse command field to icopy and allows you to interactively copy routes.

The GUI equivalent for this command is Edit—Copy.

Arguments

-once

Permits only one copy, then the mouse command mode automatically reverts to select mode.

-repeat

Permits consecutive copies, until the mode is canceled or another interactive mode is enabled.
