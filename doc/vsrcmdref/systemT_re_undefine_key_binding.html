undefine_key_binding

undefine_key_binding 
-key s_key

Description

Removes the given key from the bindkey list.

Arguments

-key s_key

Specifies the bindkey to remove.

Examples

This example removes the "CTRL+f" key from the bindkey list.

undefine_key_binding -key "CTRL+f"
