ignore_routing_grid_check_for_via_metal_layers

db.ignore_routing_grid_check_for_via_metal_layers stringlist metallayers

Description

Ignores the routing grid check for metal layers with via shapes, if the corresponding cut shape is enclosed by non-via metal.

Default value is "".

GUI Equivalent

None

Examples

getvar db.ignore_routing_grid_check_for_via_metal_layers
setvar db.ignore_routing_grid_check_for_via_metal_layers { metal1 metal2 }
