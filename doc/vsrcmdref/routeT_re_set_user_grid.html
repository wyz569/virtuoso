set_user_grid

set_user_grid 
{[ -x f_grid ] [ -y f_grid ] [ -x_offset f_offset ] [ -y_offset f_offset ]}

Description

Sets the user grid parameters for the active window.

Arguments

-x f_grid

Specifies the x-axis user grid.

-x_offset f_offset

Specifies the x-axis user grid offset from 0.

-y f_grid

Specifies the y-axis user grid.

-y_offset f_offset

Specifies the y-axis user grid offset from 0.

Examples

The following example sets the user grid values for the active window.

set_user_grid -x 0.660000 -y 0.660000 -x_offset 0.330000 -y_offset 0.33000
