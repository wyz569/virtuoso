Spine-style Routing Commands

Spine-style routing is used to reduce delay from the output driver to the farthest load and minimize the global skew. The following is a list of the spine-style routing commands.

  • balance_route
  • report_balanced_length
  • report_spine_nets
  • route_taps
  • set_route_is_spine
  • set_spine_nets
  • spine_route
  • unset_route_is_spine
  • unset_spine_nets
