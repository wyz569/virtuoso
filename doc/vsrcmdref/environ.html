Space-based Router Tcl Environment Variables

Environment variables control various aspects of the environment including the GUI style and display options. The space-based router Tcl environment variables have been grouped into following categories.

  • Database Environment Variables
    These variables are set prior to reading a design to control what is loaded.
  • Graphical User Interface Environment
    The variables control the GUI settings.
  • Command Environment
    These variables are to control the command mode settings.
  • Interactive Environment
    The variables that control the interactive mode settings.
  • Router Environment
    These variables are to control the router mode settings.
  • Power Router Environment
    These variables control the power router mode settings.
  • Verify Environment
    These variables control the verify mode settings.
