pattern_route

droute.pattern_route int patternRouteValue

Description

Determines how the detail router treats bus routes.

  • 0 means no bus routing.
  • 1 means bus routing and then detail routing.
  • 2 means bus routing only.

Default value is 0.

GUI Equivalent

None

Examples

getvar droute.pattern_route
setvar droute.pattern_route 1
