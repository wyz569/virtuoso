is_advanced_node_version

is_advanced_node

Description

Specifies whether this version of Space-based Router supports advanced node constraints.

Arguments

None

Value Returned

true

This version supports advanced node constraints

false

This version does not support advanced node constraints.

Examples

The following command determines whether advanced node constraints are supported by this release.

is_advanced_node_version
