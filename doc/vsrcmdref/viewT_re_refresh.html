refresh

refresh 
[ -window_id i_windowID | -all] 
[ -pause ]

Description

Redraws the entire artwork window, updating the screen to show recent changes, additions and modifications.

If refresh is interrupted, a partially drawn window might be left on the screen.

The GUI equivalent for this command is View—Refresh.

Arguments

-all

Refreshes all artwork windows.

-pause

Waits for the windows to be refreshed before continuing with the next command.

-window_id i_windowID

Specifies the identifier of the window to be refreshed.

Examples

refresh -window_id 1

Redraws window 1.
