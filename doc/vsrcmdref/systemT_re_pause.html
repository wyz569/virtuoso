pause

pause
[ -delay i_delay ]

Description

Pauses the system for a specific interval or until the Continue button is clicked.

Arguments

-delay i_delay

Pauses the system for an interval of time. A value of 1000 is approximately one second. If this argument is not given, you must click the Continue button to resume operation.
