allow_pin_to_pin_space_checking

db allow_pin_to_pin_space_checking boolean { true | false }

Description

Flags spacing violations between pin shapes.

Default value is false.

GUI Equivalent

None

Examples

getvar db.allow_pin_to_pin_space_checking

setvar db.allow_pin_to_pin_space_checking true
