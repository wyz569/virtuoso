delete_soft_fence

delete_soft_fence
{-name s_regionName | -all}

Description

Removes an existing fence or all soft fences.

Arguments

-all

Removes all soft fences.

-name s_regionName

Specifies the name of the fence.

Examples

The following example deletes a fence named fence1.

delete_soft_fence -name fence1
