stop_button

cmd.stop_button boolean { true | false }

Description

When true, the stop button has been clicked. Set this flag to false using setvar to clear it.

Default value is false.

GUI Equivalent

None

Examples

getvar cmd.stop_button
setvar cmd.stop_button true
