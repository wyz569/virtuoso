Virtuoso Space-based Router Command Reference
Product Version IC23.1, September 2023

checkpoint

checkpoint

Description

Synchronizes the Virtuoso Layout Editor database with the Virtuoso Space-based Router database and sets a checkpoint in the Virtuoso Layout Editor transaction history that records the routing changes in Virtuoso Space-based Router. When you return to Virtuoso Layout Editor, checkpoints can be undone in reverse order.

Arguments

None

Value Returned

None

Examples

The following example sets a checkpoint to record the routing changes.

checkpoint
