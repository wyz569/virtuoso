Use of Tcl Commands

Tcl is used to run Virtuoso®Space-based Router. All commands performed by the tool, from either the GUI or from command line input, are logged as Tcl commands in the Transcript window. The logged commands can be replayed to duplicate the session. Tcl Core commands are supported and you can also write Tcl scripts for use with Virtuoso Space-based Router.
