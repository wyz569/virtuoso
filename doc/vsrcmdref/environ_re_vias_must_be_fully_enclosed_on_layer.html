vias_must_be_fully_enclosed_on_layer

droute.vias_must_be_fully_enclosed_on_layer string layers

Description

Specifies the layers on which the router only adds vias at pins if the via can be fully enclosed in the pin shape. It applies only when vias_must_be_fully_enclosed is true.

Default value is {}.

GUI Equivalent

None

Examples

getvar droute.vias_must_be_fully_enclosed_on_layer
setvar droute.vias_must_be_fully_enclosed_on_layer { metal1 metal2 }
