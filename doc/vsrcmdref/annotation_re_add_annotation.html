add_annotation

add_annotation
-object CtuId 
-parent_group CtuId
[ -make_group [ true | false ] ] 
[ -message messagestring ] 
[ -name namestring ]

Description

Annotates the specified object.

Arguments

-make_group [ true | false ]

Creates group to contain new annotations.

-message messagestring

Message text for the new annotation.

-name namestring

Name of the new annotation.

-object CtuId

Object to be annotated.

-parent_group CtuId

Parent group for the new annotation or group.
