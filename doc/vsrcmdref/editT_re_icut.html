icut

icut
[ -once | -repeat ]

Description

Sets the mouse command field to icut and allows you to interactively choose a cut location.

The GUI equivalent for this command is Edit—Cut.

Arguments

-once

Permits only one cut, then the mouse command mode automatically reverts to select mode.

-repeat

Permits consecutive cuts, until the mode is canceled or another interactive mode is enabled.
