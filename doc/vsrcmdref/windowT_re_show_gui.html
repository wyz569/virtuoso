show_gui

show_gui 
[ -aerial_view ]
[ -color_map_legend ] 
[ -command_line ] 
[ -lods ] 
[ -net_manager ] 
[ -sequencer ] 
[ -transcript ] 
[ -view_contexts ]

Description

Displays the auxiliary areas in the GUI.

Arguments

-aerial_view

Displays the aerial view. The menu command is Window—Aerial View.

-color_map_legend

Displays the Color Map Legend.

-command_line

Displays the command entry area. The menu command is Window—Command Line.

-lods

Displays the layer object display panel. The menu command Window—Layer Object Panel.

-net_manager

Displays the Net Manager docking window.

-sequencer

Displays the Sequencer docking window.

-transcript

Displays the Transcript area. The menu command is Window—Transcript Area

-view_contexts

Displays the view contexts browser. The menu command is Window—View Contexts Browser.

Examples

This command displays the layer/object display panel and the aerial view.

show_gui -lods -aerial_view
