abstract_view_name

db.abstract_view_name string viewName

Description

Specifies the view name to search for when building instances. This environment variable is used when db.map_layout_to_abstract_view is set to true.

Default value is abstract.

GUI Equivalent

None

Examples

getvar db.abstract_view_name

setvar db.abstract_view_name layout
