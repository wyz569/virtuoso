suppress_warn_on_same_mask_num_layers

db.suppress_warn_on_same_mask_num_layers boolean { true | flase }

Description

Controls the display of a warning message when a technology library has multiple layers with the same mask number. The warning message is suppressed when the environment variable is set to true.

Default value is false.

GUI Equivalent

None

Examples

getvar db.suppress_warn_on_same_mask_num_layers
setvar db.suppress_warn_on_same_mask_num_layers true
