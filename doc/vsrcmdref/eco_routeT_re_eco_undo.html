eco_undo

eco_undo 
-eco_control d_ecoObj 
[ -no_repaint ]

Description

Reverses all operations done by the command sequence given by the ECO control and terminates the sequence.

Arguments

-eco_control d_ecoObj

Specifies the command sequence ECO control that was assigned by the originating command.

-no_repaint

Prevents the artwork from being updated when this command is run.

Examples

eco_undo -eco_control $eco
