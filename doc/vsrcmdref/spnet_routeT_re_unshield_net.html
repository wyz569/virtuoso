unshield_net

unshield_net
{-net {s_netName…} | -set d_setObj}

Description

Removes the shield net flag from one or more signal nets.

This command does not remove existing shielding around a net, but prevents add_shield_wires from adding shielding to the given nets. Use delete_shield_wires to remove shielding.

Arguments

-net {s_netName…}

Removes the shield net flag from the named signal nets.

-set d_setObj

Removes the shield net flag from the signal nets in the given set.

Examples

The following command removes the shield net flag for net2 and net3.

unshield_net -net {net2 net3}
