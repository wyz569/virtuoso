highlight

highlight
[ -once | -repeat]

Description

Sets the mouse command field to highlight and allows you to interactively highlight objects.

The GUI equivalent for this command is Edit—Highlight.

Arguments

-once

Permits only one highlight, then the mouse command mode automatically reverts to select mode.

-repeat

Permits consecutive highlights, until the mode is canceled or another interactive mode is enabled.
