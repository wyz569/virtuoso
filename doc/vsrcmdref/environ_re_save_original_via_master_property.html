save_original_via_master_property

db.save_original_via_master_property boolean { true | false }

Description

Determines whether original via properties should be saved and loaded for each remastered via. Set this variable before saving the design and before re-loading the design to permit uncloning of remastered vias.

Default value is false.

GUI Equivalent

None

Examples

getvar db.save_original_via_master_property
setvar db.save_original_via_master_property true
