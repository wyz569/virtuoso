remove_tool_bar

remove_tool_bar 
-name s_name | -empty_bars

Description

Removes the specified toolbar or all toolbars that have no buttons.

Arguments

-empty_bars

Removes toolbars that have no buttons.

-name s_name

Specifies the name of the toolbar to remove.
