What's New

Enhanced Via Clustering

A new check box, non-Manhattan Vias, has been added to the Via Cluster group on the Pre-Process tab of the Electromagnetic Solver assistant. Select this check box to cluster vias that are not aligned in orthogonal rectangles.

Validating Bump and Ball Parameters for all Instances

Earlier, the Bump and Ball Editor form could be run only on flip chip dies and ball grid array instances. The support has now been extended to all instances, including surface-mount device instances.

Support for LVS Resistors

The assistant now supports the LVS resistors in your design. If a selected net contains an LVS resistor, depending on how the resistor is placed on the schematic or its properties are set in Virtuoso, the information about the LVS resistor is also sent to the electromagnetic simulator along with the details of the net to which it is connected.

Full Cellview Model Extraction with Excluded Cells

The Extract Full Cellview command has been enhanced to let you exclude selected devices before extracting the complete cellview. This feature lets you create models in the layout-driven flow where you do not have a schematic or the connectivity information is incomplete.
