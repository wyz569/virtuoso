Electromagnetic Solver Task Assistant

The Electromagnetic Solver assistant provides an interface to create 3D models for passive devices by running s-parameter extraction. The tool allows you to run full wave 3D Solvers, Clarity3D Solver, or EMX 3D planar solver. 

You can perform the following tasks to create EM Models:
